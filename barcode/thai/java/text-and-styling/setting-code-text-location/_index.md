# สร้างบาร์โค้ด Data Matrix ด้วย Java Barcode Library และกำหนดตำแหน่งข้อความโค้ด

## บทนำ

หากคุณกำลังสร้างระบบสินค้าคงคลัง การจัดส่ง หรือการติดตามใน Java คุณจะพบว่า **java barcode library** ที่เชื่อถือได้เป็นสิ่งจำเป็นอย่างยิ่ง ด้วย **Aspose.BarCode for Java** คุณสามารถสร้าง **Data Matrix barcode** ได้ด้วยเพียงไม่กี่บรรทัดของโค้ดและควบคุมการจัดวางภาพได้อย่างเต็มที่ รวมถึงตำแหน่งที่ข้อความที่มนุษย์อ่านได้ปรากฏ ในบทเรียนนี้เราจะอธิบายตัวอย่าง **barcode generation** อย่างครบถ้วน แสดงวิธี **customize barcode text** และอธิบายว่าการวางข้อความเหนือสัญลักษณ์จะช่วยปรับปรุงการใช้งานอย่างไร

## คำตอบอย่างรวดเร็ว
- **ไลบรารีที่ใช้คืออะไร?** Aspose.BarCode for Java  
- **ประเภทบาร์โค้ดที่แสดงคืออะไร?** Data Matrix  
- **คุณกำหนดตำแหน่งข้อความโค้ดอย่างไร?** Using `CodeLocation.ABOVE` (you can also use `BELOW`)  
- **ต้องการไลเซนส์สำหรับการผลิตหรือไม่?** Yes, a commercial license is required  
- **โค้ดสามารถทำงานบน Java 8+ ได้หรือไม่?** Absolutely, it supports Java 8 and later  

## ไลบรารีบาร์โค้ด Java คืออะไร?
**java barcode library** ให้ผู้พัฒนามี API ที่พร้อมใช้เพื่อสร้าง อ่าน และจัดรูปแบบบาร์โค้ด 1‑D และ 2‑D โดยตรงจากแอปพลิเคชัน Java มันทำให้ซับซ้อนของกฎการเข้ารหัสเป็นนามธรรมและให้คุณมุ่งเน้นที่ตรรกะธุรกิจแทนการประมวลผลภาพระดับต่ำ

## ทำไมต้องใช้ไลบรารีบาร์โค้ด Java สำหรับตัวอย่างการสร้างบาร์โค้ด?
- **ความเร็ว:** สร้างบาร์โค้ดคุณภาพสูงในเวลาเพียงไม่กี่มิลลิวินาที.  
- **ความน่าเชื่อถือ:** มีการปฏิบัติตามมาตรฐาน ISO/IEC ในตัว.  
- **ความยืดหยุ่น:** ควบคุมขนาด สี ระยะขอบ และการวาง **barcode text** ได้อย่างเต็มที่.  
- **ข้ามแพลตฟอร์ม:** ทำงานบนสภาพแวดล้อมที่ใช้ JVM ใดก็ได้ ตั้งแต่แอปพลิเคชันบนเดสก์ท็อปจนถึงบริการคลาวด์.

## Data Matrix barcode คืออะไร?
Data Matrix barcode คือสัญลักษณ์สองมิติ (2‑D) ที่เก็บข้อมูลจำนวนมากในรูปแบบสี่เหลี่ยมจัตุรัสหรือสี่เหลี่ยมผืนผ้าขนาดกะทัดรัด มันถูกใช้กันอย่างแพร่หลายในการทำเครื่องหมายสินค้าขนาดเล็ก อิเล็กทรอนิกส์ และอุปกรณ์ทางการแพทย์ เนื่องจากสามารถเข้ารหัสได้ถึง 2,335 ตัวอักษรอัลฟานูเมอริก

## ทำไมต้องกำหนดตำแหน่งข้อความบาร์โค้ด?
การวางข้อความที่มนุษย์อ่านได้ **เหนือ**, **ใต้**, หรือ **ข้าง** บาร์โค้ดช่วยให้ผู้ใช้ตรวจสอบข้อมูลที่เข้ารหัสได้อย่างรวดเร็วโดยไม่ต้องสแกน การปรับตำแหน่งข้อความช่วยปรับปรุงความสอดคล้องของ UI ปฏิบัติตามแนวทางแบรนด์ และสามารถลดข้อผิดพลาดในการป้อนข้อมูลด้วยมือ

## ข้อกำหนดเบื้องต้น
- ความรู้พื้นฐานเกี่ยวกับการเขียนโปรแกรม Java.  
- ติดตั้ง Java Development Kit (JDK).  
- IDE เช่น Eclipse หรือ IntelliJ IDEA.  
- ไลบรารี Aspose.BarCode for Java (ดาวน์โหลดจาก [ที่นี่](https://releases.aspose.com/barcode/java/)).  

## นำเข้าแพ็กเกจ
ขั้นแรก ให้นำเข้าคลาส Aspose.BarCode ที่จำเป็นเข้าสู่โปรเจกต์ของคุณ:

```java
import com.aspose.barcode.generation.CodeLocation;
import com.aspose.barcode.generation.BarcodeGenerator;
```

การนำเข้าเหล่านี้ทำให้คุณเข้าถึงตัวสร้างบาร์โค้ดและ enumeration ที่ควบคุมการวางตำแหน่งข้อความได้

## คู่มือขั้นตอนโดยละเอียด

### ขั้นตอนที่ 1: กำหนดเส้นทางไดเรกทอรี
ระบุที่ที่คุณต้องการอ่าน/เขียนไฟล์ เปลี่ยนค่า placeholder ให้เป็นเส้นทางจริงบนเครื่องของคุณ.

```java
String path = "Your Directory Path";
String dataDir = "Your Document Directory";
```

### ขั้นตอนที่ 2: สร้างอินสแตนซ์ BarcodeGenerator
สร้างอินสแตนซ์ `BarcodeGenerator` ด้วยประเภท **Data Matrix** และระบุข้อความโค้ดที่ต้องการเข้ารหัส.

```java
BarcodeGenerator generator = new BarcodeGenerator(EncodeTypes.DATA_MATRIX,
        "GTIN:898978777776665655 " + "UID: 121212121212121212 " + "Batch:GH768 " + "Exp.Date:150923");
```

### ขั้นตอนที่ 3: กำหนดตำแหน่งข้อความบาร์โค้ด (ปรับแต่งข้อความบาร์โค้ด)
ใช้ enumeration `CodeLocation` เพื่อย้ายข้อความที่มนุษย์อ่านได้ ในตัวอย่างนี้เราวางข้อความ **เหนือ** บาร์โค้ด.

```java
generator.getParameters().getBarcode().getCodeTextParameters().setLocation(CodeLocation.ABOVE);
```

> **เคล็ดลับ:** เพื่อวางข้อความ **ใต้** สัญลักษณ์ เพียงเปลี่ยน `CodeLocation.ABOVE` เป็น `CodeLocation.BELOW`.

### ขั้นตอนที่ 4: บันทึกภาพบาร์โค้ดที่สร้างขึ้น
สุดท้าย ให้บันทึกภาพบาร์โค้ดลงดิสก์ ไฟล์จะประกอบด้วยสัญลักษณ์ Data Matrix พร้อมข้อความที่วางเหนือสัญลักษณ์.

```java
generator.save(dataDir + "codetextAbove.png");
```

## ปัญหาทั่วไปและวิธีแก้ไข
- **ข้อความไม่แสดง:** ตรวจสอบว่าคุณใช้เวอร์ชันของ Aspose.BarCode ที่รองรับ `CodeLocation`.  
- **ข้อผิดพลาดของเส้นทางไฟล์:** ตรวจสอบว่า `dataDir` ลงท้ายด้วยตัวคั่นไฟล์ (`/` หรือ `\\`) ที่เหมาะกับระบบปฏิบัติการของคุณ.  
- **อักขระที่ไม่รองรับ:** Data Matrix สามารถเข้ารหัสได้เพียงชุดอักขระที่จำกัด; หลีกเลี่ยงสัญลักษณ์พิเศษที่ไม่อยู่ในมาตรฐาน ISO/IEC 16022.

## คำถามที่พบบ่อย (FAQs)

### Q: ฉันสามารถปรับแต่งลักษณะของบาร์โค้ดที่สร้างได้หรือไม่?
A: ใช่, Aspose.BarCode มีตัวเลือกการปรับแต่งอย่างกว้างขวาง ให้คุณควบคุมสี, ระยะขอบ, และรูปแบบฟอนต์.

### Q: Aspose.BarCode รองรับ Java 8 และเวอร์ชันต่อ ๆ ไปหรือไม่?
A: แน่นอน, ไลบรารีทำงานกับ Java 8 และรุ่นต่อ ๆ ไปทั้งหมด.

### Q: ฉันจะรวม Aspose.BarCode เข้ากับโปรเจกต์ Java ที่มีอยู่ของฉันได้อย่างไร?
A: เพิ่มไฟล์ JAR ของ Aspose.BarCode ไปยัง classpath ของโปรเจกต์ของคุณ, นำเข้าแพ็กเกจที่จำเป็น, แล้วคุณพร้อมสร้างบาร์โค้ดแล้ว.

### Q: มีเวอร์ชันทดลองสำหรับ Aspose.BarCode หรือไม่?
A: ใช่, คุณสามารถสำรวจความสามารถของ Aspose.BarCode โดยรับเวอร์ชันทดลองฟรี [ที่นี่](https://releases.aspose.com/).

### Q: ฉันสามารถขอความช่วยเหลือหรือสนับสนุนสำหรับ Aspose.BarCode ได้จากที่ไหน?
A: เยี่ยมชม [ฟอรั่ม Aspose.BarCode](https://forum.aspose.com/c/barcode/13) เพื่อรับความช่วยเหลือจากชุมชนและการสนับสนุนอย่างเป็นทางการ.

## คำถามเพิ่มเติมที่พบบ่อย

**Q: ฉันสามารถสร้างบาร์โค้ดที่มีข้อความอยู่ใต้สัญลักษณ์ได้หรือไม่?**  
A: ใช่, ตั้งค่า `CodeLocation.BELOW` ในเมธอด `setLocation` เดียวกัน.

**Q: ไลบรารีรองรับบาร์โค้ด 2‑D อื่น ๆ เช่น QR Code หรือไม่?**  
A: แน่นอน, เพียงเปลี่ยน `EncodeTypes.DATA_MATRIX` เป็น `EncodeTypes.QR`.

**Q: ฉันจะเปลี่ยนขนาดฟอนต์ของข้อความบาร์โค้ดได้อย่างไร?**  
A: ใช้ `generator.getParameters().getBarcode().getCodeTextParameters().setFontSize(double size)`.

## สรุป

คุณได้เรียนรู้วิธี **สร้าง Data Matrix barcode** ใน Java ด้วย **java barcode library** และควบคุม **การกำหนดตำแหน่งข้อความบาร์โค้ด** อย่างแม่นยำด้วย Aspose.BarCode แล้ว ลองทดลองใช้ค่า `CodeLocation` อื่น ๆ การตั้งค่าฟอนต์ และตัวเลือกสีเพื่อให้ตรงกับความต้องการการออกแบบของแอปพลิเคชันของคุณ.
