# วิธีสร้างบาร์โค้ดขนาดต่ำสุดใน Java ด้วย Aspose.BarCode

## บทนำ

หากคุณต้องการ **สร้างบาร์โค้ดขนาดต่ำสุด** สำหรับเลย์เอาต์ UI ที่แคบ, ป้ายพร้อมพิมพ์, หรือสถานการณ์ใด ๆ ที่มิลลิเมตรทุกหน่วยมีความสำคัญ, คุณมาถูกที่แล้ว ใน **บทแนะนำการสร้างบาร์โค้ดด้วย Java** นี้ เราจะอธิบายขั้นตอนที่จำเป็นเพื่อย่อ BarCode ให้มีขนาดเล็กที่สุดเท่าที่จะทำได้โดยยังคงสามารถสแกนได้, โดยใช้ Aspose.BarCode for Java

## คำตอบสั้น ๆ
- **“บาร์โค้ดขนาดต่ำสุด” หมายถึงอะไร?** คือมิติภาพที่เล็กที่สุดที่ยังคงตอบสนองข้อกำหนดการอ่านของสัญลักษณ์นั้น  
- **คลาสใดที่สร้างบาร์โค้ด?** `BarcodeGenerator` จากไลบรารี Aspose.BarCode  
- **ต้องใช้ไลเซนส์สำหรับตัวอย่างนี้หรือไม่?** สามารถใช้รุ่นทดลองฟรีสำหรับการพัฒนา; ต้องมีไลเซนส์เชิงพาณิชย์สำหรับการใช้งานจริง  
- **สามารถเปลี่ยนขนาดได้หลังจากปิด AutoSize หรือไม่?** ได้ – คุณตั้งค่าความกว้าง/ความสูงโดยระบุเป็นมิลลิเมตรโดยตรง  
- **วิธีนี้ใช้ได้กับบาร์โค้ดทุกประเภทหรือไม่?** ส่วนใหญ่ของสัญลักษณ์ 1‑D (เช่น CODE_128, CODE_39) รองรับการกำหนดขนาดด้วยตนเอง; ตรวจสอบเอกสารสำหรับรหัส 2‑D

## “สร้างบาร์โค้ดขนาดต่ำสุด” คืออะไร?
การสร้างบาร์โค้ดขนาดต่ำสุดหมายถึงการกำหนดค่าตัวสร้างให้ **ไม่** ขยายภาพโดยอัตโนมัติ แต่คุณระบุขนาดที่ต้องการอย่างแม่นยำ เพื่อให้บาร์โค้ดพอดีกับพื้นที่แคบโดยไม่มีพื้นที่ว่างเกินจำเป็น

## ทำไมต้องใช้บทแนะนำการสร้างบาร์โค้ดด้วย Java แบบนี้?
- **การออกแบบที่ประหยัดพื้นที่** – เหมาะสำหรับหน้าจอมือถือ, ใบเสร็จ, หรือเครื่องพิมพ์ป้ายขนาดกะทัดรัด  
- **การควบคุมเต็มรูปแบบ** – คุณกำหนดขนาดพิกเซลหรือมิลลิเมตรได้ตามต้องการ  
- **ผลลัพธ์สม่ำเสมอ** – โค้ดเดียวกันทำงานได้บน Windows, Linux, และ macOS JVMs  

## ข้อกำหนดเบื้องต้น

ก่อนที่เราจะลงมือเขียนโค้ด, โปรดตรวจสอบว่าคุณมี:

1. **Java Development Kit (JDK)** – เวอร์ชันล่าสุด (แนะนำ 8+)  
2. **Aspose.BarCode for Java** – ดาวน์โหลดไลบรารีล่าสุดจากเว็บไซต์อย่างเป็นทางการ [ที่นี่](https://releases.aspose.com/barcode/java/)  

ตอนนี้เริ่มเขียนโค้ดกันเลย

## นำเข้า Namespace

ในไฟล์ซอร์ส Java ของคุณ, นำเข้าคลาส Aspose ที่จำเป็น:

```java
import com.aspose.barcode.generation.AutoSizeMode;
import com.aspose.barcode.generation.BarcodeGenerator;
```

## คู่มือขั้นตอนต่อขั้นตอนเพื่อสร้างบาร์โค้ดขนาดต่ำสุด

### ขั้นตอนที่ 1: ตั้งค่า Barcode Generator
สร้างอินสแตนซ์ `BarcodeGenerator`, เลือกสัญลักษณ์ (CODE_128 ในตัวอย่างนี้), และกำหนดข้อมูลที่ต้องการเข้ารหัส

```java
BarcodeGenerator bb = new BarcodeGenerator(EncodeTypes.CODE_128, "1234567");
```

### ขั้นตอนที่ 2: ปิด AutoSizeMode
โดยค่าเริ่มต้น Aspose.BarCode จะขยายภาพให้พอดีกับบาร์โค้ด ปิดพฤติกรรมนี้เพื่อให้คุณกำหนดขนาดของตนเอง

```java
bb.getParameters().setAutoSizeMode(AutoSizeMode.NONE);
```

### ขั้นตอนที่ 3: กำหนดความกว้างและความสูงของภาพขั้นต่ำ
ระบุความกว้างและความสูงที่เล็กที่สุดที่ยังคงให้บาร์โค้ดอ่านได้ เราใช้ 1 mm สำหรับทั้งสองมิติ, แต่คุณสามารถปรับได้ตามต้องการ

```java
bb.getParameters().getImageWidth().setMillimeters(1);
bb.getParameters().getImageHeight().setMillimeters(1);
```

> **เคล็ดลับ:** ใช้คุณสมบัติ `getImageWidth()` และ `getImageHeight()` เพื่อทดลองขนาดต่าง ๆ จนกว่าสแกนเนอร์อ่านได้อย่างมั่นใจ

### ขั้นตอนที่ 4: บันทึกภาพบาร์โค้ด
สร้างบิตแมปและเขียนลงไฟล์ PNG แทนที่ `dataDir` ด้วยเส้นทางที่คุณต้องการบันทึกภาพ

```java
javax.imageio.ImageIO.write(bb.generateBarCodeImage(), "PNG", new java.io.File(dataDir + "minimumresult.png"));
```

ทำซ้ำขั้นตอนข้างต้นสำหรับบาร์โค้ดแต่ละรายการที่ต้องการสร้างที่ขนาดต่ำสุด
