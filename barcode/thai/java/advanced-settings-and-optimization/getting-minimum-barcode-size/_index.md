# วิธีสร้างป้ายบาร์โค้ดขนาดเล็กใน Java ด้วย Aspose.BarCode

## บทนำ

หากคุณต้องการ **สร้างป้ายบาร์โค้ดขนาดเล็ก** สำหรับการจัดวาง UI ที่แคบ, แท็กพร้อมพิมพ์, หรือสถานการณ์ใด ๆ ที่ทุกมิลลิเมตรมีความสำคัญ, คุณมาถูกที่แล้ว ใน **barcode generator tutorial Java** นี้ เราจะอธิบายขั้นตอนที่จำเป็นเพื่อย่อ BarCode ให้มีขนาดเล็กที่สุดเท่าที่จะเป็นไปได้โดยยังคงสามารถสแกนได้, โดยใช้ Aspose.BarCode สำหรับ Java.

## คำตอบอย่างรวดเร็ว
- **“minimum barcode” หมายถึงอะไร?** เป็นมิติของภาพที่เล็กที่สุดที่ยังคงตอบสนองความต้องการการอ่านของสัญลักษณ์.  
- **คลาสใดที่สร้างบาร์โค้ด?** `BarcodeGenerator` จากไลบรารี Aspose.BarCode.  
- **ฉันต้องการไลเซนส์สำหรับตัวอย่างนี้หรือไม่?** การทดลองใช้ฟรีทำงานสำหรับการพัฒนา; จำเป็นต้องมีไลเซนส์เชิงพาณิชย์สำหรับการใช้งานจริง.  
- **ฉันสามารถเปลี่ยนขนาดหลังจากปิด AutoSize ได้หรือไม่?** ได้ – คุณตั้งค่าความกว้าง/ความสูงโดยตรงเป็นมิลลิเมตร.  
- **วิธีนี้ใช้ได้กับทุกประเภทของบาร์โค้ดหรือไม่?** ส่วนใหญ่ของสัญลักษณ์ 1‑D (เช่น CODE_128, CODE_39) รองรับการกำหนดขนาดด้วยตนเอง; ตรวจสอบเอกสารสำหรับโค้ด 2‑D.

## อะไรคือ “create minimum barcode”?

การสร้างบาร์โค้ดขนาดต่ำสุดหมายถึงการกำหนดค่าตัวสร้างให้ **ไม่** ขยายภาพโดยอัตโนมัติ แทนที่จะนั้นคุณระบุมิติที่ต้องการอย่างแม่นยำ ทำให้คุณสามารถใส่บาร์โค้ดลงในพื้นที่แคบ ๆ ได้โดยไม่มีพื้นที่ว่างเสียเปล่า.

## ข้อกำหนดเบื้องต้น

ก่อนที่เราจะลงลึกในโค้ด, โปรดตรวจสอบว่าคุณมี:

1. **Java Development Kit (JDK)** – เวอร์ชันล่าสุดใดก็ได้ (แนะนำ 8+).  
2. **Aspose.BarCode for Java** – ดาวน์โหลดไลบรารีล่าสุดจากเว็บไซต์ทางการ [here](https://releases.aspose.com/barcode/java/).  

ตอนนี้มาเริ่มเขียนโค้ดกัน.

## นำเข้า Namespaces

ในไฟล์ซอร์ส Java ของคุณ, ให้ import คลาส Aspose ที่จำเป็น:

```java
import com.aspose.barcode.generation.AutoSizeMode;
import com.aspose.barcode.generation.BarcodeGenerator;
```

## คู่มือขั้นตอนการสร้างบาร์โค้ดขนาดต่ำสุด

### ขั้นตอนที่ 1: ตั้งค่า Barcode Generator
สร้างอินสแตนซ์ของ `BarcodeGenerator`, เลือกสัญลักษณ์ (CODE_128 ในตัวอย่างนี้), และให้ข้อมูลที่ต้องการเข้ารหัส.

```java
BarcodeGenerator bb = new BarcodeGenerator(EncodeTypes.CODE_128, "1234567");
```

### ขั้นตอนที่ 2: ปิด AutoSizeMode
โดยค่าเริ่มต้น Aspose.BarCode จะขยายภาพเพื่อให้พอดีกับบาร์โค้ด ปิดการทำงานนี้เพื่อให้คุณกำหนดมิติของคุณเอง.

```java
bb.getParameters().setAutoSizeMode(AutoSizeMode.NONE);
```

### ขั้นตอนที่ 3: กำหนดความกว้างและความสูงขั้นต่ำของภาพ
ระบุความกว้างและความสูงที่เล็กที่สุดที่ยังคงทำให้บาร์โค้ดอ่านได้ ที่นี่เราใช้ 1 mm สำหรับทั้งสองมิติ, แต่คุณสามารถปรับได้ตามต้องการ.

```java
bb.getParameters().getImageWidth().setMillimeters(1);
bb.getParameters().getImageHeight().setMillimeters(1);
```

> **เคล็ดลับ:** ใช้คุณสมบัติ `getImageWidth()` และ `getImageHeight()` เพื่อทดลองกับขนาดต่าง ๆ จนกว่าสแกนเนอร์อ่านโค้ดได้อย่างเชื่อถือ.

### ขั้นตอนที่ 4: บันทึกภาพบาร์โค้ด
สร้างบิตแมพและเขียนลงไฟล์ PNG แทนที่ `dataDir` ด้วยเส้นทางที่คุณต้องการบันทึกภาพ.

```java
javax.imageio.ImageIO.write(bb.generateBarCodeImage(), "PNG", new java.io.File(dataDir + "minimumresult.png"));
```

ทำซ้ำขั้นตอนข้างต้นสำหรับบาร์โค้ดแต่ละอันที่คุณต้องการสร้างที่ขนาดต่ำสุด.

## ทำไมป้ายบาร์โค้ดขนาดเล็กจึงสำคัญ
เมื่อคุณพิมพ์บนแท็กขนาดจำกัด—เช่นบรรจุภัณฑ์สินค้า, สติ๊กเกอร์ RFID, หรือ QR โค้ดบนหน้าจอ—**ภาพบาร์โค้ดที่กะทัดรัด** ช่วยประหยัดวัสดุ, ลดเวลาการพิมพ์, และปรับปรุงความสวยงามโดยรวมของป้าย ป้ายบาร์โค้ดขนาดเล็กยังทำให้ผู้ออกแบบสามารถบรรจุข้อมูลมากขึ้นในพื้นที่เดียวโดยไม่เสียการอ่าน.
