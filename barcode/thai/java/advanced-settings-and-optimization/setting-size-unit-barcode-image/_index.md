# สร้าง code128 barcode java: ตั้งหน่วยขนาดสำหรับภาพ Barcode

## บทนำ

ในบทแนะนำแบบขั้นตอนนี้ คุณจะ **ใช้ Aspose.BarCode for Java** เพื่อ **สร้าง code128 barcode java**, สร้างภาพ barcode java, และควบคุมหน่วยขนาดของผลลัพธ์อย่างแม่นยำ ไม่ว่าคุณจะกำลังสร้าง barcode สำหรับระบบสินค้าคงคลัง, ตัวสร้างป้ายจัดส่ง, หรือแอปพลิเคชันที่ใช้ Java ใด ๆ ที่ต้องการ barcode ที่เชื่อถือได้, Aspose.BarCode จะให้ความยืดหยุ่นเต็มที่ด้วยเพียงไม่กี่บรรทัดของโค้ด.

## คำตอบเร็ว

- **ต้องการไลบรารีอะไร?** Aspose.BarCode for Java (aspose barcode java).  
- **ประเภท barcode ที่ครอบคลุมคืออะไร?** CODE_128 (create code128 barcode java).  
- **จะตั้งหน่วยขนาดอย่างไร?** ใช้คุณสมบัติ `BarHeight` กับ `.setPoint()`.  
- **ฉันสามารถสร้าง barcode image java ในรูปแบบอื่นได้หรือไม่?** ใช่ – PNG, JPEG, BMP, เป็นต้น.  
- **ข้อกำหนดเบื้องต้นคืออะไร?** ติดตั้ง JDK, ไลบรารี Aspose.BarCode, และ IDE สำหรับ Java.

## **create code128 barcode java** คืออะไร?

`create code128 barcode java` หมายถึงการใช้คลาส `BarcodeGenerator` พร้อม enum `EncodeTypes.CODE_128` เพื่อเข้ารหัสสตริงข้อมูลเป็นสัญลักษณ์ CODE_128 สัญลักษณ์นี้รองรับชุด ASCII ทั้งหมดและให้ความหนาแน่นของข้อมูลสูง ทำให้เหมาะสำหรับ barcode ในระบบสินค้าคงคลังและสถานการณ์ป้ายจัดส่ง.

## ทำไมต้องใช้ Aspose.BarCode เพื่อ **generate barcode image java**?

การสร้าง barcode image java ด้วย Aspose.BarCode ทำให้คุณสามารถควบคุมมิติ, สี, และความละเอียดได้ในขณะที่ยังคงการทำงานเป็น Java แท้ ไลบรารีนี้รองรับ **รูปแบบการนำเข้าและส่งออกกว่า 50 รูปแบบ** และสามารถสร้างภาพ barcode หลายร้อยหน้าโดยไม่ต้องโหลดไฟล์ทั้งหมดเข้าสู่หน่วยความจำ ส่งมอบประสิทธิภาพระดับมิลลิวินาทีสำหรับการสร้างป้ายเป็นชุด.

## ข้อกำหนดเบื้องต้น

ก่อนที่เราจะเริ่ม, โปรดตรวจสอบว่าคุณมี:

1. **Java Development Kit (JDK)** – เวอร์ชัน 8 หรือใหม่กว่า ติดตั้งบนเครื่องของคุณ.  
2. **Aspose.BarCode for Java library** – ดาวน์โหลด JAR ล่าสุดจากเว็บไซต์ Aspose (รุ่นทดลองหรือแบบมีลิขสิทธิ์).  
3. **A Java IDE** – เช่น IntelliJ IDEA, Eclipse, หรือ VS Code พร้อมส่วนขยายสำหรับ Java.  

## นำเข้า Namespaces

เพิ่มการนำเข้าที่จำเป็นที่ส่วนหัวของคลาส Java ของคุณเพื่อให้เข้าถึง API ของ Aspose.BarCode:

```java
import java.io.IOException;

import com.aspose.barcode.*;

import com.aspose.barcode.generation.BarcodeGenerator;
```

## วิธี **generate barcode image java** ด้วย Aspose.BarCode?

โหลด `BarcodeGenerator` ของคุณ, ตั้งค่าขนาด, และบันทึกภาพ – ทั้งหมดในกระบวนการที่ชัดเจนและเป็นลำดับที่สามารถใส่ในลูปสำหรับการประมวลผลเป็นกลุ่มได้ ย่อหน้าตอบโดยตรงนี้บอกคุณอย่างชัดเจนว่าต้องทำอะไรก่อนที่เราจะลงลึกในรายละเอียดขั้นตอนต่อไป.

### ขั้นตอนที่ 1: กำหนดไดเรกทอรีทรัพยากร

แรกสุด ให้ระบุโฟลเดอร์ที่ไฟล์ที่สร้างขึ้นจะถูกเขียนลงไป ไดเรกทอรีนี้ต้องมีอยู่และสามารถเขียนได้โดยกระบวนการ Java.

```java
// The path to the resource directory.
String dataDir = "Your Document Directory";
```

แทนที่ `"Your Document Directory"` ด้วยพาธเต็มที่คุณต้องการให้บันทึกภาพ barcode โฟลเดอร์นี้จะเก็บไฟล์ PNG/JPEG ที่สร้างขึ้นซึ่งคุณสามารถฝังในใบแจ้งหนี้, ใบส่งของ, หรือรายงานสินค้าคงคลังในภายหลัง.

### ขั้นตอนที่ 2: สร้างอ็อบเจ็กต์ Barcode

`EncodeTypes.CODE_128` ระบุสัญลักษณ์ barcode CODE_128.

`BarcodeGenerator` เป็นคลาสหลักของ Aspose.BarCode ที่แสดงภาพ barcode ในหน่วยความจำ การสร้างอินสแตนซ์ `create code128 barcode java` ทำได้ง่ายโดยการส่งค่า enum `EncodeTypes.CODE_128` และสตริงข้อมูลที่คุณต้องการเข้ารหัส.

```java
// Instantiate barcode object, Set the symbology type to code128 and Set the
// Code text for the barcode
BarcodeGenerator bb = new BarcodeGenerator(EncodeTypes.CODE_128, "1234567");
```

ที่นี่เราจะ **create code128 barcode java** โดยส่งค่า `EncodeTypes.CODE_128` และสตริงข้อมูล `"1234567"`.

### ขั้นตอนที่ 3: ตั้งค่า Bar Height (หน่วยขนาด)

`BarHeight` กำหนดขนาดแนวตั้งของบาร์ วิธี `.setPoint()` ตั้งค่าสูงนี้เป็นหน่วยจุด (1 จุด = 1/72 นิ้ว) ทำให้คุณควบคุมขนาดภาพสุดท้ายได้อย่างแม่นยำ.

```java
// Set the bar height to 3 points
bb.getParameters().getBarcode().getBarHeight().setPoint(3.0f);
```

เมธอด `setPoint()` กำหนดความสูงเป็นหน่วยจุด ปรับค่าตัวนี้ให้ตรงกับความต้องการของการจัดวางของคุณ — ค่าที่ใหญ่กว่าจะทำให้บาร์สูงขึ้น ซึ่งมักจำเป็นสำหรับป้ายจัดส่งความละเอียดสูง.

### ขั้นตอนที่ 4: บันทึกภาพ

การเรียก `save()` จะเขียน barcode ไปยังโฟลเดอร์ที่คุณระบุ รูปแบบภาพจะถูกกำหนดจากนามสกุลไฟล์; คุณสามารถเปลี่ยนเป็น PNG, BMP, หรือ TIFF ได้โดยการเปลี่ยนนามสกุลไฟล์.

```java
// Save the image
bb.save(dataDir + "barcode-size-unit.jpg");
```

การเรียก `save()` จะบันทึก barcode ที่สร้างลงในโฟลเดอร์ที่คุณระบุ รูปแบบภาพจะถูกกำหนดจากนามสกุลไฟล์ (ในกรณีนี้เป็น JPEG) คุณสามารถเปลี่ยนเป็น PNG, BMP, หรือ TIFF ได้โดยการเปลี่ยนนามสกุลไฟล์.

## ปัญหาที่พบบ่อยและวิธีแก้

| ปัญหา | สาเหตุ | วิธีแก้ |
|-------|--------|-----|
| **ไม่สร้างภาพ** | `dataDir` พาธไม่ถูกต้องหรือไม่มีสิทธิ์เขียน | ตรวจสอบว่าโฟลเดอร์มีอยู่และแอปพลิเคชันของคุณมีสิทธิ์เขียน |
| **Barcode เล็กเกินไป** | ความสูงของบาร์ที่ตั้งเป็นจุดต่ำเกินไปสำหรับ DPI ที่เลือก | เพิ่มค่าที่ส่งให้ `setPoint()` หรือปรับ DPI ของภาพผ่าน `bb.getParameters().getImage().setResolution()` |
| **อักขระที่ไม่รองรับ** | CODE_128 รองรับเฉพาะ ASCII; คุณส่ง Unicode | ใช้สัญลักษณ์อื่น (เช่น QR_CODE) สำหรับข้อมูลที่ไม่ใช่ ASCII |

### เคล็ดลับเพิ่มเติม (Pro tip)

`getParameters().getImage().setResolution()` ตั้งค่าความละเอียดภาพเป็น DPI.

- **การประมวลผลเป็นชุด:** ใส่ขั้นตอนข้างต้นในลูปเพื่อสร้างหลายร้อย barcode สำหรับการอัปโหลดสินค้าจำนวนมาก.  
- **การควบคุมความละเอียด:** ใช้ `bb.getParameters().getImage().setResolution(300)` เพื่อสร้างภาพ 300 dpi ซึ่งเหมาะสำหรับป้ายพิมพ์คุณภาพสูง.
