# ไลบรารีอ่านบาร์โค้ด Java: ตรวจจับการวางแนวบาร์โค้ดด้วย Aspose.BarCode

## บทนำ

หากคุณต้องการ **java barcode reader library** ที่เชื่อถือได้สำหรับแอปพลิเคชัน Java ของคุณ Aspose.BarCode for Java มีความสามารถในการจดจำบาร์โค้ดที่ทรงพลัง รวมถึงการตรวจจับการวางแนว ในบทแนะนำนี้เราจะอธิบายวิธี **read barcode from image java** และรับค่ามุมการหมุนเพื่อให้คุณจัดการบาร์โค้ดที่หมุนได้อย่างง่ายดาย.

## คำตอบอย่างรวดเร็ว
- **ไลบรารีทำอะไร?** ตรวจจับประเภทบาร์โค้ด, อ่านข้อมูล, และคืนค่ามุมการวางแนว.  
- **ประเภทบาร์โค้ดที่ใช้ในตัวอย่างคืออะไร?** Code 128 (`DecodeType.CODE_128`).  
- **ต้องใช้ลิขสิทธิ์สำหรับการทดสอบหรือไม่?** มีลิขสิทธิ์ชั่วคราวสำหรับการประเมินผล.  
- **สามารถประมวลผลหลายภาพได้หรือไม่?** ได้ – เพียงวนลูปผ่านไฟล์ภาพของคุณด้วยตรรกะการอ่านเดียวกัน.  
- **เข้ากันได้กับ Java 8+ หรือไม่?** แน่นอน, ไลบรารีทำงานกับ Java 8 และรุ่นต่อ ๆ ไป.

## ไลบรารีอ่านบาร์โค้ด Java คืออะไร?
ไลบรารีอ่านบาร์โค้ด Java ให้ API ที่ช่วยให้นักพัฒนาถอดรหัสบาร์โค้ดจากภาพ, PDF หรือสตรีมวิดีโอแบบสดโดยตรงในโค้ด Java Aspose.BarCode เป็นไลบรารีเชิงพาณิชย์ที่รองรับสัญลักษณ์บาร์โค้ดกว่า 150 แบบและมีคุณสมบัติขั้นสูงเช่นการตรวจจับการวางแนว, การตรวจสอบ checksum, และการประมวลผลหลายหน้า.

## ทำไมต้องใช้ Aspose.BarCode สำหรับการตรวจจับการวางแนว?
- **การคำนวณมุมที่แม่นยำ** – ไลบรารีคืนค่ามุมการหมุนของพื้นที่บาร์โค้ดอย่างแม่นยำ.  
- **รองรับสัญลักษณ์หลากหลาย** – ทำงานกับ Code 128, QR, DataMatrix, และอื่น ๆ อีกมาก.  
- **API ง่าย** – ต้องการโค้ดเพียงเล็กน้อยเพื่อเริ่มต้น.  
- **พร้อมใช้งานระดับองค์กร** – ประสิทธิภาพสูง, การจัดการข้อผิดพลาดที่แข็งแรง, และตัวเลือกการให้ลิขสิทธิ์.

## นำเข้า Namespaces

เพื่อเริ่มต้น ให้นำเข้า namespaces ที่จำเป็นเข้าสู่โครงการ Java ของคุณ ขั้นตอนนี้สำคัญสำหรับการเข้าถึงฟังก์ชันที่ Aspose.BarCode for Java ให้มา.

```java
// Import Aspose.BarCode namespaces
import com.aspose.barcode.barcoderecognition.BarCodeReader;

```

## วิธีอ่านบาร์โค้ด Java ด้วย Aspose.BarCode
ด้านล่างเป็นคู่มือสั้น ๆ ทีละขั้นตอนที่แสดง **how to read barcodes java** และรับค่าการวางแนวของบาร์โค้ด.

### ขั้นตอนที่ 1: สร้างอ็อบเจ็กต์ BarCodeReader
เริ่มต้นด้วยการสร้างอ็อบเจ็กต์ `BarCodeReader` โดยระบุไฟล์ภาพที่มีบาร์โค้ดและประเภทบาร์โค้ดที่ต้องการ.

```java
BarCodeReader reader = new BarCodeReader("rotatedbarcode.jpg", DecodeType.CODE_128);
```

### ขั้นตอนที่ 2: อ่านบาร์โค้ด Code128
ใช้เมธอด `readBarCodes` เพื่ออ่านบาร์โค้ด Code 128 จากภาพที่ระบุ.

```java
for (BarCodeResult result : reader.readBarCodes()) {
```

### ขั้นตอนที่ 3: ตรวจจับการวางแนวของบาร์โค้ด
ดึงพื้นที่บาร์โค้ดและรับค่ามุมการหมุน.

```java
    // detect bar code orientation
    System.out.println("Rotation Angle: " + result.getRegion().getAngle());
}
```

โดยทำตามสามขั้นตอนง่าย ๆ นี้ คุณสามารถผสานการตรวจจับการวางแนวของบาร์โค้ดเข้าไปในแอปพลิเคชัน Java ของคุณได้อย่างราบรื่นด้วย **java barcode reader library**.

## กรณีการใช้งานทั่วไป
- **การประมวลผลเอกสารอัตโนมัติ** – สแกนใบแจ้งหนี้หรือป้ายจัดส่งที่อาจมาถึงในมุมใดก็ได้.  
- **ระบบสินค้าคงคลังในร้านค้า** – อ่านบาร์โค้ดสินค้าจากกล้องที่สินค้าไม่ได้จัดเรียงอย่างสมบูรณ์.  
- **ระบบอัตโนมัติในอุตสาหกรรม** – ตรวจจับและแก้ไขการวางแนวของบาร์โค้ดบนสายการผลิตก่อนดำเนินการต่อ.
