# การจดจำบาร์โค้ด PDF417 ที่มีอักขระจีนใน Java

## คำแนะนำ

หากคุณกำลังมองหา **วิธีใช้ Aspose** เพื่ออ่านบาร์โค้ดในแอปพลิเคชัน Java ของคุณ คุณมาถูกที่แล้ว บทแนะนำนี้จะพาคุณผ่านการใช้ไลบรารี Aspose.BarCode เพื่อ **จดจำบาร์โค้ด PDF417 ที่มีอักขระจีน** เมื่อจบคู่มือคุณจะเข้าใจกระบวนการทำงานทั้งหมด ตั้งแต่การตั้งค่าสภาพแวดล้อมจนถึงการถอดรหัสข้อมูลบาร์โค้ด—เพื่อให้คุณสามารถเพิ่มความสามารถในการอ่านบาร์โค้ดให้กับระบบสินค้าคงคลัง เครื่องมือจัดการเอกสาร หรือโซลูชันใด ๆ ที่ใช้ Java ได้อย่างมั่นใจ

## คำตอบสั้น
- **ต้องใช้ไลบรารีอะไร?** Aspose.BarCode for Java (ไลบรารีอ่านบาร์โค้ดที่แข็งแรงสำหรับ Java)  
- **ประเภทบาร์โค้ดที่แสดงคืออะไร?** PDF417 ที่มีอักขระจีน  
- **ต้องมีลิขสิทธิ์สำหรับการทดสอบหรือไม่?** ทดลองใช้ฟรีได้สำหรับการพัฒนา; ต้องมีลิขสิทธิ์เชิงพาณิชย์สำหรับการใช้งานจริง  
- **รองรับเวอร์ชัน Java ใด?** Java 8 หรือใหม่กว่า (แนะนำให้ใช้ JDK เวอร์ชันล่าสุด)  
- **ข้อความถูกถอดรหัสอย่างไร?** ใช้ charset MS936 เพื่อแสดงอักขระจีนอย่างถูกต้อง

## Aspose.BarCode for Java คืออะไร?
Aspose.BarCode for Java เป็น **barcode reader library java** ที่รองรับสัญลักษณ์บาร์โค้ดกว่า 150 ชนิด ให้การถอดรหัสที่มีประสิทธิภาพสูง การสนับสนุนหลายภาษา และการผสานรวมง่ายกับโครงการ Java มาตรฐาน—ทำให้เป็นตัวเลือกอันดับต้น ๆ สำหรับงาน **java barcode recognition**

## ทำไมต้องใช้ Aspose สำหรับการจดจำบาร์โค้ดใน Java?
- **รองรับรูปแบบอย่างครบถ้วน** – PDF417, QR, Code128, และอื่น ๆ อีกมากมาย  
- **การถอดรหัสหลายภาษาที่แม่นยำ** – รองรับจีน, อาหรับ, ซีริลลิก ฯลฯ  
- **ไม่มีการพึ่งพาภายนอก** – เป็น Java แท้ ๆ ทำงานได้บนทุกแพลตฟอร์ม  
- **เอกสารและการสนับสนุนที่ยอดเยี่ยม** – คู่มือเริ่มต้นเร็ว, ฟอรั่ม, และโค้ดตัวอย่าง

## ข้อกำหนดเบื้องต้น

ก่อนจะเริ่มทำตามบทแนะนำนี้ โปรดตรวจสอบว่าคุณมีสิ่งต่อไปนี้แล้ว:

1. **Java Development Kit (JDK)** – ตรวจสอบว่าคุณได้ติดตั้ง JDK เวอร์ชันล่าสุดบนเครื่องของคุณแล้ว  
2. **Aspose.BarCode for Java** – ดาวน์โหลดและติดตั้งไลบรารี Aspose.BarCode จาก [ที่นี่](https://releases.aspose.com/barcode/java/)  
3. **ภาพบาร์โค้ด** – เตรียมภาพบาร์โค้ด PDF417 ตัวอย่างที่มีอักขระจีนสำหรับการทดสอบ

## การนำเข้าแพ็กเกจ

ในโครงการ Java ของคุณ ให้นำเข้าแพ็กเกจที่จำเป็นเพื่อใช้คุณสมบัติของ Aspose.BarCode:

```java
import java.nio.ByteBuffer;
import java.nio.charset.Charset;

import com.aspose.barcode.barcoderecognition.BarCodeReader;
import com.aspose.barcode.barcoderecognition.BarCodeResult;
import com.aspose.barcode.barcoderecognition.DecodeType;
```

## วิธีใช้ Aspose ใน Java สำหรับการจดจำบาร์โค้ด PDF417

### ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีเอกสาร
เริ่มต้นโดยตั้งค่าพาธไปยังไดเรกทอรีทรัพยากรของคุณ:

```java
String dataDir = "Your Document Directory";
```

แทนที่ `"Your Document Directory"` ด้วยพาธที่ชี้ไปยังไดเรกทอรีเอกสารจริงของคุณ

### ขั้นตอนที่ 2: โหลดภาพบาร์โค้ด
ต่อไป โหลดภาพบาร์โค้ดโดยใช้คลาส `BarCodeReader` ซึ่งบอก Aspose ว่าไฟล์ใดที่จะถอดรหัสและสัญลักษณ์ใดที่คาดหวัง:

```java
BarCodeReader reader = new BarCodeReader(dataDir + "barcode.png", DecodeType.PDF_417);
```

แทนที่ `"barcode.png"` ด้วยชื่อไฟล์จริงของภาพบาร์โค้ด PDF417 ของคุณ

### ขั้นตอนที่ 3: อ่านบาร์โค้ด
วนลูปผลลัพธ์บาร์โค้ดและดึงอาร์เรย์ไบต์สำหรับการถอดรหัส โค้ดด้านล่างแสดง **วิธีอ่านภาพบาร์โค้ดใน Java** และแปลงไบต์ดิบให้เป็นข้อความจีนที่อ่านได้:

```java
for (BarCodeResult result : reader.readBarCodes()) {
    byte[] bytes = result.getCodeBytes();
    ByteBuffer bytebuf = ByteBuffer.wrap(bytes);
    System.out.println(Charset.forName("MS936").decode(bytebuf).toString());
}
```

ขั้นตอนนี้จะอ่านบาร์โค้ด ดึงอาร์เรย์ไบต์ออกมา และถอดรหัสด้วย charset ที่ระบุ (`MS936`) ซึ่ง **แสดงอักขระจีนอย่างถูกต้อง**

## ปัญหาที่พบบ่อยและวิธีแก้
- **Charset ไม่ถูกต้อง** – หากผลลัพธ์เป็นข้อความเสียหาย ให้ตรวจสอบว่า charset ตรงกับการเข้ารหัสที่ใช้สร้างบาร์โค้ด (MS936 ใช้สำหรับจีนแบบ Simplified)  
- **ไม่พบไฟล์** – ตรวจสอบให้แน่ใจว่า `dataDir` ชี้ไปยังโฟลเดอร์ที่ถูกต้องและชื่อไฟล์ตรงกันรวมถึงตัวพิมพ์ใหญ่‑เล็ก  
- **ประเภทบาร์โค้ดไม่รองรับ** – ยืนยันว่าคุณใช้ `DecodeType.PDF_417`; ประเภทอื่นต้องใช้ค่า `DecodeType` ที่ต่างกัน
