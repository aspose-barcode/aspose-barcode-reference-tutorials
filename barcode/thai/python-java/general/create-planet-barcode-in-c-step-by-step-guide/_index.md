# สร้าง Planet Barcode ใน C# – คู่มือขั้นตอนโดยละเอียด

เคยต้องการ **สร้าง planet barcode** แต่ไม่แน่ใจว่าจะเริ่มต้นอย่างไรหรือไม่? ในบทแนะนำนี้คุณจะได้เห็นอย่างชัดเจน **วิธีสร้าง planet barcode** ด้วยไลบรารี Aspose.Barcode พร้อมตัวเลือกความสูงเริ่มต้นและกำหนดเอง

เราจะเดินผ่านกระบวนการทั้งหมด—from การติดตั้งแพ็กเกจ NuGet ไปจนถึงการปรับความสูงของบาร์และการบันทึกไฟล์ PNG. เมื่อเสร็จคุณจะมีภาพสี่ภาพพร้อมใช้งาน: Planet barcode สองแบบ (auto และความสูง 100 pixel) และ RM4SCC barcode สองแบบเพื่อเปรียบเทียบ ไม่มีส่วนเกิน เพียงโค้ดที่ใช้งานได้จริงที่คุณสามารถนำไปใส่ในโปรเจกต์ .NET ใดก็ได้

## Prerequisites

ก่อนที่เราจะเริ่มลงมือ ตรวจสอบให้แน่ใจว่าคุณมี:

* .NET 6.0 หรือใหม่กว่า (โค้ดทำงานบน .NET Framework 4.7+ ด้วยเช่นกัน)
* Visual Studio 2022 (หรือ IDE ใดก็ได้ที่คุณชอบ)
* The **Aspose.Barcode for .NET** NuGet package  
  ```bash
  dotnet add package Aspose.Barcode
  ```
* โฟลเดอร์บนดิสก์ที่สามารถเขียนไฟล์ PNG ที่สร้างได้ (แทนที่ `YOUR_DIRECTORY` ในโค้ด)

หากสิ่งใดเหล่านี้ฟังดูแปลกใหม่ ไม่ต้องกังวล—การติดตั้งแพ็กเกจ NuGet ทำได้ง่ายเพียงคลิกขวาที่โปรเจกต์ → *Manage NuGet Packages* → ค้นหา “Aspose.Barcode” → *Install*.

## Step 1 – Create a Minimal Console Project

แรกเริ่ม ให้สร้างแอปคอนโซลขนาดเล็กเพื่อให้เรารันโค้ดบาร์โค้ดแยกออกจากกันได้

```csharp
using System;
using Aspose.BarCode.Generation;

namespace PlanetBarcodeDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            // We'll call helper methods here – see the sections below.
            GeneratePlanetBarcodes();
            GenerateRm4sccBarcodes();
            Console.WriteLine("All barcodes generated successfully.");
        }

        // Helper methods are defined later.
    }
}
```

> **Pro tip:** Keep the `Main` method tidy; delegate the heavy lifting to separate methods. It makes the code easier to test and to read later on.

## Step 2 – **Create Planet Barcode** with Default Height

ตอนนี้เราจะ **สร้าง planet barcode** ด้วยความสูงเริ่มต้น (คำนวณอัตโนมัติ). `XDimension` ควบคุมความกว้างของโมดูลเดียว ส่วนความสูงของบาร์จะปล่อยให้ไลบรารีจัดการ

```csharp
static void GeneratePlanetBarcodes()
{
    // 1️⃣ Default height
    var planetDefault = new BarcodeGenerator(EncodeTypes.Planet, "123456");
    planetDefault.Parameters.Barcode.XDimension.Pixels = 4; // narrow modules
    planetDefault.Save("YOUR_DIRECTORY/PlanetBarHeightAuto.png", BarCodeImageFormat.Png);
}
```

*ทำไมจึงทำงานได้?*  
`EncodeTypes.Planet` บอก Aspose ว่าเราต้องการสัญลักษณ์ Planet, และ `"123456"` คือข้อมูลที่ต้องเข้ารหัส. โดยไม่ตั้งค่า `BarHeight` เอนจินจะเลือกความสูงที่สอดคล้องกับสเปคโดยอัตโนมัติ

### Expected output

เปิด `PlanetBarHeightAuto.png` แล้วคุณจะเห็น Planet barcode ที่สะอาดประมาณ 30 mm สูง (ขนาดจริงขึ้นกับ DPI ของตัวเขียน PNG). ไม่ต้องปรับความสูงด้วยตนเอง

## Step 3 – **How to Generate Planet Barcode** with a Custom 100‑Pixel Height

หากการออกแบบของคุณต้องการความสูงคงที่—เช่น 100 pixel—ให้ตั้งค่า `BarHeight` ก่อนบันทึก

```csharp
static void GeneratePlanetBarcodes()
{
    // 1️⃣ Default height (already shown)
    var planetDefault = new BarcodeGenerator(EncodeTypes.Planet, "123456");
    planetDefault.Parameters.Barcode.XDimension.Pixels = 4;
    planetDefault.Save("YOUR_DIRECTORY/PlanetBarHeightAuto.png", BarCodeImageFormat.Png);

    // 2️⃣ Custom 100‑pixel height
    var planetCustom = new BarcodeGenerator(EncodeTypes.Planet, "123456");
    planetCustom.Parameters.Barcode.XDimension.Pixels = 4;
    planetCustom.Parameters.Barcode.BarHeight.Pixels = 100; // explicit height
    planetCustom.Save("YOUR_DIRECTORY/PlanetBarHeight100.png", BarCodeImageFormat.Png);
}
```

สังเกตบรรทัดเพิ่ม `planetCustom.Parameters.Barcode.BarHeight.Pixels = 100;`. บรรทัดนี้บังคับให้บาร์โค้ดยืด (หรือหด) ให้สูงพอดี 100 pixel ไม่ว่าจะข้อมูลยาวเท่าใดก็ตาม

> **Watch out:** การตั้งค่าความสูงที่เล็กเกินไปอาจทำให้บาร์รวมกัน ทำให้บาร์โค้ดอ่านไม่ออก. กฎทั่วไปคือให้ความสูงอย่างน้อยสามเท่าของ X‑dimension.

## Step 4 – Generate RM4SCC Barcodes for Comparison

บางครั้งคุณอาจต้องการเปรียบเทียบ Planet กับสัญลักษณ์ไปรษณีย์อื่น ๆ. RM4SCC เป็นรูปแบบที่พบได้บ่อย; โค้ดด้านล่างทำตามขั้นตอนเดียวกับ Planet

```csharp
static void GenerateRm4sccBarcodes()
{
    // Default height
    var rm4sccDefault = new BarcodeGenerator(EncodeTypes.RM4SCC, "123456");
    rm4sccDefault.Parameters.Barcode.XDimension.Pixels = 4;
    rm4sccDefault.Save("YOUR_DIRECTORY/RM4SCCBarHeightAuto.png", BarCodeImageFormat.Png);

    // Custom 100‑pixel height
    var rm4sccCustom = new BarcodeGenerator(EncodeTypes.RM4SCC, "123456");
    rm4sccCustom.Parameters.Barcode.XDimension.Pixels = 4;
    rm4sccCustom.Parameters.Barcode.BarHeight.Pixels = 100;
    rm4sccCustom.Save("YOUR_DIRECTORY/RM4SCCBarHeight100.png", BarCodeImageFormat.Png);
}
```

การรันโปรแกรมเต็มจะสร้าง **สี่** ไฟล์ PNG:

| File | Symbology | Height |
|------|-----------|--------|
| `PlanetBarHeightAuto.png` | Planet | อัตโนมัติ |
| `PlanetBarHeight100.png` | Planet | 100 px |
| `RM4SCCBarHeightAuto.png` | RM4SCC | อัตโนมัติ |
| `RM4SCCBarHeight100.png` | RM4SCC | 100 px |

คุณสามารถเปิดแต่ละภาพในโปรแกรมดูใดก็ได้เพื่อยืนยันขนาดและคุณภาพภาพ

## Step 5 – Verify the Barcodes Programmatically (Optional)

หากต้องการความมั่นใจเพิ่มเติมว่าบาร์โค้ดที่สร้างสามารถสแกนได้, Aspose ยังมีคอมโพเนนต์ **reader**. นี่คือการตรวจสอบอย่างเร็วว่าข้อมูลที่เข้ารหัสตรงกับสตริงต้นฉบับหรือไม่

```csharp
using Aspose.BarCode.Reader;

static void VerifyBarcodes()
{
    var reader = new BarCodeReader("YOUR_DIRECTORY/PlanetBarHeight100.png", DecodeType.Planet);
    foreach (BarCodeResult result in reader.ReadBarCodes())
    {
        Console.WriteLine($"Decoded value: {result.CodeText}");
    }
}
```

การรัน `VerifyBarcodes()` จะพิมพ์ `Decoded value: 123456`, ยืนยันว่าการตั้งค่าความสูงแบบกำหนดเองไม่ได้ทำให้การเข้ารหัสเสีย

## Common Pitfalls & Tips

| ปัญหา | สาเหตุ | วิธีแก้ |
|-------|--------|----------|
| บาร์โค้ดดูบางเกินไป | X‑dimension ตั้งค่าต่ำเกินไป (เช่น 1 pixel) | เพิ่ม `XDimension.Pixels` เป็น 3‑5 สำหรับเครื่องพิมพ์ส่วนใหญ่ |
| ภาพเบลอ | บันทึกเป็น PNG ด้วย DPI เริ่มต้น (96) บนหน้าจอความละเอียดสูง | ใช้ `ImageOptions` เพื่อตั้งค่า `ResolutionX` และ `ResolutionY` เป็น 300 DPI |
| ความสูงไม่ถูกนำไปใช้ | `BarHeight` ตั้งค่าหลังจากเรียก `Save()` | ตั้งค่า `BarHeight` **ก่อน** `Save()` เสมอ |
| เส้นทางไฟล์ไม่ถูกต้อง | `YOUR_DIRECTORY` ไม่อยู่ | สร้างโฟลเดอร์โดยโปรแกรม: `Directory.CreateDirectory("YOUR_DIRECTORY");` |

## Wrap‑Up

เรา **สร้าง planet barcode** จากศูนย์, สำรวจ **วิธีสร้าง planet barcode** ด้วยความสูงอัตโนมัติและกำหนดเอง, และแม้กระทั่งเปรียบเทียบแบบเคียงข้างกับ RM4SCC ที่เทียบเท่า ตัวอย่างที่สมบูรณ์และทำงานได้อยู่ในไฟล์ `Program.cs` เพียงไฟล์เดียว, คุณจึงสามารถคัดลอก‑วางลงในโปรเจกต์ .NET ใดก็ได้และเริ่มพิมพ์บาร์โค้ดได้ทันที

### สิ่งต่อไปที่ควรทำ

* ลองค่า `XDimension` ต่าง ๆ เพื่อดูว่าความกว้างของโมดูลส่งผลต่อความน่าเชื่อถือของการสแกนอย่างไร  
* ทดลองผลลัพธ์สีโดยตั้งค่า `ImageOptions.ForeColor` และ `BackColor`  
* สำรวจสัญลักษณ์ไปรษณีย์อื่น ๆ (เช่น **Postnet**, **Intelligent Mail**) ด้วยรูปแบบเดียวกัน  

มีคำถามเกี่ยวกับกรณีขอบ—เช่นการสร้าง Planet barcode สำหรับข้อมูล 12‑digit หรือการฝัง PNG ลงใน PDF? แสดงความคิดเห็นได้เลย, เราจะสำรวจสถานการณ์เหล่านั้นร่วมกัน

Happy coding, and may your barcodes always scan on the first try!
