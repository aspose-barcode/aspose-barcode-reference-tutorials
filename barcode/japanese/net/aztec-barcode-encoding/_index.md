## なぜ aztec auto mode を使用するのですか？

`AztecAutoMode` lets the library decide whether to use **Compact**, **FullRange**, or **Rune** mode based on the amount of data you encode. This removes the guesswork and ensures the smallest possible barcode while retaining maximum readability—perfect for responsive UI designs.
