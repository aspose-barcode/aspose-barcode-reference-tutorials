## 常見問答

**Q:** *我可以在開發時不使用授權產生條碼嗎？*  
**A:** 可以，Aspose 提供臨時免費授權供評估與開發使用。正式上線時需購買完整授權。

**Q:** *條碼可以匯出成哪些影像格式？*  
**A:** 支援 PNG、JPEG、BMP、GIF、TIFF，以及 SVG、PDF 等向量格式。

**Q:** *如何從掃描的 PDF 文件中讀取條碼？*  
**A:** 使用 **document barcode recognition** API 讀取 PDF，然後呼叫 `BarCodeReader` 從每頁擷取條碼。

**Q:** *可以變更條碼條紋與背景的顏色嗎？*  
**A:** 當然可以。`BarCodeImage` 類別允許您設定 `BarColor` 與 `BackColor` 為任意 `java.awt.Color` 值。

**Q:** *我能自訂 QR Code 下方的說明文字嗎？*  
**A:** 可以，**barcode text styling** 選項允許您加入說明文字，並選擇字型、大小與對齊方式。
