## 常見問題

**Q: Can I generate a Codabar barcode without a checksum?**  
A: 可以，您可以透過設定 `generator.getParameters().getBarcode().setCodabarChecksumEnabled(false);` 來停用校驗碼，但不建議在正式環境使用。

**Q: Does Aspose.BarCode support custom start/stop characters?**  
A: 當然可以。您可透過 `Codabar` 符號設定指定起始/終止符號（例如 `*` 與 `#`）。

**Q: How do I validate a barcode that was scanned from an image?**  
A: 使用 `BarcodeReader` 解碼影像，然後呼叫 `reader.getCodeText()` 並驗證 `reader.isValidChecksum()`。

**Q: Is there a performance impact when enabling checksum calculation?**  
A: 對於一般使用情境而言，影響可忽略不計；校驗碼計算的時間複雜度為 O(n)，與資料長度成正比。

**Q: Which Java IDEs are compatible with Aspose.BarCode?**  
A: 任何支援 Java 8 以上的 IDE（如 IntelliJ IDEA、Eclipse、NetBeans、VS Code 等）皆可無縫使用。
