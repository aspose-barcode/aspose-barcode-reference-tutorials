## 常見問答

**Q: 如何決定使用哪種 ECC 模式？**  
A: 對於資料量小且錯誤更正需求有限的情況，選擇 ECC 000‑140；若資料較大且需更高可靠性，則使用 ECC 200。Macro 模式會加入額外的資料層。

**Q: 我可以在 DataMatrix 條碼中嵌入自訂文字嗎？**  
A: 可以，使用 `CodeText` 屬性設定延伸代碼文字，然後依需求設定編碼模式。

**Q: 有沒有辦法自動選擇最佳的編碼模式？**  
A: 將 `EncodeMode` 設為 `Auto`，Aspose.BarCode 會評估輸入內容並挑選最適合的模式（ASCII、C40、Bytes 等）。

**Q: 大量條碼批次的效能考量是什麼？**  
A: 重複使用 `BarCodeGenerator` 實例，並在可能的情況下啟用多執行緒。此外，依需求產生適當格式的影像（PNG 具無損品質，JPEG 可減少檔案大小）。

**Q: Aspose.BarCode 是否支援 .NET Core 與 .NET 5/6？**  
A: 當然支援 — 此函式庫完全相容於 .NET Framework、.NET Core 以及最新的 .NET 版本。
