# 條碼產生器教學 – 在 C# 中產生 PDF417 條碼

有沒有想過如何將一串希臘字母轉換成可掃描的 PDF417 條碼？在本 **barcode generator tutorial** 中，我們將逐步說明您需要了解的所有內容，從安裝函式庫到將圖像儲存至磁碟。最後，您將擁有一個完整功能的 C# 程式，能夠 **generates PDF417 barcode** 並具備正確的 ECI 編碼。

如果您曾經嘗試過條碼專案卻因字元集問題卡住了，您並不孤單。好消息是，解決方案只需要幾行程式碼——只要您了解每個設定的原因。

## 您將學到什麼

* 如何設定 Aspose.BarCode for .NET 套件（我們將使用的函式庫）。  
* 產生含希臘字元的 **generate PDF417 barcode** 所需的完整程式碼。  
* 為何啟用 ECI 模式並選擇 ISO‑8859‑7 對非拉丁字母至關重要。  
* 調整影像格式、解析度與錯誤更正等級的技巧。  

不需要事先了解 PDF417；只要具備基本的 C# 應用知識即可。

## 前置條件

* .NET 6.0 或更新版本（範例目標為 .NET 6，但 .NET 5 或 .NET Framework 4.8 亦可使用）。  
* 最新版本的 Visual Studio 或您偏好的任何 C# IDE。  
* 具備網際網路連線以下載 NuGet 套件 `Aspose.BarCode`。

既然基礎已經清楚，讓我們正式進入 **barcode generator tutorial** 的內容。

## 步驟 1：安裝 Aspose.BarCode NuGet 套件

取得可靠的 **barcode generator tutorial** 實作最簡單的方法是使用 Aspose.BarCode for .NET。開啟終端機（或套件管理員主控台）並執行以下指令：

```bash
dotnet add package Aspose.BarCode
```

或者，在 Visual Studio 中右鍵點擊專案 → *Manage NuGet Packages* → 搜尋 “Aspose.BarCode” 並點擊 **Install**。

> **專業提示：** 請使用最新的穩定版；截至 2026 年 7 月，版本為 23.12。這可確保您擁有最新的 ECI 支援。

## 步驟 2：建立新的 Console 專案

如果您尚未有專案，請快速建立一個：

```bash
dotnet new console -n Pdf417Generator
cd Pdf417Generator
```

開啟 `Program.cs`；稍後我們會將其內容替換為完整範例。

## 步驟 3：定義要編碼的文字（希臘字元）

希臘字母超出 ASCII 範圍，因此需要 Unicode 相容的字串。在本 **barcode generator tutorial** 中，我們將使用經典的 “ΑΒΓΔΕ” 序列：

```csharp
// Step 3: Define the text to encode (Greek characters)
string data = "ΑΒΓΔΕ";
```

為什麼不直接使用拉丁文字？因為本教學的目的在於展示 ECI 編碼如何彌合條碼規格與非拉丁字母之間的差距。

## 步驟 4：以 ECI 支援初始化 PDF417 產生器

**barcode generator tutorial** 的核心是 `BarcodeGenerator` 物件。我們會指示它使用 PDF417、啟用 ECI 模式，並選擇希臘專用的字元集 ISO‑8859‑7：

```csharp
using Aspose.BarCode;
using Aspose.BarCode.Generation;

using (var generator = new BarcodeGenerator(EncodeTypes.Pdf417, data))
{
    // Enable ECI mode so the barcode can carry Unicode information
    generator.Parameters.Barcode.Pdf417.EncodeMode = Pdf417EncodeMode.ECI;

    // Set the ECI encoding to ISO‑8859‑7 (Greek)
    generator.Parameters.Barcode.Pdf417.ECIEncoding = ECIEncodings.ISO_8859_7;
```

### 為什麼需要 ECI？

ECI（Extended Channel Interpretation，擴充通道解釋）告訴掃描器後續的位元屬於特定的代碼頁。若未啟用，大多數掃描器會預設使用 ISO‑8859‑1，導致希臘字元變成亂碼。啟用 ECI 可確保條碼在現代裝置上普遍可讀。

## 步驟 5：將條碼儲存為 PNG 圖像

最後，我們將條碼寫入磁碟。PNG 為無失真格式，適用於大多數 UI 場景：

```csharp
    // Step 5: Save the generated barcode as a PNG image
    generator.Save("Pdf417ECI.png", BarCodeImageFormat.Png);
}
```

以上即為完整程式碼。將所有部分組合起來，即可得到完整且可執行的程式。

## 完整可執行範例

以下是完整的原始檔，您可以直接複製貼上至 `Program.cs`。它包含必要的 `using` 指示詞以及 `Main` 方法的包裝：

```csharp
using System;
using Aspose.BarCode;
using Aspose.BarCode.Generation;

namespace Pdf417Generator
{
    class Program
    {
        static void Main(string[] args)
        {
            // Step 1: Define the text to encode (Greek characters)
            string data = "ΑΒΓΔΕ";

            // Step 2: Create a PDF417 barcode generator with the data
            using (var generator = new BarcodeGenerator(EncodeTypes.Pdf417, data))
            {
                // Step 3: Enable ECI mode and set the character set to ISO‑8859‑7 (Greek)
                generator.Parameters.Barcode.Pdf417.EncodeMode = Pdf417EncodeMode.ECI;
                generator.Parameters.Barcode.Pdf417.ECIEncoding = ECIEncodings.ISO_8859_7;

                // Optional: Adjust error correction level (0‑8). Higher = more robust.
                generator.Parameters.Barcode.Pdf417.ErrorCorrectionLevel = 5;

                // Step 4: Save the generated barcode as a PNG image
                string outputPath = "Pdf417ECI.png";
                generator.Save(outputPath, BarCodeImageFormat.Png);

                Console.WriteLine($"PDF417 barcode saved to {outputPath}");
            }
        }
    }
}
```

### 預期輸出

執行程式 (`dotnet run`) 後，會在專案資料夾產生名為 **Pdf417ECI.png** 的檔案。使用任何圖像檢視器開啟，即可看到傳統的 PDF417 堆疊條碼。使用支援 PDF417 的現代智慧手機條碼應用程式掃描，應會顯示希臘字串 “ΑΒΓΔΕ”。

## 常見問題與邊緣案例

| Question | Answer |
|----------|--------|
| **如果需要 UTF‑8 而非 ISO‑8859‑7，該怎麼辦？** | 設定 `generator.Parameters.Barcode.Pdf417.ECIEncoding = ECIEncodings.UTF8;`。ECI 支援 UTF‑8、UTF‑16 以及多種區域代碼頁。 |
| **我可以調整條碼尺寸嗎？** | 可以。調整 `generator.Parameters.ImageWidth` 與 `ImageHeight`，或使用 `Resolution` 來控制 DPI。 |
| **如何加入可讀的說明文字？** | 啟用 `generator.Parameters.Barcode.Pdf417.ShowCodeText = true;`，並透過 `CodeTextParameters` 自訂字型。 |
| **如果掃描器仍顯示亂碼怎麼辦？** | 確認掃描器韌體是否支援 ECI。較舊的裝置可能會忽略 ECI 區段，回退至 ISO‑8859‑1。 |
