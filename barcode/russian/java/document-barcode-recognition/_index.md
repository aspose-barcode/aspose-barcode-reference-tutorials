- **Что делает Aspose.BarCode?** Предоставляет чистый Java API для генерации, обнаружения и распознавания штрихкодов в изображениях и документах.  
- **Какие форматы поддерживаются?** PDF, Microsoft Word, изображения и штрихкоды на основе Unicode.  
- **Нужна ли лицензия?** Бесплатная пробная версия подходит для разработки; коммерческая лицензия требуется для продакшна.  
- **Можно ли извлечь несколько штрихкодов из одного файла?** Да — API возвращает коллекцию обнаруженных штрихкодов в указанном порядке.

## Часто задаваемые вопросы

**Q: Могу ли я использовать Aspose.BarCode в коммерческом проекте?**  
A: Да. Для использования в продакшене требуется действующая лицензия Aspose, а бесплатная пробная версия доступна для оценки.

**Q: Работает ли библиотека с PDF, защищёнными паролем?**  
A: Абсолютно. Вы можете передать пароль PDF при загрузке документа, и обнаружение штрихкодов будет выполнено нормально.

**Q: Какие версии Java поддерживаются?**  
A: API поддерживает Java 8 и выше, включая Java 11, Java 17 и более новые LTS‑выпуски.

**Q: Как улучшить скорость обнаружения при обработке больших пакетов документов?**  
A: Используйте `BarCodeReader` в многопоточном режиме и включите опцию `ReadQuality` для балансировки скорости и точности.

**Q: Что делать, если штрихкод не обнаружен?**  
A: Убедитесь, что симвология штрихкода поддерживается и разрешение изображения соответствует минимальной рекомендации по DPI (300 dpi для большинства 1D штрихкодов).
