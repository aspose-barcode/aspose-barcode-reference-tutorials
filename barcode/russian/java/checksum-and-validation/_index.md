## Часто задаваемые вопросы

**Q: Могу ли я сгенерировать штрих‑код Codabar без контрольной суммы?**  
A: Да, вы можете отключить контрольную сумму, установив `generator.getParameters().getBarcode().setCodabarChecksumEnabled(false);`, но это не рекомендуется для продакшн‑использования.

**Q: Поддерживает ли Aspose.BarCode пользовательские символы начала/конца?**  
A: Абсолютно. Вы можете указать символы начала/конца (например, `*` и `#`) через настройки символогии `Codabar`.

**Q: Как проверить штрих‑код, считанный с изображения?**  
A: Используйте `BarcodeReader` для декодирования изображения, затем вызовите `reader.getCodeText()` и проверьте `reader.isValidChecksum()`.

**Q: Влияет ли включение расчёта контрольной суммы на производительность?**  
A: Нагрузка незначительна для типичных сценариев; расчёт контрольной суммы выполняется за O(n) времени относительно длины данных.

**Q: Какие IDE для Java совместимы с Aspose.BarCode?**  
A: Любая IDE, поддерживающая Java 8 и выше (IntelliJ IDEA, Eclipse, NetBeans, VS Code и др.), работает без проблем.
