# Как создать штрих‑код Code128 с пустыми полосами в Java

## Введение

Интеграция генерации штрих‑кодов в Java‑приложения часто требуется, будь то системы учёта, платформы для билетов или решения точек продаж. С помощью **Aspose.BarCode for Java** вы можете **создавать изображения штрих‑кода Code128** всего в несколько строк кода. В этом руководстве мы пройдём шаги по **генерации изображений штрих‑кода Java**, содержащих пустые (не заполненные) полосы, предоставляя вам полный контроль над визуальным стилем штрих‑кода.

## Быстрые ответы
- **Какая библиотека?** Aspose.BarCode for Java  
- **Основной сценарий использования?** Создание штрих‑кодов Code128 с пустыми полосами для пользовательских UI‑дизайнов  
- **Требования?** Java JDK 8+, Aspose.BarCode JAR, доступ на запись в папку для выходного изображения  
- **Типичное время реализации?** 5‑10 минут для базового примера  
- **Можно ли настроить цвета?** Да — API предоставляет обширные возможности стилизации  

## Что такое штрих‑код Code128?

Code128 — это высокоплотная линейная символьная система штрих‑кодов, кодирующая полный набор символов ASCII. Она широко используется в логистике, доставке и розничной торговле, поскольку может компактно представлять буквенно‑цифровые данные. По умолчанию Aspose.BarCode отображает полосы Code128 как сплошные прямоугольники; установка свойства **FilledBars** в `false` превращает их в пустые (контурные) полосы, что полезно для стилистического брендинга.

## Почему стоит использовать Aspose.BarCode for Java?

- **Богатый набор функций** — поддерживает более 50 типов штрих‑кодов, включая QR, DataMatrix и PDF417.  
- **Отсутствие внешних зависимостей** — один JAR работает на любой Java‑среде.  
- **Тонкая настройка** — вы можете регулировать ширину полос, цвета, шрифты и свойство `FilledBars`.  
- **Кросс‑платформенность** — работает на Windows, Linux и macOS без нативных библиотек.

## Требования

Before you start, make sure you have:

1. **Среда разработки Java** — установлен JDK 8 или новее и настроена переменная `JAVA_HOME`.  
2. **Aspose.BarCode for Java** — скачайте последнюю JAR‑файл со [страницы загрузки](https://releases.aspose.com/barcode/java/).  
3. **Каталог документов** — папка на вашем компьютере, куда будет сохранено сгенерированное изображение штрих‑кода.

## Импорт пакетов

In your Java project, import the classes you’ll need to work with Aspose.BarCode:

```java
import java.io.IOException;
import com.aspose.barcode.BarCodeImageFormat;
import com.aspose.barcode.generation.BarcodeGenerator;
```

Эти импорты предоставляют доступ к генератору штрих‑кодов, перечислению форматов изображений и обработке исключений.

## Шаг 1: Настройка каталога ресурсов

```java
// The path to the resource directory.
String dataDir = "Your Document Directory";
```

Замените `"Your Document Directory"` на абсолютный путь к папке, которую вы создали в разделе требований. Здесь будет записан файл PNG.

## Шаг 2: Создание экземпляра генератора штрих‑кода Code128

```java
// Create an instance of BarcodeGenerator and initialize it with CodeText and Symbology
BarcodeGenerator generator = new BarcodeGenerator(com.aspose.barcode.EncodeTypes.CODE_128, "TEXT");
```

Здесь мы создаём экземпляр `BarcodeGenerator` с символьностью **CODE_128** и примерным текстом `"TEXT"`. Это **пример генератора штрих‑кода Java**, который можно адаптировать под любую строку, которую нужно закодировать.

## Шаг 3: Установка свойства FilledBars в False

```java
// Set the FilledBars property to false
generator.getParameters().getBarcode().setFilledBars(false);
```

Установив `FilledBars` в `false`, сгенерированный штрих‑код будет отображать пустые полосы вместо сплошных, достигая желаемого визуального эффекта.

## Шаг 4: Сохранение изображения штрих‑кода

```java
// Save the resultant barcode image on disk
generator.save(dataDir + "barcodeWithEmptyBars.png", BarCodeImageFormat.PNG);
```

Метод `save` записывает штрих‑код в указанный каталог в формате PNG. При необходимости можно изменить `BarCodeImageFormat.PNG` на `JPG`, `BMP` или `TIFF`.

## Распространённые сценарии использования

- **Кастомный брендинг** — пустые полосы можно накладывать на фоновые узоры или цвета.  
- **Готовые к печати ресурсы** — генерировать PNG высокого разрешения для включения в счета или транспортные этикетки.  
- **Динамическая генерация** — создавать штрих‑коды «на лету» в веб‑службах или настольных приложениях.

## Советы по устранению неполадок

- **Недопустимый путь** — убедитесь, что `dataDir` заканчивается разделителем файлов (`/` или `\\`) или объединяйте с помощью `File.separator`.  
- **Требуется лицензия** — пробная версия подходит для оценки, но для продакшна нужна полная лицензия.  
- **Неподдерживаемые символы** — Code128 поддерживает полный набор ASCII; избегайте Unicode‑символов, если только не переключаетесь на другую символьность.

## Заключение

Теперь вы знаете, как **создавать изображения штрих‑кода Code128** с пустыми полосами с помощью Aspose.BarCode for Java. Этот простой подход позволяет быстро генерировать настраиваемую графику штрих‑кодов, будь то настольный инструмент учёта или облачный API.

**Дополнительные вопросы**

**Q: Как сгенерировать изображение штрих‑кода в другом формате, например JPEG?**  
A: Измените второй аргумент метода `save` на `BarCodeImageFormat.JPEG`.

**Q: Могу ли я генерировать несколько штрих‑кодов в цикле?**  
A: Да — создайте новый `BarcodeGenerator` внутри цикла или переиспользуйте тот же экземпляр с новым `CodeText`.
