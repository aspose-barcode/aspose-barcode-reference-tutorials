# Как создать изображение штрих‑кода с пустыми полосами в Java

## Введение

В современных Java‑приложениях возможность **создавать изображения штрих‑кода** «на лету» является ценным функционалом для управления запасами, систем билетов и многих других бизнес‑сценариев. Aspose.BarCode for Java предоставляет мощный, простой в использовании API, позволяющий генерировать штрих‑коды высокого качества всего в несколько строк кода. В этом руководстве мы пошагово рассмотрим процесс **создания штрих‑кода**, который выводит пустые (не заполненные) полосы — идеальное решение для дизайнов с полой визуализацией.

## Быстрые ответы
- **Что означает «пустые полосы»?** Полосы отображаются без стандартного сплошного заполнения, создавая полый вид.  
- **Какой тип штрих‑кода используется?** Code 128 (широко поддерживаемая линейная символьная система).  
- **Нужна ли лицензия для пробного использования?** Доступна бесплатная пробная версия; для использования в продакшене требуется лицензия.  
- **Какие форматы вывода поддерживаются?** PNG, JPEG, BMP, GIF, TIFF и другие.  
- **Совместим ли он со всеми версиями Java?** Да, библиотека работает с Java 8 и новее.

## Что такое создание изображения штрих‑кода?
Создание изображения штрих‑кода означает преобразование строки данных в визуальное представление, которое могут считывать сканеры. Полученное изображение можно сохранить в PNG, JPEG или других популярных форматах и внедрить в PDF, напечатать на этикетках или отобразить в пользовательском интерфейсе.

## Почему стоит использовать библиотеку Aspose.BarCode Java?
- **Богатый набор функций** — поддерживает более 50 символьных систем, включая Code 128, QR, DataMatrix и другие.  
- **Тонкая настройка** — свойства такие как `FilledBars`, цвета, отступы и размер изображения легко конфигурируются.  
- **Отсутствие внешних зависимостей** — один JAR‑файл покрывает всё, упрощая развертывание.  
- **Полная документация** — примеры, справочник API и форумы помогут быстро освоиться.

## Предварительные требования

Прежде чем приступить к генерации штрих‑кода, убедитесь, что у вас есть следующие условия:

1. **Среда разработки Java** — установлена Java 8+ и совместимая IDE или система сборки.  
2. **Aspose.BarCode for Java** — скачайте и установите библиотеку Aspose.BarCode for Java со [страницы загрузки](https://releases.aspose.com/barcode/java/).  
3. **Каталог документов** — создайте каталог в системе для хранения сгенерированного изображения штрих‑кода.

## Импорт пакетов

В вашем Java‑проекте импортируйте необходимые пакеты для работы с Aspose.BarCode:

```java
import java.io.IOException;
import com.aspose.barcode.BarCodeImageFormat;
import com.aspose.barcode.generation.BarcodeGenerator;
```

## Как создать изображение штрих‑кода с пустыми полосами

### Шаг 1: Настройка каталога ресурсов

```java
// The path to the resource directory.
String dataDir = "Your Document Directory";
```

Замените `"Your Document Directory"` на реальный путь, где вы хотите сохранить выходной файл.

### Шаг 2: Создание экземпляра BarcodeGenerator (Code128 Barcode Java)

```java
// Create an instance of BarcodeGenerator and initialize it with CodeText and Symbology
BarcodeGenerator generator = new BarcodeGenerator(com.aspose.barcode.EncodeTypes.CODE_128, "TEXT");
```

Здесь мы используем символьную систему **Code 128** — одну из самых популярных линейных штрих‑кодов, делая это классическим примером **code128 barcode java**.

### Шаг 3: Установка свойства FilledBars в False

```java
// Set the FilledBars property to false
generator.getParameters().getBarcode().setFilledBars(false);
```

Установка `FilledBars` в `false` заставляет Aspose.BarCode отрисовывать полосы как пустые (полые) формы вместо сплошных блоков.

### Шаг 4: Сохранение изображения штрих‑кода

```java
// Save the resultant barcode image on disk
generator.save(dataDir + "barcodeWithEmptyBars.png", BarCodeImageFormat.PNG);
```

Изображение сохраняется в формате PNG, но вы можете изменить перечисление `BarCodeImageFormat`, чтобы получить JPEG, BMP или другие поддерживаемые типы.

Повторите эти шаги в вашем Java‑приложении, чтобы без труда **создавать изображения штрих‑кода** с пустыми полосами, используя Aspose.BarCode for Java.
