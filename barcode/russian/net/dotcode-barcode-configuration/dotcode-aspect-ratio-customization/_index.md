# Как сгенерировать изображение штрихкода с пользовательским соотношением сторон DotCode с помощью Aspose.BarCode для .NET

## Introduction

Если вы разработчик .NET и вам нужно **генерировать изображения штрихкода**, которые соответствуют определённому макету, Aspose.BarCode для .NET делает это без усилий. Одна из самых мощных возможностей — настройка соотношения сторон DotCode — идеально подходит для этикеток в здравоохранении, почтовых бирках или любой упаковки, где пространство ограничено. В этом руководстве мы пройдём весь процесс, от настройки проекта до сохранения конечного изображения.

## Quick Answers
- **Что контролирует «соотношение сторон»?** Оно определяет пропорцию высоты к ширине каждого модуля DotCode.  
- **Зачем его настраивать?** Чтобы разместить штрихкод в узких пространствах или соответствовать требованиям бренда без потери читаемости.  
- **Нужна ли лицензия?** Бесплатная пробная версия подходит для разработки; коммерческая лицензия требуется для продакшна.  
- **Какие версии .NET поддерживаются?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.  
- **Сколько времени это занимает?** Менее пяти минут для генерации пользовательского изображения штрихкода.

## What is a DotCode barcode?

Что такое штрихкод DotCode?

DotCode — это высокоплотный двумерный штрихкод, способный хранить до 1500 символов. Он широко используется в отраслях, требующих компактного и устойчивого к ошибкам кодирования данных — например, в медицинских устройствах, сортировке почты и учёте запасов.

## Why customize the aspect ratio?

Почему настраивать соотношение сторон?

Настройка соотношения сторон позволяет вам:
* Разместить штрихкод в ограниченных размерах этикетки.  
* Согласовать штрихкод с существующими сетками дизайна.  
* Оптимизировать работу сканера для определённых разрешений принтера.

## Import Namespaces

Сначала импортируйте пространство имён, содержащее классы генерации штрихкода:

```csharp
using Aspose.BarCode.Generation;
```

## How to generate barcode image with custom DotCode aspect ratio

Ниже представлено пошаговое руководство. Каждый шаг включает краткое объяснение и точный код, который необходимо скопировать.

### Step 1: Initialize the Barcode Generator

```csharp
using (BarcodeGenerator gen = new BarcodeGenerator(EncodeTypes.DotCode, "Aspose"))
{
    // Subsequent configuration goes here
}
```

Мы создаём экземпляр `BarcodeGenerator`, указываем `EncodeTypes.DotCode` и передаём строку данных `"Aspose"`, которая будет закодирована.

### Step 2: Set the X‑Dimension (size) of the barcode

```csharp
gen.Parameters.Barcode.XDimension.Pixels = 10;
```

X‑размер контролирует ширину каждого модуля. Отрегулируйте значение в пикселях, чтобы увеличить или уменьшить весь штрихкод.

### Step 3: Customize the Aspect Ratio

```csharp
gen.Parameters.Barcode.DotCode.AspectRatio = 0.5f;
```

Здесь мы задаём соотношение сторон **0.5** (высота в половину ширины). Не стесняйтесь экспериментировать со значениями от **0.2f** до **1.0f**, чтобы соответствовать ограничениям вашего макета.

### Step 4: Save the generated barcode image

```csharp
gen.Save($"{path}DotCodeAspectRatio0.5.png", BarCodeImageFormat.Png);
```

Замените `{path}` на папку, подготовленную ранее. Изображение сохраняется в формате PNG, готовое к встраиванию в отчёты, печати на этикетках или отображению в пользовательском интерфейсе.

## Common Issues & Tips

| Проблема | Решение |
|----------|----------|
| **Штрихкод выглядит размытым** | Увеличьте значение `XDimension.Pixels` или сохраните в формате более высокого разрешения (например, BMP). |
| **Сканер не может прочитать** | Убедитесь, что соотношение сторон не слишком экстремально; держите его ≥ 0.2. |
| **Ошибка «Путь не найден»** | Убедитесь, что каталог существует и приложение имеет права записи. |
| **Исключение лицензии** | Используйте пробную лицензию для разработки; примените коммерческую лицензию перед развертыванием. |

**Вопрос: Могу ли я генерировать изображение штрихкода в форматах, отличных от PNG?**  
Ответ: Конечно. Метод `Save` поддерживает BMP, JPEG, GIF, TIFF и другие — просто измените значение перечисления `BarCodeImageFormat`.

**Вопрос: Как изменить цвет переднего плана штрихкода?**  
Ответ: Используйте `gen.Parameters.Barcode.BarcodeColor = System.Drawing.Color.Blue;` перед вызовом `Save`.

**Вопрос: Можно ли добавить читаемую подпись под штрихкодом?**  
Ответ: Да. Установите `gen.Parameters.Barcode.CodeLocation = CodeLocation.Below;` и при необходимости настройте параметры `gen.Parameters.Barcode.Font`.

**Вопрос: Влияет ли соотношение сторон на коррекцию ошибок?**  
Ответ: Уровень коррекции ошибок остаётся неизменным; изменяется только визуальная форма каждого модуля.

**Вопрос: Могу ли я генерировать несколько штрихкодов в цикле с разными соотношениями сторон?**  
Ответ: Конечно. Разместите код конфигурации внутри цикла `foreach` и изменяйте `AspectRatio` для каждой итерации.
