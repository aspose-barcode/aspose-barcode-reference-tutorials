# Настройка строк и столбцов Codablock F в Aspose.BarCode для .NET

В этом пошаговом руководстве вы **create barcode image c#** путем настройки параметров строк и столбцов Codablock F с использованием Aspose.BarCode для .NET. Codablock F — универсальная 2D‑символика штрих‑кода, часто используемая для **generate shipping label barcode** изображений в логистике, упаковке и учёте запасов. Мы пройдём каждый пример, объясним, почему важна каждая настройка, и покажем, как **set barcode size** в соответствии с требованиями вашей этикетки.

## Импорт пространств имён

Начните с импорта необходимого пространства имён в ваш проект C#. Это даст вам доступ к классам генерации штрих‑кода.

```csharp
using Aspose.BarCode.Generation;
```

## Шаг 1: Инициализация `BarcodeGenerator`

Мы создаём экземпляр `BarcodeGenerator`, указываем, что хотим штрих‑код Codablock F, и передаём данные для кодирования.

```csharp
string path = "Your Directory Path";
System.Console.WriteLine("CodablockFRowCol:");

BarcodeGenerator gen = new BarcodeGenerator(EncodeTypes.CodablockF, "Aspose.Barcode");
```

> **Совет:** Держите переменную `path` указывающей на папку с правом записи; иначе `Save` выбросит исключение.

## Шаг 2: Установка столбцов Codablock F

Если нужен более широкий штрих‑код, увеличьте количество столбцов. Здесь мы задаём 4 столбца и позволяем библиотеке автоматически определить количество строк.

```csharp
// Set CodablockF columns to 4
gen.Parameters.Barcode.Codablock.Columns = 4;
gen.Parameters.Barcode.Codablock.Rows = 0;
gen.Save($"{path}CodablockFCol4.png", BarCodeImageFormat.Png);
```

## Шаг 3: Установка строк Codablock F

Для более высокого штрих‑кода (полезно, когда горизонтальное пространство ограничено) задайте количество строк. Этот пример создаёт штрих‑код из 4 строк.

```csharp
// Set CodablockF rows to 4
gen.Parameters.Barcode.Codablock.Columns = 0;
gen.Parameters.Barcode.Codablock.Rows = 4;
gen.Save($"{path}CodablockFRow4.png", BarCodeImageFormat.Png);
```

## Шаг 4: Установка одновременно столбцов и строк

Когда требуется точный контроль над размерами штрих‑кода, укажите оба значения. Следующий код создаёт штрих‑код с 4 столбцами и 6 строками.

```csharp
// Set CodablockF columns to 4 and rows to 6
gen.Parameters.Barcode.Codablock.Columns = 4;
gen.Parameters.Barcode.Codablock.Rows = 6;
gen.Save($"{path}CodablockFRow6Col4.png", BarCodeImageFormat.Png);
```

## Как установить размер штрих‑кода для транспортных этикеток

Свойства `Columns` и `Rows` фактически определяют размер штрих‑кода. Если нужна конкретная размерность в пикселях, можно также отрегулировать `ImageWidth` и `ImageHeight` в `gen.Parameters.Image`. Комбинируя эти настройки, вы сможете **generate shipping label barcode** изображения, соответствующие спецификациям перевозчика.
