# Создание изображения штрихкода – Одномерный Code 93 с Aspose.BarCode

## Введение

В современную цифровую эпоху штрихкоды стали неотъемлемой частью нашей жизни, упрощая такие процессы, как управление запасами, маркировка продуктов и отслеживание продаж в точках продаж. **Создание изображения штрихкода** часто является первым шагом в интеграции этих идентификаторов в ваши приложения. Aspose.BarCode for .NET предоставляет надёжный, простой в использовании API, позволяющий создавать высококачественные штрихкоды Code 93 всего за несколько строк кода C#. Независимо от того, разрабатываете ли вы систему склада, розничное приложение или кастомный инструмент отчётности, этот учебник проведёт вас через полный **aspose barcode example**, показывающий, как генерировать, настраивать и **save barcode PNG** файлы.

## Быстрые ответы
- **Что покрывает учебник?** Создание и сохранение изображения штрихкода Code 93 с обработкой контрольной суммы.  
- **Какая библиотека используется?** Aspose.BarCode for .NET (последний стабильный релиз).  
- **Нужна ли лицензия?** Бесплатная пробная версия подходит для разработки; коммерческая лицензия требуется для продакшна.  
- **Можно ли изменить формат вывода?** Да — вы можете сохранять как PNG, JPEG, BMP и т.д., изменив `BarCodeImageFormat`.  
- **Каковы минимальные требования?** .NET Framework 4.6+ или .NET Core 3.1+ и Visual Studio.

## Что такое штрихкод Code 93?
Code 93 — это высокоплотная алфавитно-цифровая символьная система, поддерживающая полный набор символов ASCII. Часто выбирается из‑за компактного размера и встроенной контрольной суммы, помогающей обеспечить целостность данных при сканировании.

## Почему генерировать изображения штрихкодов с помощью Aspose.BarCode?

- **Полная документация** и большой набор примеров, ускоряющих разработку.

## Импорт пространств имён

Сначала импортируйте необходимые пространства имён, чтобы получить доступ к классам генерации штрихкода.

```csharp
using Aspose.BarCode;
using Aspose.BarCode.Generation;
```

## Шаг 1: Установите путь к каталогу

Укажите, где будет сохраняться сгенерированное изображение штрихкода. Замените заполнитель на действительный каталог на вашем компьютере.

```csharp
string path = "Your Directory Path";
```

## Шаг 2: Создайте односторонний Code 93 штрихкод

Создайте экземпляр `BarcodeGenerator` с типом `Code93Extended` и данными, которые вы хотите закодировать.

```csharp
BarcodeGenerator gen = new BarcodeGenerator(EncodeTypes.Code93Extended, "CODE");
```

## Шаг 3: Включите контрольную сумму (по желанию)

Code 93 включает контрольную сумму по умолчанию. Вы можете переключить её с помощью свойства `IsChecksumEnabled`.

```csharp
gen.Parameters.Barcode.IsChecksumEnabled = EnableChecksum.Yes;
```

## Шаг 4: Сохраните изображение штрихкода (Save Barcode PNG)

Сгенерируйте изображение и сохраните его как PNG‑файл в указанный ранее каталог.

```csharp
gen.Save($"{path}OneCSCode93WithChecksum.png", BarCodeImageFormat.Png);
```

## Шаг 5: Обработка исключений — генерация без контрольной суммы

Если вам нужно создать штрихкод **без** контрольной суммы, необходимо обработать потенциальные исключения, которые могут возникнуть.

```csharp
try
{
    gen.Parameters.Barcode.IsChecksumEnabled = EnableChecksum.No;
    gen.GenerateBarCodeImage();
}
catch (Exception e)
{
    Console.WriteLine(e.Message);
}
```

### Распространённые проблемы и решения
- **Неверный путь** — убедитесь, что каталог существует и приложение имеет права на запись.  
- **Неподдерживаемые символы** — Code 93 поддерживает полный набор ASCII, но управляющие символы могут вызвать ошибки.  
- **Лицензия не установлена** — без действующей лицензии библиотека работает в режиме оценки и может добавить водяной знак.
