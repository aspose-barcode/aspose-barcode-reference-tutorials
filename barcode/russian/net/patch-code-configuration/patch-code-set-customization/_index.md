# Создание нескольких штрихкодов – Настройка набора Patch Code

В этом руководстве вы **создадите несколько штрихкодов** с помощью Aspose.BarCode for .NET, сосредоточив внимание на семействе Patch Code. Независимо от того, разрабатываете ли вы систему управления документами или вам нужно маркировать активы, генерация нескольких изображений штрихкодов одновременно экономит время и уменьшает количество ошибок. Мы пройдемся по требованиям, импортируем необходимые пространства имён, а затем покажем пошаговый пример, который позволяет **настраивать значения штрихкода Patch** и **сохранять файлы PNG штрихкода** на диск.

## Быстрые ответы
- **Что покрывает это руководство?** Создание нескольких штрихкодов Patch Code, настройка их текста и сохранение в виде PNG‑изображений.  
- **Какая библиотека используется?** Aspose.BarCode for .NET.  
- **Нужна ли лицензия?** Бесплатная пробная версия подходит для тестирования; для продакшна требуется коммерческая лицензия.  
- **Какие версии .NET поддерживаются?** .NET Framework 4.5+ и .NET Core/5/6+.  
- **Сколько штрихкодов можно сгенерировать?** Любое количество – просто измените свойство `CodeText` и вызовите `Save` для каждого изображения.

## Что означает «создание нескольких штрихкодов» с Patch Code?
Штрихкоды Patch Code – это компактная, высокоплотная символьная система, часто используемая для отслеживания документов. Изменяя свойство `CodeText` у одного экземпляра `BarcodeGenerator`, вы можете **создавать несколько штрихкодов** в цикле или последовательности операторов, каждый из которых сохраняется как отдельный PNG‑файл.

## Почему стоит использовать Aspose.BarCode .NET для генерации изображений штрихкодов?
- **Полнофункциональное API** – поддерживает десятки символьных систем, включая Patch Code.  
- **Без внешних зависимостей** – чистая .NET‑библиотека, легко интегрируется.  
- **Широкие возможности настройки** – цвета, шрифты, размеры и форматы изображений полностью конфигурируемы.  
- **Надёжный результат** – генерирует чёткие, сканируемые изображения, подходящие для производства.

## Как создать несколько штрихкодов – Пошаговое руководство

### Шаг 1: Указание пути к каталогу
Задайте путь к каталогу, куда будут сохраняться сгенерированные изображения штрихкодов. Замените `"Your Directory Path"` на желаемое расположение папки.

```csharp
string path = "Your Directory Path";
```

### Шаг 2: Инициализация генератора штрихкодов
Мы будем использовать класс `BarcodeGenerator` для создания штрихкодов Patch Code. Инициализируйте генератор, указав тип штрихкода и начальный текст кода:

```csharp
BarcodeGenerator gen = new BarcodeGenerator(EncodeTypes.PatchCode, "Patch I");
```

### Шаг 3: Настройка параметров текста кода
Можно настроить параметры текста штрихкода. Здесь мы задаём размер шрифта 20 пикселей, чтобы текст был чётко читаемым:

```csharp
gen.Parameters.Barcode.CodeTextParameters.FontMode = FontMode.Manual;
gen.Parameters.Barcode.CodeTextParameters.Font.Size.Pixels = 20;
```

### Шаг 4: Генерация и сохранение штрихкодов
Теперь меняем свойство `CodeText` для каждого варианта и **сохраняем файлы PNG штрихкода**. Это та часть, где мы действительно **создаём несколько штрихкодов** за один запуск:

```csharp
// Patch I
gen.CodeText = "Patch I";
gen.Save($"{path}PatchCodeI.png", BarCodeImageFormat.Png);

// Patch II
gen.CodeText = "Patch II";
gen.Save($"{path}PatchCodeII.png", BarCodeImageFormat.Png);

// Patch III
gen.CodeText = "Patch III";
gen.Save(`${path}PatchCodeIII.png`, BarCodeImageFormat.Png);

// Patch IV
gen.CodeText = "Patch IV";
gen.Save(`${path}PatchCodeIV.png`, BarCodeImageFormat.Png);

// Patch T
gen.CodeText = "Patch T";
gen.Save(`${path}PatchCodeT.png`, BarCodeImageFormat.Png);

// Patch VI
gen.CodeText = "Patch VI";
gen.Save(`${path}PatchCodeVI.png`, BarCodeImageFormat.Png);
```

> **Совет:** Если нужно сгенерировать десятки штрихкодов Patch Code, оберните последний блок в цикл `foreach`, который будет проходить по коллекции строк кода.

Поздравляем! Вы успешно **создали несколько штрихкодов** с помощью Aspose.BarCode for .NET. Та же схема работает для любой другой поддерживаемой символьной системы — просто замените `EncodeTypes.PatchCode` на нужный тип.

## Распространённые ошибки и их устранение

| Симптом | Возможная причина | Решение |
|---------|-------------------|---------|
| PNG‑файлы пустые | Неправильный путь к папке вывода или отсутствует завершающий слеш | Убедитесь, что `path` заканчивается обратным слешем (`\\`) или используйте `Path.Combine`. |
| Штрихкод выглядит размытым | Формат изображения установлен с низким DPI | Увеличьте `gen.Parameters.ImageResolution` перед сохранением. |
| Текст обрезан | Размер шрифта слишком велик для размеров штрихкода | Уменьшите `Font.Size.Pixels` или увеличьте размеры штрихкода через `gen.Parameters.ImageSize`. |
