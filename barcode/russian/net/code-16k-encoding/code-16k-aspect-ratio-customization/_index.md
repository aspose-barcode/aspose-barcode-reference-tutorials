## Импорт пространств имён

Для начала необходимо импортировать требуемые пространства имён, чтобы получить доступ к функционалу, предоставляемому Aspose.BarCode for .NET. Вот как это сделать:

## Шаг 1: Определите путь к каталогу

Перед созданием штрих‑кодов укажите путь к каталогу, где вы хотите сохранять сгенерированные изображения. Вы можете сделать это, задав переменную `path` в вашем коде.

```csharp
string path = "Your Directory Path";
```

Убедитесь, что заменили `"Your Directory Path"` на фактический путь в вашей системе.

## Шаг 2: Создайте штрих‑код Code16K с заданным соотношением сторон

BarCodeGenerator — основной класс, используемый для создания штрих‑кодов. Он предоставляет свойства для символьного набора, размеров и формата вывода.

```csharp
System.Console.WriteLine("Code16K Aspect Ratio:");

BarcodeGenerator gen = new BarcodeGenerator(EncodeTypes.Code16K, "Aspose.BarCode");

// Set the X-dimension (width of the barcode bars) in pixels
gen.Parameters.Barcode.XDimension.Pixels = 2;

// Set Code 16K aspect ratio to 10
gen.Parameters.Barcode.Code16K.AspectRatio = 10;
gen.Save($"{path}Code16KAspectRatio10.png", BarCodeImageFormat.Png);

// Set Code 16K aspect ratio to 20
gen.Parameters.Barcode.Code16K.AspectRatio = 20;
gen.Save($"{path}Code16KAspectRatio20.png", BarCodeImageFormat.Png);
```

Код инициализирует генератор штрих‑кодов, задаёт X‑dimension (ширину полос) равной 2 пикселям, а затем генерирует штрих‑коды Code 16K с соотношениями сторон 10 и 20. Полученные изображения сохраняются в указанный вами каталог.

## Как создать пользовательский штрих‑код .net с Code 16K?

Свойство AspectRatio управляет масштабированием ширины к высоте штрих‑кода Code 16K. Установите его в нужное целочисленное значение перед сохранением изображения.

Загрузите библиотеку Aspose.BarCode, настройте символьный набор `Code16K`, задайте свойство `AspectRatio` и вызовите `Save` для записи файла изображения. ## Распространённые ошибки и советы

- **Ограничения соотношения сторон**: Чрезвычайно высокие значения могут привести к слишком тонким полоскам, которые сложно сканировать надёжно. Тестируйте с вашим целевым сканером.
- **Формат изображения**: PNG хорошо подходит для большинства случаев, но вы также можете экспортировать в JPEG или BMP, изменив перечисление `BarCodeImageFormat`.
- **Формат пути**: Убедитесь, что путь к каталогу заканчивается слешем (`\` или `/`), соответствующим вашей ОС, иначе вызов `Save` может завершиться ошибкой.
