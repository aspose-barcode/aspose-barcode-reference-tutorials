## Что такое Aztec Code?

Aztec Code — это двумерный штрих‑код, способный хранить большие объёмы данных в компактном квадратном шаблоне. В отличие от QR‑кодов, он не требует окружающей «тихой зоны», что делает его идеальным для ограниченного пространства.

## Почему использовать Aspose.BarCode для .NET при создании aztec‑code?

- **Полный контроль** над параметрами кодирования (набор символов, коррекция ошибок, размер).  
- **Надёжный движок распознавания**, который читает Aztec‑коды из изображений, потоков или веб‑камер.  
- **Кросс‑платформенная** поддержка .NET Framework, .NET Core и .NET 5/6.  
- **Без внешних зависимостей** — всё работает в управляемом коде.

## Импорт пространств имён

Сначала вам нужно импортировать необходимые пространства имён для использования Aspose.BarCode for .NET в вашем C#‑приложении. Добавьте следующий код в начало вашего файла `.cs`:

```csharp
using System;
using System.Text;
using Aspose.BarCode.Generation;
using Aspose.BarCode.BarCodeRecognition;
```

## Как создать aztec‑code с помощью Aspose.BarCode для .NET

### Шаг 1: Укажите путь к каталогу

Установите путь, куда вы хотите сохранить сгенерированное изображение Aztec‑code. Замените `"Your Directory Path"` на желаемый путь к каталогу.

```csharp
string path = "Your Directory Path";
```

### Шаг 2: Инициализировать генератор Aztec‑code

Создайте экземпляр `BarcodeGenerator` с параметром `EncodeTypes`, установленным в Aztec, и укажите текст, который нужно закодировать.

```csharp
BarcodeGenerator gen = new BarcodeGenerator(EncodeTypes.Aztec, "Aspose常に先を行く");
```

### Шаг 3: Установить параметры штрих‑кода

Настройте параметры штрих‑кода. В этом примере мы задаём XDimension в пикселях и кодировку текста кода в UTF‑8.

```csharp
gen.Parameters.Barcode.XDimension.Pixels = 4;
gen.Parameters.Barcode.Aztec.CodeTextEncoding = Encoding.UTF8;
```

### Шаг 4: Сохранить изображение Aztec‑code

Сохраните сгенерированное изображение Aztec‑code в указанный каталог.

```csharp
gen.Save($"{path}AztecCodeTextEncoding.png", BarCodeImageFormat.Png);
```

### Шаг 5: Распознать Aztec‑code

Попробуйте распознать только что созданный Aztec‑code. Для этого мы используем `BarCodeReader`.

```csharp
BarCodeReader read = new BarCodeReader(gen.GenerateBarCodeImage(), DecodeType.Aztec);
foreach (BarCodeResult result in read.ReadBarCodes())
    Console.WriteLine("AztecCodeTextEncoding:" + result.GetCodeText(Encoding.UTF8));
```

## Распространённые ошибки и советы

- **Проблемы с путём к файлу** – Убедитесь, что переменная `path` заканчивается разделителем каталогов (`\` или `/`), соответствующим вашей ОС.  
- **Несоответствие кодировки** – Всегда задавайте `CodeTextEncoding`, соответствующую набору символов исходного текста; иначе вывод может быть искажён.  
- **Исключения лицензии** – Без действующей лицензии сгенерированное изображение может содержать водяной знак.
