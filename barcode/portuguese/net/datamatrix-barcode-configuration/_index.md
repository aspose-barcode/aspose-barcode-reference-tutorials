Usando Aspose.BarCode, você pode **generate datamatrix barcode** imagens, PDFs ou SVGs diretamente de suas aplicações .NET, proporcionando controle total sobre tamanho, correção de erros e modo de codificação.

## Perguntas Frequentes

**Q: Como decido qual modo ECC usar?**  
A: Escolha ECC 000‑140 para conjuntos de dados pequenos com correção de erros limitada, ou ECC 200 para dados maiores e maior confiabilidade. O modo macro adiciona camadas de dados adicionais.

**Q: Posso incorporar texto personalizado em um código de barras DataMatrix?**  
A: Sim, use a propriedade `CodeText` para definir o texto de código estendido, então configure o modo de codificação conforme necessário.

**Q: Existe uma forma de selecionar automaticamente o melhor modo de codificação?**  
A: Defina `EncodeMode` como `Auto` e o Aspose.BarCode avaliará a entrada e escolherá o modo ideal (ASCII, C40, Bytes, etc.).

**Q: Quais são as considerações de desempenho para lotes grandes de códigos de barras?**  
A: Reutilize a instância `BarCodeGenerator` e habilite multithreading onde possível. Além disso, gere imagens em um formato adequado (PNG para sem perdas, JPEG para tamanho menor).

**Q: O Aspose.BarCode suporta .NET Core e .NET 5/6?**  
A: Absolutamente – a biblioteca é totalmente compatível com .NET Framework, .NET Core e as versões mais recentes do .NET.
