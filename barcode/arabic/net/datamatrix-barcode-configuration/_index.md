## الأسئلة المتكررة

**س: كيف أقرر أي وضع ECC أستخدم؟**  
ج: اختر ECC 000‑140 لمجموعات بيانات صغيرة مع تصحيح أخطاء محدود، أو ECC 200 لبيانات أكبر وموثوقية أعلى. يضيف وضع الماكرو طبقات بيانات إضافية.

**س: هل يمكنني تضمين نص مخصص في باركود DataMatrix؟**  
ج: نعم، استخدم الخاصية `CodeText` لتعيين نص الرمز الموسع، ثم قم بتكوين وضع الترميز حسب الحاجة.

**س: هل هناك طريقة لاختيار وضع الترميز الأنسب تلقائيًا؟**  
ج: اضبط `EncodeMode` إلى `Auto` وسيتولى Aspose.BarCode تقييم الإدخال واختيار الوضع المثالي (ASCII، C40، Bytes، إلخ).

**س: ما هي الاعتبارات المتعلقة بالأداء عند معالجة دفعات باركود كبيرة؟**  
ج: أعد استخدام كائن `BarCodeGenerator`، وفعل المعالجة المتعددة الخيوط حيثما أمكن. كذلك، أنشئ الصور بصيغة مناسبة (PNG للغية، JPEG للحجم الأصغر).

**س: هل يدعم Aspose.BarCode .NET Core و .NET 5/6؟**  
ج: بالتأكيد – المكتبة متوافقة تمامًا مع .NET Framework، .NET Core، وأحدث إصدارات .NET.
