## استيراد المساحات الاسمية

لبدء العمل، استورد مساحة الأسماء المطلوبة من Aspose.BarCode إلى مشروعك:

```csharp
using Aspose.BarCode.Generation;
```

## الخطوة 1: إعداد مولد الباركود

`BarcodeGenerator` هو الفئة الأساسية في Aspose.BarCode التي تنشئ وتُكوّن صور الباركود.

أنشئ مثيلًا من `BarcodeGenerator`، حدد النوع **Aztec**، وقدم البيانات التي تريد ترميزها.

```csharp
string path = "Your Directory Path";
System.Console.WriteLine("AztecErrorLevelExample:");

BarcodeGenerator gen = new BarcodeGenerator(EncodeTypes.Aztec, "Åspóse.Barcóde© is a powerful library to generate & recognize 1D & 2D barcodes");
```

> **نصيحة احترافية:** استبدل `"Your Directory Path"` بمسار مطلق أو نسبي حيث لديك أذونات كتابة.

## الخطوة 2: تعريف البُعد X

خاصية `XDimension` تحدد حجم الوحدة الفردية (البكسل) في الباركود المُولد.

البُعد X يتحكم في عرض أصغر وحدة (بكسل) في الباركود. ضبطه على 4 بكسل ينتج صورة واضحة قابلة للمسح.

```csharp
gen.Parameters.Barcode.XDimension.Pixels = 4;
```

## الخطوة 3: اختيار وضع رمز أزتك

`AztecSymbolMode` يحدد كيف تختار المكتبة حجم المصفوفة لرمز أزتك.

يدعم أزتك عدة أوضاع للرمز. استخدام `FullRange` يسمح للمكتبة باختيار الحجم الأمثل تلقائيًا بناءً على بياناتك وإعدادات تصحيح الأخطاء.

```csharp
gen.Parameters.Barcode.Aztec.AztecSymbolMode = AztecSymbolMode.FullRange;
```

## الخطوة 4: ضبط سعة تصحيح الأخطاء

`AztecErrorLevel` يحدد نسبة البيانات الزائدة المضافة لتصحيح الأخطاء.

الآن نقوم بضبط مستوى تصحيح الأخطاء. في هذا المثال ننشئ رمزين شريطيين — أحدهما بمستوى خطأ معتدل 5 % والآخر بمستوى قوي 50 % — لتوضيح الفرق البصري.

```csharp
// Set error correction capacity to 5%
gen.Parameters.Barcode.Aztec.AztecErrorLevel = 5;
gen.Save($"{path}AztecErrorLevel5.png", BarCodeImageFormat.Png);

// Set error correction capacity to 50%
gen.Parameters.Barcode.Aztec.AztecErrorLevel = 50;
gen.Save($"{path}AztecErrorLevel50.png", BarCodeImageFormat.Png);
```

تشغيل الشيفرة سيولد ملفي PNG في المجلد المحدد. النسخة بنسبة 50 % تحتوي على بيانات زائدة أكثر، مما يجعلها أكثر تحملًا للضرر على حساب رمز أكبر قليلًا.
