# كيفية إنشاء باركود بأقل حجم في Java باستخدام Aspose.BarCode

## مقدمة

إذا كنت بحاجة إلى **إنشاء باركود بأقل حجم** للصور لتصاميم واجهة المستخدم الضيقة، أو الملصقات الجاهزة للطباعة، أو أي سيناريو حيث كل مليمتر مهم، فأنت في المكان الصحيح. في هذا **دليل توليد الباركود Java**، سنستعرض الخطوات الدقيقة المطلوبة لتقليص الباركود إلى أصغر حجم ممكن مع الحفاظ على إمكانية مسحه، باستخدام Aspose.BarCode للـ Java.

## إجابات سريعة
- **ماذا يعني “باركود بأقل حجم”؟** هو أصغر أبعاد الصورة التي لا تزال تلبي متطلبات قابلية القراءة للرمز.  
- **أي فئة تُولّد الباركود؟** `BarcodeGenerator` من مكتبة Aspose.BarCode.  
- **هل أحتاج إلى ترخيص لهذا المثال؟** النسخة التجريبية المجانية تكفي للتطوير؛ الترخيص التجاري مطلوب للإنتاج.  
- **هل يمكنني تغيير الحجم بعد تعطيل AutoSize؟** نعم – يمكنك تحديد قيم العرض/الارتفاع صراحةً بالمليمترات.  
- **هل هذا النهج صالح لجميع أنواع الباركود؟** معظم الرموز أحادية الأبعاد (مثل CODE_128، CODE_39) تدعم تحديد الحجم يدويًا؛ تحقق من الوثائق للرموز الثنائية الأبعاد.

## ما هو “إنشاء باركود بأقل حجم”؟
إنشاء باركود بأقل حجم يعني ضبط المولد بحيث **لا** يقوم تلقائيًا بتكبير الصورة. بدلاً من ذلك، تحدد الأبعاد الدقيقة التي تحتاجها، مما يتيح لك وضع الباركود في مساحات ضيقة دون مساحة بيضاء مهدرة.

## لماذا تستخدم دليل توليد باركود Java مثل هذا؟
- **تصاميم موفرة للمساحة** – مثالية لشاشات الهواتف المحمولة، الإيصالات، أو طابعات الملصقات المدمجة.  
- **تحكم كامل** – أنت الذي تحدد الحجم الدقيق بالبكسل أو بالمليمتر.  
- **نتائج متسقة** – يعمل نفس الكود عبر JVMs على Windows وLinux وmacOS.

## المتطلبات المسبقة

قبل أن نغوص في الكود، تأكد من أن لديك:

1. **مجموعة تطوير جافا (JDK)** – أي نسخة حديثة (يوصى بالإصدار 8 أو أعلى).  
2. **Aspose.BarCode للـ Java** – قم بتحميل أحدث مكتبة من الموقع الرسمي [هنا](https://releases.aspose.com/barcode/java/).  

الآن لنبدأ بالبرمجة.

## استيراد المساحات الاسمية

في ملف Java المصدر الخاص بك، استورد الفئات المطلوبة من Aspose:

```java
import com.aspose.barcode.generation.AutoSizeMode;
import com.aspose.barcode.generation.BarcodeGenerator;
```

## دليل خطوة بخطوة لإنشاء باركود بأقل حجم

### الخطوة 1: إعداد مولد الباركود
أنشئ كائن `BarcodeGenerator`، اختر الترميز (CODE_128 في هذا المثال)، وقدم البيانات لتشفيرها.

```java
BarcodeGenerator bb = new BarcodeGenerator(EncodeTypes.CODE_128, "1234567");
```

### الخطوة 2: تعطيل وضع AutoSizeMode
بشكل افتراضي، يقوم Aspose.BarCode بتوسيع الصورة لتناسب الباركود. قم بإيقاف هذا السلوك حتى تتمكن من تحديد أبعادك الخاصة.

```java
bb.getParameters().setAutoSizeMode(AutoSizeMode.NONE);
```

### الخطوة 3: تحديد الحد الأدنى لعرض وارتفاع الصورة
حدد أصغر عرض وارتفاع لا يزال يسمح بقراءة الباركود. هنا نستخدم 1 مم لكلا البعدين، لكن يمكنك تعديلهما حسب الحاجة.

```java
bb.getParameters().getImageWidth().setMillimeters(1);
bb.getParameters().getImageHeight().setMillimeters(1);
```

> **نصيحة احترافية:** استخدم خصائص `getImageWidth()` و `getImageHeight()` لتجربة أحجام مختلفة حتى يقرأ الماسح الضوئي الرمز بثقة.

### الخطوة 4: حفظ صورة الباركود
أنشئ الـ bitmap واكتبها إلى ملف PNG. استبدل `dataDir` بالمسار الذي تريد حفظ الصورة فيه.

```java
javax.imageio.ImageIO.write(bb.generateBarCodeImage(), "PNG", new java.io.File(dataDir + "minimumresult.png"));
```

كرر الخطوات السابقة لكل باركود تحتاج إلى توليده بأقل حجم.
