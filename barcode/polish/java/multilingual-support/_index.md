## Czym jest wielojęzyczny kod kreskowy PDF417 w Javie?
PDF417 to układany liniowy format kodu kreskowego, który może kodować duże ilości danych, w tym znaki Unicode. Gdy kod PDF417 zawiera znaki z języków takich jak chiński czy turecki, potrzebny jest dekoder rozumiejący Unicode i potrafiący prawidłowo mapować te glify. API Aspose.BarCode dla Javy zapewnia wbudowaną obsługę wielojęzyczną, umożliwiając odczyt i interpretację tych znaków bez dodatkowej konfiguracji.

## Typowe problemy i rozwiązania
- **Incorrect character rendering** – Verify that the source image is high‑resolution and that the `BarcodeReader` is configured with `BarcodeReadType.PDF417`. → **Nieprawidłowe renderowanie znaków** – Upewnij się, że obraz źródłowy ma wysoką rozdzielczość oraz że `BarcodeReader` jest skonfigurowany z `BarcodeReadType.PDF417`.  
- **License not found** – Ensure the `Aspose.BarCode.lic` file is placed in the classpath or set programmatically via `License.setLicense()`. → **Nie znaleziono licencji** – Upewnij się, że plik `Aspose.BarCode.lic` znajduje się w classpath lub ustaw go programowo za pomocą `License.setLicense()`.  
- **Performance on large batches** – Use `BarcodeReader.setReadQuality(BarcodeReadQuality.Low)` for faster processing when perfect accuracy is not critical. → **Wydajność przy dużych partiach** – Użyj `BarcodeReader.setReadQuality(BarcodeReadQuality.Low)` dla szybszego przetwarzania, gdy perfekcyjna dokładność nie jest krytyczna.

## Najczęściej zadawane pytania

**Q: Czy mogę dekodować kody PDF417, które mieszają wiele języków w jednym symbolu?**  
A: Yes. Aspose.BarCode’s Unicode engine handles mixed‑language data automatically. → **Tak. Silnik Unicode Aspose.BarCode automatycznie obsługuje dane mieszane językowo.**

**Q: Czy istnieje limit liczby znaków, które mogę zdekodować?**  
A: PDF417 can store up to 1,850 text characters; the library will return the full decoded string as long as the barcode is not damaged. → **PDF417 może przechowywać do 1 850 znaków tekstowych; biblioteka zwróci pełny zdekodowany ciąg, o ile kod nie jest uszkodzony.**

**Q: Czy muszę ustawiać specjalne opcje dla tureckich znaków?**  
A: No special options are required; the default Unicode handling covers Turkish characters out of the box. → **Nie są wymagane specjalne opcje; domyślna obsługa Unicode obejmuje tureckie znaki od razu.**

**Q: Jak obsłużyć pliki PDF chronione hasłem, które zawierają kody PDF417?**  
A: Load the PDF with the appropriate password using `BarCodeReader`’s overload that accepts a `PdfDocument` object. → **Załaduj PDF z odpowiednim hasłem, używając przeciążenia `BarCodeReader`, które przyjmuje obiekt `PdfDocument`.**

**Q: Czy biblioteka jest kompatybilna z Androidem?**  
A: Yes, Aspose.BarCode for Java works on Android Studio projects with a minimum API level of 21. → **Tak, Aspose.BarCode for Java działa w projektach Android Studio przy minimalnym poziomie API 21.**
