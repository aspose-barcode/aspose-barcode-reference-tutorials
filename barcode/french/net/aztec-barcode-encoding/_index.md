## Qu’est‑ce qu’un exemple de code‑barres aztec ?

Un **exemple de code‑barres aztec** montre comment instancier le `BarCodeBuilder`, définir la symbologie sur Aztec et rendre l’image ou le SVG. L’exemple ci‑dessous (incluse dans les tutoriels liés) montre le code minimal dont vous avez besoin, ainsi que des paramètres optionnels tels que le niveau de correction d’erreurs et le mode de symbole.

## Pourquoi utiliser le mode auto aztec ?

`AztecAutoMode` permet à la bibliothèque de décider d’utiliser le mode **Compact**, **FullRange** ou **Rune** en fonction de la quantité de données que vous encodez. Cela élimine les approximations et garantit le code‑barres le plus petit possible tout en conservant une lisibilité maximale — parfait pour les conceptions d’interface réactives.
