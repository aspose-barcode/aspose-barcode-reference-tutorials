## Qu'est-ce que generate datamatrix barcode ?
Un code-barres DataMatrix encode les données dans un motif bidimensionnel de cellules noires et blanches. Avec Aspose.BarCode, vous pouvez **generate datamatrix barcode** des images, PDF ou SVG directement depuis vos applications .NET, vous offrant un contrôle total sur la taille, la correction d'erreurs et le mode d'encodage.

## Questions fréquentes

**Q : Comment choisir le mode ECC à utiliser ?**  
R : Choisissez ECC 000‑140 pour de petits ensembles de données avec une correction d'erreurs limitée, ou ECC 200 pour des données plus volumineuses et une fiabilité accrue. Le mode macro ajoute des couches de données supplémentaires.

**Q : Puis-je intégrer du texte personnalisé dans un code-barres DataMatrix ?**  
R : Oui, utilisez la propriété `CodeText` pour définir le texte de code étendu, puis configurez le mode d'encodage selon les besoins.

**Q : Existe-t-il un moyen de sélectionner automatiquement le meilleur mode d'encodage ?**  
R : Définissez `EncodeMode` sur `Auto` et Aspose.BarCode évaluera l'entrée et choisira le mode optimal (ASCII, C40, Bytes, etc.).

**Q : Quelles sont les considérations de performance pour de gros lots de codes-barres ?**  
R : Réutilisez l'instance `BarCodeGenerator` et activez le multithreading lorsque c'est possible. De plus, générez les images dans un format adapté (PNG pour sans perte, JPEG pour une taille plus petite).

**Q : Aspose.BarCode prend‑il en charge .NET Core et .NET 5/6 ?**  
R : Absolument – la bibliothèque est entièrement compatible avec .NET Framework, .NET Core et les dernières versions de .NET.
