# GS1 कूपन UPC‑A कोड 128 एन्कोडिंग

## परिचय

बारकोड रिटेल शेल्फ़, वेयरहाउस और मोबाइल कूपन के पीछे की चुप्पी कामगार हैं। यदि आपको कभी .NET एप्लिकेशन में **generate barcode from string** डेटा की आवश्यकता पड़ी है, तो Aspose.BarCode for .NET आपको इसे साफ़ और भरोसेमंद तरीके से करने का तरीका देता है। इस **barcode generation tutorial C#** में आप एक पूर्ण **barcode generator C# example** देखेंगे जो एक साधारण टेक्स्ट स्ट्रिंग से GS1 कूपन UPC‑A कोड 128 बारकोड बनाता है। इस गाइड के अंत तक आप बारकोड को सीधे अपने प्रोजेक्ट्स में एम्बेड कर पाएँगे, बिना लो‑लेवल एन्कोडिंग लॉजिक से जूझे।

## GS1 कूपन UPC‑A कोड 128 क्या है?

GS1 कूपन UPC‑A कोड 128 बारकोड पारंपरिक UPC‑A न्यूमेरिक फ़ॉर्मेट को अतिरिक्त GS1 एप्लिकेशन आइडेंटिफ़ायर्स (AIs) के साथ मिलाता है, जो कूपन‑विशिष्ट डेटा जैसे डिस्काउंट राशि या समाप्ति तिथि ले जाते हैं। इस जानकारी को **string** के रूप में एन्कोड करके और Aspose को कन्वर्ज़न संभालने देने से आप मैन्युअल चेकसम गणना और फ़ॉर्मेट की जटिलताओं से बचते हैं।

## इस कार्य के लिए Aspose.BarCode क्यों उपयोग करें?

- **Zero‑dependency एन्कोडिंग** – लाइब्रेरी सटीक GS1 नियमों को जानती है।  
- **उच्च‑गुणवत्ता आउटपुट** – एक ही कॉल से PNG, JPEG, SVG, या PDF जेनरेट करें।  
- **पूर्ण नियंत्रण** – C# से बाहर निकले बिना डाइमेंशन, रंग और क्वाइट ज़ोन को ट्यून करें।  

## स्ट्रिंग से बारकोड जेनरेट करने के लिए स्टेप‑बाय‑स्टेप गाइड – GGS1 कूपन UPC‑A कोड 128

आइए Aspose.BarCode for .NET का उपयोग करके GGS1 कूपन UPC‑A कोड 128 बारकोड जेनरेट करने की चरण‑बद्ध प्रक्रिया को देखें। इस उदाहरण में हम कोड को समझने योग्य चरणों में विभाजित करेंगे।

### चरण 1: डायरेक्टरी पाथ सेट करें

उस डायरेक्टरी पाथ को परिभाषित करके शुरू करें जहाँ आप जेनरेटेड बारकोड इमेज को सेव करना चाहते हैं।

```csharp
string path = "Your Directory Path";
```

`"Your Directory Path"` को अपने सिस्टम पर वास्तविक पाथ से बदलें।

### चरण 2: बारकोड जेनरेटर बनाएं

इच्छित एन्कोडिंग टाइप और एन्कोड करने के डेटा के साथ एक `BarcodeGenerator` ऑब्जेक्ट इनिशियलाइज़ करें। इस केस में हम डेटा `"123456789012(8110)ASPOSE"` के साथ GGS1 कूपन UPC‑A कोड 128 बारकोड बना रहे हैं।

```csharp
BarcodeGenerator gen = new BarcodeGenerator(EncodeTypes.UpcaGs1Code128Coupon, "123456789012(8110)ASPOSE");
```

यदि आवश्यक हो तो आप डेटा को अपने अनुसार बदल सकते हैं।

### चरण 3: बारकोड पैरामीटर्स कस्टमाइज़ करें

आप अपने बारकोड के विभिन्न पैरामीटर्स को फाइन‑ट्यून कर सकते हैं, जैसे X‑Dimension (सबसे छोटे बार का आकार), इमेज फ़ॉर्मेट आदि। इस उदाहरण में हमने X‑Dimension को 2 पिक्सेल सेट किया है।

```csharp
gen.Parameters.Barcode.XDimension.Pixels = 2;
```

इन पैरामीटर्स को अपने प्रोजेक्ट की ज़रूरतों के अनुसार समायोजित करने में संकोच न करें।

### चरण 4: बारकोड इमेज सेव करें

अब जेनरेटेड बारकोड को अपनी निर्दिष्ट डायरेक्टरी में इमेज के रूप में सेव करें। हम इसे PNG फ़ॉर्मेट में सेव कर रहे हैं।

```csharp
gen.Save($"{path}Gs1CouponUpcaCode128.png", BarCodeImageFormat.Png);
```

फ़ाइलनाम और इमेज फ़ॉर्मेट को अपनी आवश्यकता अनुसार बदल सकते हैं।

इन चार सरल चरणों का पालन करके, आपने Aspose.BarCode for .NET का उपयोग करके सफलतापूर्वक GGS1 कूपन UPC‑A कोड 128 बारकोड जेनरेट कर लिया है।
