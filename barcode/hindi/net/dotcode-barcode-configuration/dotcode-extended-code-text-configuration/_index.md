# Aspose.BarCode for .NET के साथ डॉटकोड विस्तारित कोडटेक्स्ट कैसे बनाएं

## परिचय

बारकोड जेनरेशन और प्रबंधन के क्षेत्र में, Aspose.BarCode for .NET एक बहुमुखी और कुशल समाधान के रूप में उभरता है। चाहे आपको उत्पादों, इन्वेंट्री या किसी अन्य अनुप्रयोग के लिए बारकोड जनरेट करने हों, Aspose.BarCode for .NET आपकी सभी जरूरतों को पूरा करता है। इस व्यापक ट्यूटोरियल में, हम **डॉटकोड विस्तारित कोडटेक्स्ट** बनाएंगे और यह समझेंगे कि यह क्षमता आधुनिक डेटा‑समृद्ध वातावरण में क्यों आवश्यक है। डॉटकोड एक द्वि‑आयामी मैट्रिक्स बारकोड है जो टेक्स्टुअल और बाइनरी दोनों डेटा को एन्कोड कर सकता है, जिससे यह विभिन्न उद्योगों में एक मूल्यवान टूल बन जाता है।

## त्वरित उत्तर
- **“डॉटकोड विस्तारित कोडटेक्स्ट बनाना” का क्या अर्थ है?** इसका मतलब है एक DotCode बारकोड बनाना जिसमें FNC1, ECICodetext, साधारण टेक्स्ट और सिम्बॉल सेपरेटर सभी एक ही विस्तारित पेलोड में शामिल हों।  
- **कौन सी लाइब्रेरी आवश्यक है?** Aspose.BarCode for .NET।  
- **क्या लाइसेंस चाहिए?** मूल्यांकन के लिए एक अस्थायी लाइसेंस काम करता है; उत्पादन के लिए पूर्ण लाइसेंस आवश्यक है।  
- **कौन से .NET संस्करण समर्थित हैं?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7+।  
- **इम्प्लीमेंटेशन में कितना समय लगेगा?** बुनियादी उदाहरण के लिए लगभग 10‑15 मिनट।

## डॉटकोड विस्तारित कोडटेक्स्ट कैसे बनाएं

नीचे एक संक्षिप्त, चरण‑दर‑चरण गाइड दिया गया है जो दिखाता है कि विस्तारित कोडटेक्स्ट कैसे बनाते हैं और बारकोड इमेज कैसे रेंडर करते हैं।

## पूर्वापेक्षाएँ

चरण‑दर‑चरण गाइड में जाने से पहले, कुछ पूर्वापेक्षाएँ हैं जिन्हें आपके पास होना आवश्यक है:

1. Aspose.BarCode for .NET: सुनिश्चित करें कि आपके पास Aspose.BarCode for .NET लाइब्रेरी स्थापित है और तैयार है। यदि नहीं, तो आप इसे [Aspose.BarCode for .NET documentation](https://reference.aspose.com/barcode/net/) से डाउनलोड कर सकते हैं।

2. डेवलपमेंट एनवायरनमेंट: आपके सिस्टम पर एक कार्यशील .NET डेवलपमेंट एनवायरनमेंट, आदर्श रूप से Visual Studio, स्थापित होना चाहिए।

इन पूर्वापेक्षाओं को पूरा करने के बाद, हम DotCode विस्तारित कोडटेक्स्ट जेनरेट करने की प्रक्रिया आगे बढ़ा सकते हैं।

## नेमस्पेसेस इम्पोर्ट करें

सबसे पहले, आपको अपने .NET प्रोजेक्ट में आवश्यक नेमस्पेसेस इम्पोर्ट करने होंगे ताकि Aspose.BarCode लाइब्रेरी की आवश्यक कार्यक्षमताओं तक पहुंच सकें। इसे करने का तरीका इस प्रकार है:

```csharp
using Aspose.BarCode.Generation;
```

अब जब हमने पूर्वापेक्षाएँ कवर कर ली हैं, चलिए DotCode विस्तारित कोडटेक्स्ट जेनरेट करने की प्रक्रिया को चरण‑दर‑चरण समझते हैं।

## चरण 1: डायरेक्टरी पाथ निर्धारित करें

इस चरण में, आपको उस डायरेक्टरी पाथ को निर्दिष्ट करना होगा जहाँ आप उत्पन्न DotCode विस्तारित कोडटेक्स्ट इमेज को सेव करना चाहते हैं।

```csharp
string path = "Your Directory Path";
```

`"Your Directory Path"` को अपने सिस्टम के वास्तविक पाथ से बदलें।

## चरण 2: DotCode विस्तारित कोडटेक्स्ट बनाएं

DotCode विस्तारित कोडटेक्स्ट बनाने के लिए नीचे दिए गए उप‑चरणों का पालन करें:

### 2.1 FNC1 फ़ॉर्मेट आइडेंटिफ़ायर जोड़ें

FNC1 फ़ॉर्मेट आइडेंटिफ़ायर का उपयोग नए डेटा फ़ील्ड की शुरुआत को दर्शाने के लिए किया जाता है। यह DotCode विस्तारित कोडटेक्स्ट का एक अनिवार्य भाग है।

```csharp
DotCodeExtCodetextBuilder textBuilder = new DotCodeExtCodetextBuilder();
textBuilder.AddFNC1FormatIdentifier();
```

### 2.2 ECICodetext जोड़ें

ECICodetext वह जगह है जहाँ आप विशेष अक्षर और अंतरराष्ट्रीय टेक्स्ट एन्कोड कर सकते हैं। इस उदाहरण में, हमने UTF‑8 एन्कोडिंग का उपयोग करके `"犬Right狗"` को एन्कोड किया है।

```csharp
textBuilder.AddECICodetext(ECIEncodings.UTF8, "犬Right狗");
```

### 2.3 साधारण कोडटेक्स्ट जोड़ें

आप DotCode विस्तारित कोडटेक्स्ट में साधारण टेक्स्ट भी जोड़ सकते हैं। यहाँ हमने `"Plain text"` जोड़ा है।

```csharp
textBuilder.AddPlainCodetext("Plain text");
```

### 2.4 FNC3 सिम्बॉल सेपरेटर जोड़ें

FNC3 सिम्बॉल सेपरेटर का उपयोग कोड के विभिन्न सेक्शन को अलग करने के लिए किया जाता है।

```csharp
textBuilder.AddFNC3SymbolSeparator();
```

### 2.5 FNC3 रीडर इनिशियलाइज़ेशन जोड़ें

यह चरण FNC3 रीडर इनिशियलाइज़ेशन जानकारी जोड़ता है।

```csharp
textBuilder.AddFNC3ReaderInitialization();
```

### 2.6 कोडटेक्स्ट जेनरेट करें

अब, `textBuilder` ऑब्जेक्ट पर `GetExtendedCodetext` मेथड को कॉल करके DotCode विस्तारित कोडटेक्स्ट जेनरेट करें।

```csharp
string codetext = textBuilder.GetExtendedCodetext();
```

## चरण 3: DotCode इमेज जेनरेट करें

DotCode विस्तारित कोडटेक्स्ट को इमेज के रूप में जेनरेट करने के लिए नीचे दिए गए उप‑चरणों का पालन करें:

#### 4.1 बारकोड जेनरेटर इनिशियलाइज़ करें

उपयुक्त पैरामीटर के साथ `BarcodeGenerator` को इनिशियलाइज़ करें। इस मामले में, हम `EncodeTypes.DotCode` और जेनरेटेड कोडटेक्स्ट का उपयोग करते हैं।

```csharp
using (BarcodeGenerator gen = new BarcodeGenerator(EncodeTypes.DotCode, codetext))
{
    // Set the X-dimension for the barcode (adjust as needed).
    gen.Parameters.Barcode.XDimension.Pixels = 10;

    // Set the DotCode encoding mode to ExtendedCodetext.
    gen.Parameters.Barcode.DotCode.DotCodeEncodeMode = DotCodeEncodeMode.ExtendedCodetext;

    // Save the generated barcode image.
    gen.Save($"{path}DotCodeExtendedCodetext.png", BarCodeImageFormat.Png);
}
```

और बस! आपने सफलतापूर्वक Aspose.BarCode for .NET का उपयोग करके DotCode विस्तारित कोडटेक्स्ट जेनरेट कर लिया है।

## निष्कर्ष

Aspose.BarCode for .NET एक शक्तिशाली टूल है जो बारकोड जेनरेशन को सरल बनाता है। इस ट्यूटोरियल में, हमने **डॉटकोड विस्तारित कोडटेक्स्ट** बनाने पर ध्यान केंद्रित किया, जो विभिन्न उद्योगों में आवश्यक है, विशेष रूप से जहाँ बहुभाषी और विशेष अक्षर एन्कोडिंग की आवश्यकता होती है। ऊपर बताए गए चरणों का पालन करके, आप अपनी विशिष्ट जरूरतों के लिए आसानी से DotCode विस्तारित कोडटेक्स्ट बना सकते हैं।

यदि आपको आगे मार्गदर्शन चाहिए या कोई प्रश्न हैं, तो [Aspose.BarCode for .NET documentation](https://reference.aspose.com/barcode/net/) देखें या [Aspose.BarCode support forum](https://forum.aspose.com/c/barcode/13) पर समुदाय से संपर्क करें।

## अक्सर पूछे जाने वाले प्रश्न

**प्र: क्या मैं जेनरेटेड बारकोड को मोबाइल ऐप में उपयोग कर सकता हूँ?**  
उ: हाँ। जेनरेटर द्वारा उत्पन्न PNG इमेज को iOS, Android या किसी भी क्रॉस‑प्लेटफ़ॉर्म मोबाइल एप्लिकेशन में एम्बेड किया जा सकता है।

**प्र: यदि मुझे टेक्स्ट के बजाय बाइनरी डेटा एन्कोड करना हो तो क्या करूँ?**  
उ: `AddECICodetext` मेथड को उपयुक्त `ECIEncodings` (जैसे `ECIEncodings.Base64`) के साथ उपयोग करके बाइनरी पेलोड एम्बेड करें।

**प्र: बारकोड का आकार पढ़ने की क्षमता को प्रभावित किए बिना कैसे बदलूँ?**  
उ: `XDimension.Pixels` प्रॉपर्टी को समायोजित करें; उच्च मान मॉड्यूल आकार बढ़ाते हैं, जबकि कम मान बारकोड को अधिक कॉम्पैक्ट बनाते हैं।

**प्र: क्या बारकोड के चारों ओर क्वाइट ज़ोन जोड़ना संभव है?**  
उ: हाँ। इच्छित क्वाइट ज़ोन को पिक्सेल में परिभाषित करने के लिए `gen.Parameters.Barcode.Margin` सेट करें।

**प्र: क्या लाइब्रेरी .NET 8 को सपोर्ट करती है?**  
उ: नवीनतम Aspose.BarCode रिलीज़ .NET 8 के साथ संगत हैं; बस उपयुक्त NuGet पैकेज संस्करण को रेफ़रेंसेज़ में जोड़ें।
