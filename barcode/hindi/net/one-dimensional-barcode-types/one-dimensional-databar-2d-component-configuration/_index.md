# वन-डायमेंशनल डेटाबार 2D कॉम्पोनेन्ट कॉन्फ़िगरेशन

इस ट्यूटोरियल में आप **create barcode generator Aspose** को Aspose.BarCode .NET लाइब्रेरी का उपयोग करके वन‑डायमेंशनल डेटाबार 2D कॉम्पोनेन्ट के लिए बनाएँगे। चाहे आप रिटेल लेबल, इन्वेंटरी टैग या कोई भी एप्लिकेशन बना रहे हों जिसे कॉम्पैक्ट, हाई‑डेंसिटी डेटा की आवश्यकता हो, यह गाइड आपको प्रोजेक्ट सेटअप से लेकर अंतिम PNG इमेज को सेव करने तक हर कदम पर ले जाएगा।

## त्वरित उत्तर
- **2D कॉम्पोनेन्ट फ्लैग क्या करता है?** यह जेनरेटर को बताता है कि डेटाबार बारकोड के अंदर एक कॉम्पोज़िट 2D सिम्बल एम्बेड करना है या नहीं।  
- **क्या मैं X‑डायमेंशन बदल सकता हूँ?** हाँ, `XDimension.Pixels` प्रॉपर्टी मॉड्यूल की चौड़ाई को नियंत्रित करती है।  
- **उदाहरण में कौन सा इमेज फॉर्मेट उपयोग किया गया है?** PNG, `BarCodeImageFormat.Png` के माध्यम से।  
- **क्या विकास के लिए लाइसेंस की आवश्यकता है?** परीक्षण के लिए एक फ्री ट्रायल काम करता है; उत्पादन के लिए एक कमर्शियल लाइसेंस आवश्यक है।  
- **क्या कोड .NET Core के साथ संगत है?** बिल्कुल—Aspose.BarCode .NET Framework और .NET Core दोनों को सपोर्ट करता है।

## वन‑डायमेंशनल डेटाबार 2D कॉम्पोनेन्ट क्या है?
डेटाबार 2D कॉम्पोनेन्ट एक पारंपरिक लीनियर बारकोड को एक छोटे 2D कॉम्पोज़िट सिम्बल के साथ जोड़ता है, जिससे आप अतिरिक्त जानकारी (जैसे URL या अतिरिक्त डेटा फ़ील्ड) को बिना कुल बारकोड आकार बढ़ाए स्टोर कर सकते हैं।

## इस कार्य के लिए Aspose.BarCode क्यों उपयोग करें?
- **पूर्ण .NET इंटीग्रेशन** – C# प्रोजेक्ट्स के साथ सहजता से काम करता है।  
- **समृद्ध कॉन्फ़िगरेशन API** – डाइमेंशन को ट्यून करें, 2D कॉम्पोनेन्ट को एनेबल/डिसेबल करें, और कई आउटपुट फॉर्मेट में से चुनें।  
- **कोई बाहरी डिपेंडेंसी नहीं** – लाइब्रेरी सेल्फ‑कंटेन्ड है, जिससे डिप्लॉयमेंट सरल हो जाता है।

## नेमस्पेसेस इम्पोर्ट करें

सबसे पहले आपको Aspose.BarCode नेमस्पेस इम्पोर्ट करना होगा ताकि आप उसकी क्लासेज़ तक पहुंच सकें।

```csharp
using Aspose.BarCode;
```

## आउटपुट पाथ निर्धारित करें

निर्दिष्ट करें कि जेनरेट किए गए बारकोड इमेजेज़ आपके फ़ाइल सिस्टम में कहाँ सेव होंगे।

```csharp
string path = "Your Directory Path";
```

`"Your Directory Path"` को अपने मशीन पर वास्तविक फ़ोल्डर पाथ से बदलें।

## बारकोड जेनरेटर बनाएं

`BarcodeGenerator` को डेटाबार एक्सपैंडेड टाइप के साथ इंस्टैंशिएट करें और वह डेटा प्रदान करें जिसे आप एन्कोड करना चाहते हैं।

```csharp
BarcodeGenerator gen = new BarcodeGenerator(EncodeTypes.DatabarExpanded, "(01)12345678901231");
```

सैंपल डेटा को अपने स्वयं के GS1‑एप्लिकेशन आइडेंटिफ़ायर या अन्य पेलोड से बदलने में संकोच न करें।

## वन‑डायमेंशनल डेटाबार 2D के लिए बारकोड जेनरेटर Aspose कैसे बनाएं

अब विज़ुअल प्रॉपर्टीज़ और 2D कॉम्पोनेन्ट फ्लैग को कॉन्फ़िगर करें, फिर इमेजेज़ सेव करें।

```csharp
gen.Parameters.Barcode.XDimension.Pixels = 2;

// Disable 2D component flag
gen.Parameters.Barcode.DataBar.Is2DCompositeComponent = false;
gen.Save($"{path}Databar2DComponentDisabled.png", BarCodeImageFormat.Png);

// Enable 2D component flag
gen.Parameters.Barcode.DataBar.Is2DCompositeComponent = true;
gen.Save($"{path}Databar2DComponentEnabled.png", BarCodeImageFormat.Png);
```

- **XDimension** प्रत्येक बारकोड मॉड्यूल की चौड़ाई को नियंत्रित करता है।  
- `Is2DCompositeComponent` को **false** सेट करने से एक शुद्ध लीनियर डेटाबार बनता है।  
- इसे **true** सेट करने से कॉम्पोज़िट 2D सिम्बल जुड़ जाता है, जो अतिरिक्त डेटा एन्कोड करने में उपयोगी है।

## सामान्य समस्याएँ एवं टिप्स

- **Invalid Path** – सुनिश्चित करें कि फ़ोल्डर मौजूद है और एप्लिकेशन को लिखने की अनुमति है।  
- **License Exception** – यदि लाइसेंसिंग वार्निंग दिखे, तो बारकोड जेनरेट करने से पहले अपना Aspose लाइसेंस लागू करें।  
- **Image Not Visible** – जाँचें कि `BarCodeImageFormat` आपके द्वारा उपयोग किए गए फ़ाइल एक्सटेंशन से मेल खाता है।

## Frequently Asked Questions

**Q: क्या मैं PNG के अलावा अन्य फॉर्मेट में बारकोड जेनरेट कर सकता हूँ?**  
A: हाँ, `Save` मेथड BMP, JPEG, GIF, TIFF आदि फॉर्मेट को सपोर्ट करता है, बस उपयुक्त `BarCodeImageFormat` निर्दिष्ट करें।

**Q: मैं बारकोड पर कस्टम रंग कैसे लागू करूँ?**  
A: `gen.Parameters.Barcode.ForeColor` और `gen.Parameters.Barcode.BackColor` का उपयोग करके फोरग्राउंड और बैकग्राउंड रंग सेट करें।

**Q: क्या बारकोड इमेज में लोगो एम्बेड करना संभव है?**  
A: Aspose.BarCode एक `Image` प्रॉपर्टी प्रदान करता है जहाँ आप बारकोड जेनरेट होने के बाद लोगो ओवरले कर सकते हैं।

**Q: कौन‑से .NET संस्करण सपोर्टेड हैं?**  
A: लाइब्रेरी .NET Framework 4.5+, .NET Core 3.1+, .NET 5+, और .NET 6+ के साथ काम करती है।

**Q: कम‑रिज़ॉल्यूशन प्रिंट्स के लिए स्कैनिंग रिलेबिलिटी कैसे बढ़ाएँ?**  
A: `XDimension` वैल्यू बढ़ाएँ और बारकोड व बैकग्राउंड के बीच पर्याप्त कंट्रास्ट सुनिश्चित करें।
