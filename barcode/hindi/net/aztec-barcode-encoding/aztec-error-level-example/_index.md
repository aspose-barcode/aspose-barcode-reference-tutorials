## त्रुटि सुधार के साथ aztec barcode .net कैसे बनाएं?
`BarcodeGenerator` को लोड करें, Aztec सिम्बोलॉजी चुनें, वांछित त्रुटि‑सुधार प्रतिशत सेट करें, और `Save` को कॉल करें – यह वह सब है जो आपको बारकोड छवि उत्पन्न करने के लिए चाहिए। लाइब्रेरी आकार, एन्कोडिंग, और त्रुटि‑सुधार डेटा को स्वचालित रूप से संभालती है, इसलिए आप उस बिजनेस लॉजिक पर ध्यान केंद्रित कर सकते हैं जो एन्कोड करने के लिए टेक्स्ट प्रदान करता है।

## त्रुटि सुधार के साथ Aztec बारकोड बनाना क्या है?
Aztec बारकोड एक कॉम्पैक्ट 2‑D मैट्रिक्स कोड है जो बड़ी मात्रा में डेटा संग्रहीत कर सकता है, और त्रुटि सुधार अतिरिक्त जानकारी जोड़ता है जिससे प्रतीक को तब भी पढ़ा जा सकता है जब उसका कुछ हिस्सा क्षतिग्रस्त या अस्पष्ट हो। त्रुटि‑सुधार स्तर को समायोजित करने से आप डेटा क्षमता को प्रतिरोध के साथ बदल सकते हैं, जिससे बारकोड औद्योगिक लेबलिंग या मोबाइल टिकट स्कैनिंग जैसे कठोर वातावरण के लिए उपयुक्त बनता है।

## चरण 1: बारकोड जेनरेटर सेट अप करें
`BarcodeGenerator` Aspose.BarCode की मुख्य क्लास है जो बारकोड छवियां बनाती और कॉन्फ़िगर करती है।

`BarcodeGenerator` का एक इंस्टेंस बनाएं, **Aztec** प्रकार निर्दिष्ट करें, और वह डेटा प्रदान करें जिसे आप एन्कोड करना चाहते हैं।

```csharp
string path = "Your Directory Path";
System.Console.WriteLine("AztecErrorLevelExample:");

BarcodeGenerator gen = new BarcodeGenerator(EncodeTypes.Aztec, "Åspóse.Barcóde© is a powerful library to generate & recognize 1D & 2D barcodes");
```

> **प्रो टिप:** `"Your Directory Path"` को उस पूर्ण या सापेक्ष पाथ से बदलें जहाँ आपके पास लिखने की अनुमति हो।

## चरण 2: X‑Dimension निर्धारित करें
`XDimension` प्रॉपर्टी उत्पन्न बारकोड में एकल मॉड्यूल (पिक्सेल) का आकार निर्धारित करती है।

X‑Dimension बारकोड में सबसे छोटे मॉड्यूल (पिक्सेल) की चौड़ाई को नियंत्रित करता है। इसे 4 पिक्सेल पर सेट करने से एक स्पष्ट, स्कैन करने योग्य छवि प्राप्त होती है।

```csharp
gen.Parameters.Barcode.XDimension.Pixels = 4;
```

## चरण 3: Aztec Symbol Mode चुनें
`AztecSymbolMode` निर्धारित करता है कि लाइब्रेरी Aztec बारकोड के लिए मैट्रिक्स आकार कैसे चुनती है।

Aztec कई सिम्बॉल मोड्स का समर्थन करता है। `FullRange` का उपयोग करने से लाइब्रेरी आपके डेटा और त्रुटि‑सुधार सेटिंग्स के आधार पर स्वचालित रूप से इष्टतम आकार चुनती है।

```csharp
gen.Parameters.Barcode.Aztec.AztecSymbolMode = AztecSymbolMode.FullRange;
```

## चरण 4: त्रुटि‑सुधार क्षमता सेट करें
`AztecErrorLevel` त्रुटि सुधार के लिए जोड़ी गई अतिरिक्त डेटा का प्रतिशत सेट करता है।

अब हम त्रुटि‑सुधार स्तर को समायोजित करते हैं। इस उदाहरण में हम दो बारकोड बनाते हैं—एक 5 % त्रुटि स्तर के साथ और दूसरा 50 % स्तर के साथ—ताकि दृश्य अंतर को दर्शाया जा सके।

```csharp
// Set error correction capacity to 5%
gen.Parameters.Barcode.Aztec.AztecErrorLevel = 5;
gen.Save($"{path}AztecErrorLevel5.png", BarCodeImageFormat.Png);

// Set error correction capacity to 50%
gen.Parameters.Barcode.Aztec.AztecErrorLevel = 50;
gen.Save($"{path}AztecErrorLevel50.png", BarCodeImageFormat.Png);
```

कोड चलाने से निर्दिष्ट फ़ोल्डर में दो PNG फ़ाइलें बनेंगी। 50 % संस्करण में अधिक अतिरिक्त डेटा होता है, जिससे यह थोड़ा बड़ा प्रतीक होने के बावजूद क्षति के प्रति अधिक सहनशील बन जाता है।

| Scanner cannot read the code | डेटा की मात्रा के लिए त्रुटि स्तर बहुत अधिक है | `AztecErrorLevel` कम करें या एन्कोडेड टेक्स्ट को छोटा करें। |
