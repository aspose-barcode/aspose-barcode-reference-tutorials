```csharp
string path = "Your Directory Path";
System.Console.WriteLine("AztecSymbolModeExample:");

BarcodeGenerator gen = new BarcodeGenerator(EncodeTypes.Aztec, "Åspóse.Barcóde©");
```

इस चरण में हमने जनरेटर सेट किया और एन्कोड करने के लिए कोड टेक्स्ट प्रदान किया।

## Aztec Symbol Mode को Auto पर कैसे सेट करें?

`AztecSymbolMode` एनेमरेशन Aztec बारकोड के चार संभावित सिम्बॉल मोड्स को परिभाषित करता है, और **Auto** विकल्प लाइब्रेरी को सभी डेटा और एरर‑करेक्शन आवश्यकताओं को बनाए रखते हुए सबसे कॉम्पैक्ट आकार स्वचालित रूप से चुनने देता है। यह मोड अधिकांश परिदृश्यों के लिए आदर्श है जहाँ आप मैन्युअल ट्यूनिंग के बिना सबसे छोटा संभव बारकोड चाहते हैं।

```csharp
gen.Parameters.Barcode.Aztec.AztecSymbolMode = AztecSymbolMode.Auto;
gen.Save($"{path}AztecSymbolModeAuto.png", BarCodeImageFormat.Png);
```

यह चरण सुनिश्चित करता है कि Aztec Symbol Mode स्वचालित रूप से निर्धारित हो, और परिणामी बारकोड इमेज सहेजी जाए।

## FullRange Symbol Mode को कैसे मजबूर करें?

जब आपको अधिकतम डेटा क्षमता चाहिए, तो आप `AztecSymbolMode` एनेमरेशन के **FullRange** मान को चुन सकते हैं। यह मोड स्वचालित आकार घटाव को निष्क्रिय करता है, जिससे बारकोड सभी अक्षरों की पूरी रेंज को संग्रहीत कर सकता है और सबसे अधिक सूचना घनत्व प्राप्त करता है, जो बड़े डेटा सेट के लिए उपयोगी है।

```csharp
gen.Parameters.Barcode.Aztec.AztecSymbolMode = AztecSymbolMode.FullRange;
gen.Save($"{path}AztecSymbolModeFullRange.png", BarCodeImageFormat.Png);
```

यह FullRange Aztec Symbol Mode के साथ एक बारकोड बनाएगा।

## Compact Aztec बारकोड कैसे जनरेट करें?

`AztecSymbolMode` एनेमरेशन का **Compact** मान मैट्रिक्स में उपयोग किए गए लेयर्स की संख्या घटाकर एक छोटा बारकोड उत्पन्न करता है। इससे अधिक स्थान‑कुशल इमेज बनती है, जो मोबाइल स्क्रीन या छोटे लेबल जैसे सीमित डिस्प्ले क्षेत्र वाले अनुप्रयोगों के लिए उपयुक्त है।

```csharp
gen.Parameters.Barcode.Aztec.AztecSymbolMode = AztecSymbolMode.Compact;
gen.Save($"{path}AztecSymbolModeCompact.png", BarCodeImageFormat.Png);
```

यह चरण बारकोड को Compact Aztec Symbol Mode के साथ जनरेट करने के लिए कॉन्फ़िगर करता है।

## Rune Aztec बारकोड कैसे बनाएं?

**Rune** मोड Aztec सिम्बोलॉजी का एक विशेष वैरिएंट है जो कस्टम कैरेक्टर सेट का उपयोग करके संख्यात्मक डेटा एन्कोड करता है, एक विशिष्ट दृश्य शैली प्रदान करता है जबकि अन्य मोड्स के समान एरर‑करेक्शन शक्ति बनाए रखता है। यह तब उपयोगी है जब आपको संख्यात्मक जानकारी पर ज़ोर देने वाला बारकोड चाहिए।

```csharp
gen.CodeText = "123"; // Change the code text if needed
gen.Parameters.Barcode.Aztec.AztecSymbolMode = AztecSymbolMode.Rune;
gen.Save($"{path}AztecSymbolModeRune.png", BarCodeImageFormat.Png);
```

यह चरण कोड टेक्स्ट को "123" में बदलता है और Rune Aztec Symbol Mode के साथ एक बारकोड जनरेट करता है।
