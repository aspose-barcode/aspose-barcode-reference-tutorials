# कोड128 बारकोड को खाली बार के साथ जावा में बनाना

## परिचय

बारकोड जनरेशन को जावा एप्लिकेशन में एकीकृत करना अक्सर आवश्यक होता है, चाहे आप इन्वेंटरी सिस्टम, टिकटिंग प्लेटफ़ॉर्म, या पॉइंट‑ऑफ़‑सेल समाधान बना रहे हों। **Aspose.BarCode for Java** के साथ, आप केवल कुछ लाइनों के कोड में **Code128 बारकोड** छवियां बना सकते हैं। इस ट्यूटोरियल में हम उन चरणों को देखेंगे जो **बारकोड जावा** छवियां उत्पन्न करते हैं जिनमें खाली (भरे नहीं हुए) बार होते हैं, जिससे आपको बारकोड की दृश्य शैली पर पूर्ण नियंत्रण मिलता है।

## त्वरित उत्तर
- **कौन सी लाइब्रेरी?** Aspose.BarCode for Java  
- **मुख्य उपयोग केस?** कस्टम UI डिज़ाइनों के लिए खाली बार के साथ Code128 बारकोड बनाना  
- **पूर्वापेक्षाएँ?** Java JDK 8+, Aspose.BarCode JAR, आउटपुट छवि के लिए फ़ोल्डर में लिखने की अनुमति  
- **आम कार्यान्वयन समय?** बेसिक उदाहरण के लिए 5‑10 मिनट  
- **क्या मैं रंग कस्टमाइज़ कर सकता हूँ?** हाँ – API विस्तृत स्टाइलिंग विकल्प प्रदान करता है  

## कोड128 बारकोड क्या है?

Code128 एक हाई‑डेंसिटी लीनियर बारकोड सिम्बोलॉजी है जो पूरे ASCII कैरेक्टर सेट को एन्कोड करती है। यह लॉजिस्टिक्स, शिपिंग और रिटेल में व्यापक रूप से उपयोग होती है क्योंकि यह अल्फ़ान्यूमेरिक डेटा को कॉम्पैक्ट रूप में प्रस्तुत कर सकती है। डिफ़ॉल्ट रूप से, Aspose.BarCode Code128 बार को ठोस आयतों के रूप में रेंडर करता है; **FilledBars** प्रॉपर्टी को `false` सेट करने से वे खाली (आउटलाइन) बार बन जाते हैं, जो स्टाइलिश ब्रांडिंग के लिए उपयोगी है।

## जावा के लिए Aspose.BarCode क्यों उपयोग करें?

- **समृद्ध फीचर सेट** – 50 से अधिक बारकोड प्रकारों का समर्थन करता है, जिसमें QR, DataMatrix, और PDF417 शामिल हैं।  
- **कोई बाहरी निर्भरताएँ नहीं** – एक सिंगल JAR किसी भी जावा रनटाइम पर काम करता है।  
- **सूक्ष्म नियंत्रण** – आप बार की चौड़ाई, रंग, फ़ॉन्ट और `FilledBars` प्रॉपर्टी को समायोजित कर सकते हैं।  
- **क्रॉस‑प्लेटफ़ॉर्म** – Windows, Linux, और macOS पर बिना नेटिव लाइब्रेरी के काम करता है।

## पूर्वापेक्षाएँ

शुरू करने से पहले, सुनिश्चित करें कि आपके पास है:

1. **जावा विकास पर्यावरण** – JDK 8 या उससे बाद का स्थापित और `JAVA_HOME` कॉन्फ़िगर किया हुआ।  
2. **Aspose.BarCode for Java** – नवीनतम JAR को [download page](https://releases.aspose.com/barcode/java/) से डाउनलोड करें।  
3. **डॉक्यूमेंट डायरेक्टरी** – आपके कंप्यूटर पर एक फ़ोल्डर जहाँ उत्पन्न बारकोड छवि सहेजी जाएगी।

## पैकेज आयात करें

अपने जावा प्रोजेक्ट में, उन क्लासों को आयात करें जिनकी आपको Aspose.BarCode के साथ काम करने के लिए आवश्यकता होगी:

```java
import java.io.IOException;
import com.aspose.barcode.BarCodeImageFormat;
import com.aspose.barcode.generation.BarcodeGenerator;
```

ये इम्पोर्ट्स आपको बारकोड जेनरेटर, इमेज फॉर्मेट एनेमरेशन, और एक्सेप्शन हैंडलिंग तक पहुंच देते हैं।

## चरण 1: संसाधन निर्देशिका सेट करें

```java
// The path to the resource directory.
String dataDir = "Your Document Directory";
```

`"Your Document Directory"` को उस फ़ोल्डर के पूर्ण पाथ से बदलें जिसे आपने पूर्वापेक्षाओं में बनाया था। यही वह स्थान है जहाँ PNG फ़ाइल लिखी जाएगी।

## चरण 2: कोड128 बारकोड जेनरेटर इंस्टेंस बनाएं

```java
// Create an instance of BarcodeGenerator and initialize it with CodeText and Symbology
BarcodeGenerator generator = new BarcodeGenerator(com.aspose.barcode.EncodeTypes.CODE_128, "TEXT");
```

यहाँ हम `BarcodeGenerator` को **CODE_128** सिम्बोलॉजी और सैंपल टेक्स्ट `"TEXT"` के साथ इंस्टैंशिएट करते हैं। यह एक **barcode generator Java example** है जिसे आप किसी भी स्ट्रिंग के लिए अनुकूलित कर सकते हैं जिसे आप एन्कोड करना चाहते हैं।

## चरण 3: FilledBars प्रॉपर्टी को False सेट करें

```java
// Set the FilledBars property to false
generator.getParameters().getBarcode().setFilledBars(false);
```

`FilledBars` को `false` सेट करने से उत्पन्न बारकोड ठोस बार की बजाय खाली बार दिखाएगा, जिससे हम चाहते हुए दृश्य प्रभाव प्राप्त करते हैं।

## चरण 4: बारकोड छवि सहेजें

```java
// Save the resultant barcode image on disk
generator.save(dataDir + "barcodeWithEmptyBars.png", BarCodeImageFormat.PNG);
```

`save` मेथड बारकोड को निर्दिष्ट डायरेक्टरी में PNG फॉर्मेट में लिखता है। यदि कोई अलग इमेज टाइप चाहिए तो आप `BarCodeImageFormat.PNG` को `JPG`, `BMP`, या `TIFF` में बदल सकते हैं।

## सामान्य उपयोग केस

- **कस्टम ब्रांडिंग** – खाली बार को बैकग्राउंड पैटर्न या रंगों के साथ ओवरले किया जा सकता है।  
- **प्रिंट‑रेडी एसेट्स** – इनवॉइस या शिपिंग लेबल में शामिल करने के लिए हाई‑रेज़ोल्यूशन PNG बनाएं।  
- **डायनामिक जनरेशन** – वेब सर्विसेज या डेस्कटॉप एप्लिकेशन्स में तुरंत बारकोड बनाएं।

## समस्या निवारण टिप्स

- **अमान्य पाथ** – सुनिश्चित करें कि `dataDir` फ़ाइल सेपरेटर (`/` या `\\`) पर समाप्त हो या `File.separator` का उपयोग करके जोड़ें।  
- **लाइसेंस आवश्यक** – ट्रायल मूल्यांकन के लिए काम करता है, लेकिन प्रोडक्शन के लिए पूर्ण लाइसेंस चाहिए।  
- **असमर्थित कैरेक्टर** – Code128 पूरे ASCII सेट को सपोर्ट करता है; यूनिकोड कैरेक्टर से बचें जब तक आप किसी अन्य सिम्बोलॉजी पर न जाएँ।

## निष्कर्ष

आपने अब **Code128 बारकोड** छवियों को खाली बार के साथ Aspose.BarCode for Java का उपयोग करके बनाना सीख लिया है। यह सरल तरीका आपको कस्टमाइज़ेबल बारकोड ग्राफिक्स जल्दी से जनरेट करने देता है, चाहे आप डेस्कटॉप इन्वेंटरी टूल बना रहे हों या क्लाउड‑आधारित API।
