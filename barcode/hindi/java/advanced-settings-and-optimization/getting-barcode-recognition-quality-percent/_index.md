# Aspose.Barcode Java – बारकोड पहचान गुणवत्ता प्रतिशत में प्राप्त करना

## परिचय

यदि आपको Java एप्लिकेशन में **बारकोड पढ़ने की गुणवत्ता** का **मूल्यांकन** करने की आवश्यकता है, तो **Aspose.Barcode Java** एक सरल API प्रदान करता है जो पहचान गुणवत्ता को प्रतिशत के रूप में लौटाता है। इस ट्यूटोरियल में हम उस प्रतिशत को प्राप्त करने के लिए आवश्यक सटीक चरणों को समझाएंगे, यह बताएँगे कि यह मीट्रिक क्यों महत्वपूर्ण है, और दिखाएँगे कि इसे आपके मौजूदा कोड बेस में कैसे एकीकृत किया जाए।

## त्वरित उत्तर
- **“पढ़ने की गुणवत्ता” का क्या अर्थ है?** यह वह आत्मविश्वास स्कोर (0‑100 %) है जो लाइब्रेरी प्रत्येक डिकोड किए गए बारकोड को देती है।  
- **कौन सा लाइब्रेरी संस्करण आवश्यक है?** कोई भी नवीनतम Aspose.Barcode Java रिलीज़ (उदाहरण में नवीनतम 24.x श्रृंखला उपयोग की गई है)।  
- **क्या मुझे लाइसेंस चाहिए?** परीक्षण के लिए एक अस्थायी लाइसेंस काम करता है; उत्पादन के लिए पूर्ण लाइसेंस आवश्यक है।  
- **क्या मैं सभी बारकोड प्रकार पढ़ सकता हूँ?** हाँ – `DecodeType.ALL_SUPPORTED_TYPES` फ़्लैग Aspose.Barcode द्वारा समर्थित सभी फ़ॉर्मेट को सक्षम करता है।  
- **क्या QR कोड के लिए गुणवत्ता मान भरोसेमंद है?** बिल्कुल – वही आत्मविश्वास एल्गोरिद्म 1‑D और 2‑D दोनों सिम्बोलॉजी पर लागू होता है।

## Aspose.Barcode Java क्या है और बारकोड पढ़ने की गुणवत्ता का मूल्यांकन कैसे करें?

**Aspose.Barcode Java** एक पूरी तरह प्रबंधित लाइब्रेरी है जो डेवलपर्स को बाहरी निर्भरताओं के बिना बारकोड उत्पन्न करने, पढ़ने और विश्लेषण करने देती है। इसकी सबसे उपयोगी डायग्नोस्टिक्स में से एक **पढ़ने की गुणवत्ता** मीट्रिक है, जो बताता है कि इंजन ने किसी प्रतीक को कितनी आत्मविश्वास से डिकोड किया। जब आपको यह तय करना हो कि स्कैन को स्वीकार किया जाए, पुनः‑कैप्चर का अनुरोध किया जाए, या त्रुटि‑हैंडलिंग लॉजिक को ट्रिगर किया जाए, तो यह मीट्रिक आवश्यक है।

## बारकोड पढ़ने की गुणवत्ता के लिए Aspose.Barcode Java क्यों उपयोग करें?

- **सभी समर्थित सिम्बोलॉजी में लगातार आत्मविश्वास स्कोर**।  
- **कोई नेटिव DLL नहीं** – शुद्ध Java, इसलिए यह किसी भी JVM‑संगत प्लेटफ़ॉर्म पर काम करता है।  
- **सूक्ष्म नियंत्रण**: आप प्रत्येक बारकोड के लिए गुणवत्ता प्राप्त कर सकते हैं, न कि केवल एक समग्र पास/फ़ेल।  
- **प्रदर्शन‑अनुकूलित** पढ़ने वाला इंजन जो डेस्कटॉप से क्लाउड सेवाओं तक स्केल करता है।

## पूर्वापेक्षाएँ

शुरू करने से पहले, सुनिश्चित करें कि आपके पास है:

- **Java विकास पर्यावरण** – JDK 8 या नया, आपके पसंदीदा IDE (IntelliJ, Eclipse, VS Code, आदि) के साथ।  
- **Aspose.Barcode Java लाइब्रेरी** – आधिकारिक साइट से नवीनतम JAR डाउनलोड करें: [Aspose.Barcode for Java](https://releases.aspose.com/barcode/java/).  
- **एक नमूना बारकोड छवि** – ट्यूटोरियल में `code39Extended.jpg` का उपयोग किया गया है जो `BarcodeReader/advanced_features/` फ़ोल्डर में स्थित है।

अब जब सेटअप हो गया है, चलिए कोड में डुबकी लगाते हैं।

## नेमस्पेस आयात करें

निम्नलिखित इम्पोर्ट्स आपको क्वालिटी एक्सट्रैक्शन के लिए आवश्यक रीडर और रिज़ल्ट क्लासेज़ तक पहुँच प्रदान करते हैं।

```java
import com.aspose.barcode.barcoderecognition.BarCodeReader;
import com.aspose.barcode.barcoderecognition.BarCodeResult;
```

### चरण 1: रिसोर्स डायरेक्टरी पाथ सेट करें

सैंपल इमेज वाले फ़ोल्डर को परिभाषित करें। `Utils.getDataDir` एक हेल्पर है जो वर्तमान प्रोजेक्ट के लिए एब्सोल्यूट पाथ को रिजॉल्व करता है।

```java
// The path to the resource directory.
String dataDir = Utils.getDataDir(GetBarCodeRecognitionQualityInPercent.class)
        + "BarcodeReader/advanced_features/";
```

### चरण 2: BarCodeReader ऑब्जेक्ट को इनिशियलाइज़ करें

`BarCodeReader` इंस्टेंस बनाएं, इसे इमेज फ़ाइल की ओर इंगित करें और इसे **सभी समर्थित बारकोड प्रकार** आज़माने के लिए कहें।

```java
// Initialize the BarCodeReader object
BarCodeReader reader = new BarCodeReader(dataDir + "code39Extended.jpg",
        com.aspose.barcode.barcoderecognition.DecodeType.ALL_SUPPORTED_TYPES);
```

### चरण 3: बारकोड पढ़ें और क्वालिटी प्रतिशत प्राप्त करें

प्रत्येक पहचाने गए बारकोड पर इटररेट करें, उसका टेक्स्ट, प्रकार, और `getReadingQuality()` द्वारा लौटाए गए **पढ़ने की गुणवत्ता** प्रतिशत को प्रिंट करें।

```java
// Call the read method
for (BarCodeResult result : reader.readBarCodes()) {
    System.out.println(result.getCodeText() + " Type: " + result.getCodeType());
    double percent = result.getReadingQuality();
    System.out.println("Percent: " + percent);
}
```

**यहाँ क्या हो रहा है?**  
- `readBarCodes()` एक कलेक्शन लौटाता है जिसमें `BarCodeResult` ऑब्जेक्ट्स होते हैं, प्रत्येक पाए गए बारकोड के लिए एक।  
- `getReadingQuality()` एक `double` देता है जो `0` से `100` के बीच होता है, जो आत्मविश्वास स्तर को दर्शाता है।  
- आप इस मान का उपयोग यह तय करने के लिए कर सकते हैं कि स्कैन स्वीकार्य है या उपयोगकर्ता को पुनः प्रयास करने के लिए कहना चाहिए।
