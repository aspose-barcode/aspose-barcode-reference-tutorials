# बारकोड से टेक्स्ट निकालें: PDF417 चीनी अक्षर जावा में

## परिचय

जावा एप्लिकेशन में बारकोड पहचान को एकीकृत करना एक मूल्यवान कौशल है, विशेष रूप से जब आपको बहुभाषी डेटा वाले **बारकोड छवियों से टेक्स्ट निकालना** हो। इस ट्यूटोरियल में, हम Aspose.BarCode for Java का उपयोग करके PDF417 बारकोड को चीनी अक्षरों के साथ पहचानने की प्रक्रिया दिखाएंगे। अंत तक, आप बारकोड डेटा को पढ़ना और उसे पठनीय टेक्स्ट में डिकोड करना जानेंगे।

## त्वरित उत्तर
- **कौन सी लाइब्रेरी PDF417 के साथ चीनी अक्षर समर्थन करती है?** Aspose.BarCode for Java।  
- **चीनी डिकोडिंग के लिए कौन सा कैरेक्टर सेट आवश्यक है?** MS936 (GBK)।  
- **प्रोडक्शन उपयोग के लिए लाइसेंस चाहिए?** हाँ, एक व्यावसायिक लाइसेंस आवश्यक है।  
- **क्या यह किसी भी जावा संस्करण पर चल सकता है?** यह जावा 8 और उसके बाद के संस्करणों के साथ काम करता है।  
- **क्या मुफ्त ट्रायल उपलब्ध है?** बिल्कुल – इसे Aspose की साइट से डाउनलोड करें।

## “बारकोड से टेक्स्ट निकालना” क्या है?

बारकोड से टेक्स्ट निकालना का अर्थ है एन्कोडेड डेटा को उसके मूल मानव‑पठनीय रूप में बदलना। PDF417 बारकोड जो चीनी अक्षर संग्रहीत करते हैं, उनके लिए सही कैरेक्टर एन्कोडिंग चुनना आवश्यक है ताकि बाइट्स सही glyphs में मैप हो सकें।

## Aspose.BarCode for Java क्यों उपयोग करें?

Aspose.BarCode विभिन्न बारकोड सिम्बोलॉजीज, जिसमें PDF417 भी शामिल है, के लिए मजबूत समर्थन प्रदान करता है और जटिल कैरेक्टर सेट्स को बॉक्स से बाहर संभालता है। यह लो‑लेवल इमेज प्रोसेसिंग को एब्स्ट्रैक्ट करता है, जिससे आप डिकोडिंग जटिलताओं के बजाय बिजनेस लॉजिक पर ध्यान केंद्रित कर सकते हैं।

## पूर्वापेक्षाएँ

शुरू करने से पहले सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

1. **जावा डेवलपमेंट किट (JDK)** – नवीनतम संस्करण अनुशंसित है।  
2. **Aspose.BarCode for Java** – इसे [यहाँ](https://releases.aspose.com/barcode/java/) से डाउनलोड करें।  
3. **एक PDF417 बारकोड इमेज** जिसमें चीनी अक्षर हों (उदाहरण के लिए `barcode.png`)।

## पैकेज इम्पोर्ट करें

अपने जावा प्रोजेक्ट में Aspose.BarCode के साथ काम करने के लिए आवश्यक क्लासेज़ इम्पोर्ट करें:

```java
import java.nio.ByteBuffer;
import java.nio.charset.Charset;

import com.aspose.barcode.barcoderecognition.BarCodeReader;
import com.aspose.barcode.barcoderecognition.BarCodeResult;
import com.aspose.barcode.barcoderecognition.DecodeType;
```

## चरण 1: डॉक्यूमेंट डायरेक्टरी सेट करें

उस फ़ोल्डर को निर्दिष्ट करें जहाँ आपका बारकोड इमेज स्थित है:

```java
String dataDir = "Your Document Directory";
```

`"Your Document Directory"` को अपने मशीन पर वास्तविक पाथ से बदलें।

## चरण 2: बारकोड इमेज लोड करें

एक `BarCodeReader` इंस्टेंस बनाएं जो PNG फ़ाइल की ओर इशारा करे और रीडर को PDF417 सिम्बोलॉजी खोजने के लिए बताएं:

```java
BarCodeReader reader = new BarCodeReader(dataDir + "barcode.png", DecodeType.PDF_417);
```

## चरण 3: बारकोड पढ़ें

डिटेक्शन परिणामों पर इटरेट करें, रॉ बाइट एरे निकालें, और GBK (MS936) कैरेक्टर सेट का उपयोग करके डिकोड करें:

```java
for (BarCodeResult result : reader.readBarCodes()) {
    byte[] bytes = result.getCodeBytes();
    ByteBuffer bytebuf = ByteBuffer.wrap(bytes);
    System.out.println(Charset.forName("MS936").decode(bytebuf).toString());
}
```

यह लूप **बारकोड से टेक्स्ट निकालता** है और डिकोड किए गए चीनी अक्षरों को कंसोल पर प्रिंट करता है।

## PDF417 के साथ बारकोड कैसे पढ़ें?

ऊपर दिया गया कोड **बारकोड डेटा** को चरण‑दर‑चरण पढ़ने का तरीका दर्शाता है:

1. सही `DecodeType` के साथ रीडर को **इनिशियलाइज़** करें।  
2. लौटाए गए प्रत्येक `BarCodeResult` पर **इटरेट** करें।  
3. उपयुक्त charset (`MS936` चीनी के लिए) का उपयोग करके रॉ बाइट्स को **कन्वर्ट** करें।  

यदि आपका बारकोड अन्य भाषाओं को संग्रहीत करता है, तो बस उस डेटा से मेल खाने वाले charset में स्विच करें।

**अतिरिक्त प्रश्न‑उत्तर**

**प्र: यदि मेरा बारकोड इमेज JPEG फॉर्मेट में है तो क्या होगा?**  
**उ:** `BarCodeReader` JPEG, PNG, BMP और अन्य सामान्य इमेज फॉर्मेट्स को सपोर्ट करता है—सिर्फ फ़ाइल एक्सटेंशन को उसी अनुसार बदलें।

**प्र: क्या मैं फ़ाइल के बजाय स्ट्रीम से बारकोड डिकोड कर सकता हूँ?**  
**उ:** हाँ, आप `BarCodeReader` कंस्ट्रक्टर में `InputStream` पास करके ऑन‑द‑फ़्लाई डिकोडिंग कर सकते हैं।

**प्र: क्या Aspose.BarCode अन्य कैरेक्टर सेट्स को सपोर्ट करता है?**  
**उ:** बिल्कुल। बाइट्स को UTF‑8, ISO‑8859‑1 आदि के लिए डिकोड करने हेतु `Charset.forName("<your‑charset>")` का उपयोग करें।
