# code128 barkod oluştur java: Barkod Görüntüsü için Boyut Birimini Ayarla

## Giriş

Bu adım‑adım öğreticide **Aspose.BarCode for Java**'ı **create code128 barcode java** oluşturmak, bir barcode image java üretmek ve çıktının boyut birimini hassas bir şekilde kontrol etmek için kullanacaksınız. Envanter sistemi, gönderi etiketi oluşturucu veya güvenilir barkodlara ihtiyaç duyan herhangi bir Java‑tabanlı uygulama geliştiriyor olun, Aspose.BarCode sadece birkaç satır kodla tam esneklik sağlar.

- **Hangi barkod türü kapsanıyor?** CODE_128 (create code128 barcode java).  
- **Boyut birimini nasıl ayarlarım?** `BarHeight` özelliğini `.setPoint()` ile kullanın.  
- **barcode image java'yi başka formatlarda oluşturabilir miyim?** Evet – PNG, JPEG, BMP, vb.  
- **Önkoşullar nelerdir?** JDK kurulu, Aspose.BarCode kütüphanesi ve bir Java IDE.  

## **create code128 barcode java** nedir?

`create code128 barcode java`, `BarcodeGenerator` sınıfını `EncodeTypes.CODE_128` enumu ile kullanarak bir veri dizesini CODE_128 sembolojisine kodlamak anlamına gelir. Bu semboloji tam ASCII setini destekler ve yüksek veri yoğunluğu sunar, bu da envanter sistemi ve gönderi etiketi senaryoları için barkod oluşturmayı ideal kılar.

## Aspose.BarCode'u **generate barcode image java** oluşturmak için neden kullanmalısınız?

## Aspose.BarCode ile **generate barcode image java** nasıl oluşturulur?

`BarcodeGenerator`'ınızı yükleyin, boyutu yapılandırın ve görüntüyü kaydedin – toplu işleme için bir döngü içinde sarılabilecek net, lineer bir akışta. Bu doğrudan‑cevap paragrafı, adım‑adım detaylara girmeden önce tam olarak ne yapmanız gerektiğini söyler.

### Adım 1: Kaynak Dizinini Tanımla

İlk olarak, oluşturulan dosyaların yazılacağı klasörü belirtin. Bu dizin mevcut olmalı ve Java süreci tarafından yazılabilir olmalıdır.

```java
// The path to the resource directory.
String dataDir = "Your Document Directory";
```

`"Your Document Directory"` ifadesini barkod görüntüsünün kaydedileceği mutlak yol ile değiştirin. Bu klasör, daha sonra faturalar, paketleme fişleri veya envanter raporlarına ekleyebileceğiniz oluşturulan PNG/JPEG dosyalarını tutacaktır.

### Adım 2: Barkod Nesnesini Oluştur

`EncodeTypes.CODE_128`, CODE_128 barkod sembolojisini belirtir.

`BarcodeGenerator`, bellek içinde bir barkod görüntüsünü temsil eden Aspose.BarCode'un temel sınıfıdır. Bir `create code128 barcode java` örneği oluşturmak, `EncodeTypes.CODE_128` enumunu ve kodlamak istediğiniz veri dizesini geçirmek kadar basittir.

```java
// Instantiate barcode object, Set the symbology type to code128 and Set the
// Code text for the barcode
BarcodeGenerator bb = new BarcodeGenerator(EncodeTypes.CODE_128, "1234567");
```

Burada `EncodeTypes.CODE_128` ve veri dizesi "1234567" geçirerek **create code128 barcode java** oluşturuyoruz.

### Adım 3: Bar Height (Boyut Birimi) Ayarla

`BarHeight`, çubukların dikey boyutunu tanımlar. `.setPoint()` metodu bu yüksekliği puan cinsinden ayarlar (1 puan = 1/72 inç), size son görsel boyut üzerinde hassas kontrol sağlar.

```java
// Set the bar height to 3 points
bb.getParameters().getBarcode().getBarHeight().setPoint(3.0f);
```

`setPoint()` metodu yüksekliği puan olarak tanımlar. Bu değeri düzenleme gereksinimlerinize göre ayarlayın—daha büyük değerler daha uzun çubuklar üretir, bu genellikle yüksek çözünürlüklü gönderi etiketleri için gerekir.

### Adım 4: Görüntüyü Kaydet

`save()` çağrısı, barkodu belirttiğiniz klasöre yazar. Görüntü formatı dosya uzantısından çıkarılır; uzantıyı değiştirerek PNG, BMP veya TIFF'e geçebilirsiniz.

```java
// Save the image
bb.save(dataDir + "barcode-size-unit.jpg");
```

`save()` çağrısı, oluşturulan barkodu belirttiğiniz klasöre yazar. Görüntü formatı dosya uzantısından çıkarılır (bu durumda JPEG). Uzantıyı değiştirerek PNG, BMP veya TIFF'e geçebilirsiniz.
