# Buat Planet Barcode di C# – Panduan Langkah‑per‑Langkah

Pernah perlu **membuat planet barcode** tetapi tidak yakin harus mulai dari mana? Dalam tutorial ini Anda akan melihat secara tepat **cara menghasilkan planet barcode** menggunakan pustaka Aspose.Barcode, dengan opsi tinggi default dan kustom.

Kami akan menelusuri seluruh proses—dari menginstal paket NuGet hingga menyesuaikan tinggi bar dan menyimpan file PNG. Pada akhir tutorial Anda akan memiliki empat gambar siap pakai: dua Planet barcode (otomatis dan tinggi 100 pixel) serta dua RM4SCC barcode untuk perbandingan. Tanpa basa‑basi, hanya kode praktis yang dapat Anda sisipkan ke proyek .NET mana pun.

## Prasyarat

* .NET 6.0 atau lebih baru (kode ini juga berfungsi pada .NET Framework 4.7+)
* Visual Studio 2022 (atau IDE lain yang Anda sukai)
* Paket NuGet **Aspose.Barcode for .NET**  
  ```bash
  dotnet add package Aspose.Barcode
  ```
* Sebuah folder di disk tempat file PNG yang dihasilkan dapat ditulis (ganti `YOUR_DIRECTORY` dalam kode).

Jika ada yang belum Anda kenal, jangan khawatir—menginstal paket NuGet semudah klik kanan pada proyek → *Manage NuGet Packages* → cari “Aspose.Barcode” → *Install*.

## Langkah 1 – Buat Proyek Konsol Minimal

```csharp
using System;
using Aspose.BarCode.Generation;

namespace PlanetBarcodeDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            // We'll call helper methods here – see the sections below.
            GeneratePlanetBarcodes();
            GenerateRm4sccBarcodes();
            Console.WriteLine("All barcodes generated successfully.");
        }

        // Helper methods are defined later.
    }
}
```

> **Pro tip:** Jaga metode `Main` tetap rapi; delegasikan pekerjaan berat ke metode terpisah. Ini membuat kode lebih mudah diuji dan dibaca nanti.

## Langkah 2 – **Buat Planet Barcode** dengan Tinggi Default

Sekarang kita benar‑benar **membuat planet barcode** menggunakan tinggi default (yang dihitung otomatis). `XDimension` mengontrol lebar satu modul, sementara tinggi bar dibiarkan ditentukan oleh pustaka.

```csharp
static void GeneratePlanetBarcodes()
{
    // 1️⃣ Default height
    var planetDefault = new BarcodeGenerator(EncodeTypes.Planet, "123456");
    planetDefault.Parameters.Barcode.XDimension.Pixels = 4; // narrow modules
    planetDefault.Save("YOUR_DIRECTORY/PlanetBarHeightAuto.png", BarCodeImageFormat.Png);
}
```

*Mengapa ini berhasil?*  
`EncodeTypes.Planet` memberi tahu Aspose bahwa kita menginginkan simbologi Planet, dan `"123456"` adalah data yang akan dikodekan. Dengan tidak menyetel `BarHeight`, mesin secara otomatis memilih tinggi yang memenuhi spesifikasi.

### Output yang Diharapkan

Buka `PlanetBarHeightAuto.png` dan Anda akan melihat Planet barcode yang bersih, kira‑kira setinggi 30 mm (ukuran pasti tergantung DPI penulis PNG). Tidak perlu mengatur tinggi secara manual.

## Langkah 3 – **Cara Menghasilkan Planet Barcode** dengan Tinggi 100‑Pixel Kustom

Jika desain Anda memerlukan tinggi tetap—misalnya 100 pixel—cukup set properti `BarHeight` sebelum menyimpan.

```csharp
static void GeneratePlanetBarcodes()
{
    // 1️⃣ Default height (already shown)
    var planetDefault = new BarcodeGenerator(EncodeTypes.Planet, "123456");
    planetDefault.Parameters.Barcode.XDimension.Pixels = 4;
    planetDefault.Save("YOUR_DIRECTORY/PlanetBarHeightAuto.png", BarCodeImageFormat.Png);

    // 2️⃣ Custom 100‑pixel height
    var planetCustom = new BarcodeGenerator(EncodeTypes.Planet, "123456");
    planetCustom.Parameters.Barcode.XDimension.Pixels = 4;
    planetCustom.Parameters.Barcode.BarHeight.Pixels = 100; // explicit height
    planetCustom.Save("YOUR_DIRECTORY/PlanetBarHeight100.png", BarCodeImageFormat.Png);
}
```

Perhatikan baris tambahan `planetCustom.Parameters.Barcode.BarHeight.Pixels = 100;`. Baris ini memaksa barcode memperluas (atau menyusut) tepat menjadi 100 pixel tinggi, terlepas dari panjang data.

> **Watch out:** Menetapkan tinggi yang terlalu kecil dapat menyebabkan bar saling menempel, membuat barcode tidak dapat dibaca. Aturan praktis yang aman adalah menjaga tinggi setidaknya tiga kali X‑dimension.

## Langkah 4 – Hasilkan Barcode RM4SCC untuk Perbandingan

Kadang‑kadang Anda perlu membandingkan Planet dengan simbologi pos lainnya. RM4SCC adalah format umum lain; kode di bawah ini meniru langkah‑langkah yang kami gunakan untuk Planet.

```csharp
static void GenerateRm4sccBarcodes()
{
    // Default height
    var rm4sccDefault = new BarcodeGenerator(EncodeTypes.RM4SCC, "123456");
    rm4sccDefault.Parameters.Barcode.XDimension.Pixels = 4;
    rm4sccDefault.Save("YOUR_DIRECTORY/RM4SCCBarHeightAuto.png", BarCodeImageFormat.Png);

    // Custom 100‑pixel height
    var rm4sccCustom = new BarcodeGenerator(EncodeTypes.RM4SCC, "123456");
    rm4sccCustom.Parameters.Barcode.XDimension.Pixels = 4;
    rm4sccCustom.Parameters.Barcode.BarHeight.Pixels = 100;
    rm4sccCustom.Save("YOUR_DIRECTORY/RM4SCCBarHeight100.png", BarCodeImageFormat.Png);
}
```

Menjalankan program lengkap sekarang menghasilkan **empat** file PNG:

| File | Simbologi | Tinggi |
|------|-----------|--------|
| `PlanetBarHeightAuto.png` | Planet | Auto |
| `PlanetBarHeight100.png` | Planet | 100 px |
| `RM4SCCBarHeightAuto.png` | RM4SCC | Auto |
| `RM4SCCBarHeight100.png` | RM4SCC | 100 px |

Anda dapat membuka masing‑masing gambar di penampil apa pun untuk mengonfirmasi dimensi dan kualitas visualnya.

## Langkah 5 – Verifikasi Barcode Secara Programatis (Opsional)

Jika Anda ingin memastikan barcode yang dihasilkan dapat dipindai, Aspose juga menyediakan komponen **reader**. Berikut pemeriksaan cepat bahwa data yang dikodekan cocok dengan string asli kami.

```csharp
using Aspose.BarCode.Reader;

static void VerifyBarcodes()
{
    var reader = new BarCodeReader("YOUR_DIRECTORY/PlanetBarHeight100.png", DecodeType.Planet);
    foreach (BarCodeResult result in reader.ReadBarCodes())
    {
        Console.WriteLine($"Decoded value: {result.CodeText}");
    }
}
```

Menjalankan `VerifyBarcodes()` mencetak `Decoded value: 123456`, mengonfirmasi bahwa tinggi kustom tidak merusak proses enkoding.

## Kesalahan Umum & Tips

| Masalah | Mengapa Terjadi | Solusi |
|---------|----------------|--------|
| Barcode terlihat terlalu tipis | X‑dimension diatur terlalu rendah (misalnya 1 pixel) | Tingkatkan `XDimension.Pixels` menjadi 3‑5 untuk kebanyakan printer |
| Gambar blur | Menyimpan sebagai PNG dengan DPI default (96) pada layar beresolusi tinggi | Gunakan `ImageOptions` untuk mengatur `ResolutionX` dan `ResolutionY` menjadi 300 DPI |
| Tinggi tidak diterapkan | `BarHeight` disetel *setelah* memanggil `Save()` | Selalu set `BarHeight` **sebelum** `Save()` |
| Path file tidak valid | `YOUR_DIRECTORY` tidak ada | Buat folder secara programatis: `Directory.CreateDirectory("YOUR_DIRECTORY");` |
