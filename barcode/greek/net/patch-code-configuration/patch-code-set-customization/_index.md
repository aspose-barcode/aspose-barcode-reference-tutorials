# Δημιουργία Πολλαπλών Γραμμωτών – Προσαρμογή Συνόλου Patch Code

Σε αυτό το tutorial θα **δημιουργήσετε πολλαπλούς γραμμωτούς κώδικες** με το Aspose.BarCode for .NET, εστιάζοντας στην οικογένεια Patch Code. Είτε δημιουργείτε σύστημα διαχείρισης εγγράφων είτε χρειάζεστε ετικέτες για περιουσιακά στοιχεία, η δημιουργία πολλών εικόνων γραμμωτών κωδίκων ταυτόχρονα εξοικονομεί χρόνο και μειώνει τα σφάλματα. Θα περάσουμε από τις προαπαιτήσεις, θα εισάγουμε τα απαιτούμενα namespaces, και στη συνέχεια θα δείξουμε ένα βήμα‑βήμα παράδειγμα που σας επιτρέπει να **προσαρμόσετε τις τιμές του patch barcode** και να **αποθηκεύσετε αρχεία PNG γραμμωτού** στο δίσκο.

## Γρήγορες Απαντήσεις
- **Τι καλύπτει αυτός ο οδηγός;** Δημιουργία πολλαπλών γραμμωτών κωδίκων Patch Code, προσαρμογή του κειμένου τους, και αποθήκευση ως εικόνες PNG.  
- **Ποια βιβλιοθήκη χρησιμοποιείται;** Aspose.BarCode for .NET.  
- **Χρειάζομαι άδεια;** Μια δωρεάν δοκιμή λειτουργεί για δοκιμές· απαιτείται εμπορική άδεια για παραγωγή.  
- **Ποιες εκδόσεις .NET υποστηρίζονται;** .NET Framework 4.5+ και .NET Core/5/6+.  
- **Πόσους γραμμωτούς κώδικες μπορώ να δημιουργήσω;** Οποιονδήποτε αριθμό – απλώς αλλάξτε την ιδιότητα `CodeText` και καλέστε `Save` για κάθε εικόνα.

## Τι σημαίνει “δημιουργία πολλαπλών γραμμωτών κωδίκων” με Patch Code;
Τα γραμμωτά Patch Code είναι μια συμπαγής, υψηλής πυκνότητας συμβολοσειρά που χρησιμοποιείται συχνά για παρακολούθηση εγγράφων. Αλλάζοντας την ιδιότητα `CodeText` ενός μόνο αντικειμένου `BarcodeGenerator`, μπορείτε να **δημιουργήσετε πολλαπλούς γραμμωτούς κώδικες** σε βρόχο ή σειρά εντολών, ο καθένας αποθηκευμένος ως ξεχωριστό αρχείο PNG.

## Γιατί να χρησιμοποιήσετε το Aspose.BarCode .NET για δημιουργία εικόνων γραμμωτών κωδίκων;
- **Πλήρης API** – υποστηρίζει δεκάδες συμβολισμούς, συμπεριλαμβανομένου του Patch Code.  
- **Χωρίς εξωτερικές εξαρτήσεις** – καθαρή βιβλιοθήκη .NET, εύκολη ενσωμάτωση.  
- **Πλούσια προσαρμογή** – χρώματα, γραμματοσειρές, μεγέθη και μορφές εικόνας είναι όλα ρυθμιζόμενα.  
- **Αξιόπιστο αποτέλεσμα** – δημιουργεί καθαρές, αναγνώσιμες εικόνες κατάλληλες για παραγωγή.

## Εισαγωγή Namespaces

Πριν βυθιστείτε στα παραδείγματα κώδικα, πρέπει να εισάγετε τα απαραίτητα namespaces ώστε η βιβλιοθήκη Aspose.BarCode να είναι διαθέσιμη στο έργο σας. Δείτε πώς μπορείτε να το κάνετε:

### Βήμα 1: Ανοίξτε το .NET Project σας
Εκκινήστε το Visual Studio και ανοίξτε το .NET project όπου θέλετε να χρησιμοποιήσετε το Aspose.BarCode.

### Βήμα 2: Προσθήκη Αναφορών
Κάντε δεξί κλικ στο έργο σας στον Solution Explorer, επιλέξτε **Add** > **Reference**, και περιηγηθείτε στη βιβλιοθήκη Aspose.BarCode που κατεβάσατε νωρίτερα.

### Βήμα 3: Εισαγωγή Namespaces
Στο αρχείο κώδικα, προσθέστε τα παρακάτω namespaces στην αρχή:

```csharp
using Aspose.BarCode;
using Aspose.BarCode.Generation;
```

Τώρα που έχετε τις προαπαιτήσεις και τα namespaces, ας προχωρήσουμε στο βασικό παράδειγμα που δείχνει **πώς να δημιουργήσετε εικόνες γραμμωτών κωδίκων** για διάφορες παραλλαγές Patch Code.

## Πώς να δημιουργήσετε πολλαπλούς γραμμωτούς κώδικες – Οδηγός Βήμα‑Βήμα

### Βήμα 1: Ρύθμιση Διαδρομής Καταλόγου
Ξεκινήστε ορίζοντας τη διαδρομή του καταλόγου όπου θέλετε να αποθηκεύσετε τις παραγόμενες εικόνες γραμμωτών κωδίκων. Αντικαταστήστε το `"Your Directory Path"` με την επιθυμητή τοποθεσία φακέλου.

```csharp
string path = "Your Directory Path";
```

### Βήμα 2: Αρχικοποίηση του Barcode Generator
Θα χρησιμοποιήσουμε την κλάση `BarcodeGenerator` για να δημιουργήσουμε γραμμωτούς κώδικες Patch Code. Αρχικοποιήστε το γεννήτρια με τον τύπο γραμμωτού κώδικα και ένα αρχικό κείμενο κώδικα:

```csharp
BarcodeGenerator gen = new BarcodeGenerator(EncodeTypes.PatchCode, "Patch I");
```

### Βήμα 3: Προσαρμογή Παραμέτρων Κειμένου Κώδικα
Μπορείτε να προσαρμόσετε τις παραμέτρους του κειμένου κώδικα του γραμμωτού κώδικα. Εδώ, ορίζουμε το μέγεθος γραμματοσειράς στα 20 pixel ώστε το κείμενο να είναι ευκρινώς αναγνώσιμο:

```csharp
gen.Parameters.Barcode.CodeTextParameters.FontMode = FontMode.Manual;
gen.Parameters.Barcode.CodeTextParameters.Font.Size.Pixels = 20;
```

### Βήμα 4: Δημιουργία και Αποθήκευση Γραμμωτών Κωδίκων
Τώρα αλλάζουμε την ιδιότητα `CodeText` για κάθε παραλλαγή και **αποθηκεύουμε αρχεία PNG γραμμωτού**. Αυτό είναι το τμήμα όπου πραγματικά **δημιουργούμε πολλαπλούς γραμμωτούς κώδικες** σε μία εκτέλεση:

```csharp
// Patch I
gen.CodeText = "Patch I";
gen.Save($"{path}PatchCodeI.png", BarCodeImageFormat.Png);

// Patch II
gen.CodeText = "Patch II";
gen.Save($"{path}PatchCodeII.png", BarCodeImageFormat.Png);

// Patch III
gen.CodeText = "Patch III";
gen.Save(`${path}PatchCodeIII.png`, BarCodeImageFormat.Png);

// Patch IV
gen.CodeText = "Patch IV";
gen.Save(`${path}PatchCodeIV.png`, BarCodeImageFormat.Png);

// Patch T
gen.CodeText = "Patch T";
gen.Save(`${path}PatchCodeT.png`, BarCodeImageFormat.Png);

// Patch VI
gen.CodeText = "Patch VI";
gen.Save(`${path}PatchCodeVI.png`, BarCodeImageFormat.Png);
```

> **Συμβουλή:** Αν χρειάζεται να δημιουργήσετε δεκάδες γραμμωτούς κώδικες Patch Code, τυλίξτε το τελευταίο μπλοκ σε έναν βρόχο `foreach` που διατρέχει μια συλλογή από συμβολοσειρές κώδικα.

Συγχαρητήρια! Έχετε δημιουργήσει με επιτυχία **πολλαπλούς γραμμωτούς κώδικες** με το Aspose.BarCode for .NET. Το ίδιο μοτίβο λειτουργεί για οποιονδήποτε άλλο υποστηριζόμενο συμβολισμό—απλώς αλλάξτε το `EncodeTypes.PatchCode` στον επιθυμητό τύπο.

## Συνηθισμένα Προβλήματα & Επίλυση

| Συμπτωμα | Πιθανή Αιτία | Διόρθωση |
|----------|--------------|----------|
| Τα αρχεία PNG είναι κενά | Η διαδρομή του φακέλου εξόδου είναι άκυρη ή λείπει το τελικό slash | Επαληθεύστε ότι το `path` τελειώνει με ανάστροφο κάθετο (`\\`) ή χρησιμοποιήστε το `Path.Combine`. |
| Ο γραμμωτός κώδικας φαίνεται θολός | Η μορφή εικόνας έχει οριστεί σε χαμηλό DPI | Ρυθμίστε το `gen.Parameters.ImageResolution` πριν την αποθήκευση. |
| Το κείμενο κόβεται | Το μέγεθος γραμματοσειράς είναι πολύ μεγάλο για το μέγεθος του γραμμωτού κώδικα | Μειώστε το `Font.Size.Pixels` ή αυξήστε τις διαστάσεις του γραμμωτού κώδικα μέσω του `gen.Parameters.ImageSize`. |
