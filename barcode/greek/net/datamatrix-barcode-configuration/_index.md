## Συχνές Ερωτήσεις

**Q: Πώς αποφασίζω ποια λειτουργία ECC να χρησιμοποιήσω;**  
A: Επιλέξτε ECC 000‑140 για μικρά σύνολα δεδομένων με περιορισμένη διόρθωση σφαλμάτων, ή ECC 200 για μεγαλύτερα δεδομένα και υψηλότερη αξιοπιστία. Η λειτουργία Macro προσθέτει επιπλέον στρώματα δεδομένων.

**Q: Μπορώ να ενσωματώσω προσαρμοσμένο κείμενο σε barcode DataMatrix;**  
A: Ναι, χρησιμοποιήστε την ιδιότητα `CodeText` για να ορίσετε εκτεταμένο κείμενο κώδικα, στη συνέχεια διαμορφώστε τη λειτουργία κωδικοποίησης όπως απαιτείται.

**Q: Υπάρχει τρόπος να επιλέγεται αυτόματα η καλύτερη λειτουργία κωδικοποίησης;**  
A: Ορίστε την `EncodeMode` σε `Auto` και το Aspose.BarCode θα αξιολογήσει την είσοδο και θα επιλέξει τη βέλτιστη λειτουργία (ASCII, C40, Bytes, κ.λπ.).

**Q: Ποιες είναι οι επιδόσεις για μεγάλες παρτίδες barcode;**  
A: Επαναχρησιμοποιήστε το αντικείμενο `BarCodeGenerator` και ενεργοποιήστε το multi‑threading όπου είναι δυνατόν. Επίσης, δημιουργήστε εικόνες σε κατάλληλη μορφή (PNG για lossless, JPEG για μικρότερο μέγεθος).

**Q: Υποστηρίζει το Aspose.BarCode .NET Core και .NET 5/6;**  
A: Απολύτως – η βιβλιοθήκη είναι πλήρως συμβατή με .NET Framework, .NET Core και τις τελευταίες εκδόσεις του .NET.
