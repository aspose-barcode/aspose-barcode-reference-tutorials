## Εισαγωγή Namespaces

Για να ξεκινήσετε, εισάγετε το απαιτούμενο namespace ώστε να έχετε πρόσβαση στις κλάσεις δημιουργίας γραμμωτού κώδικα:

```csharp
using Aspose.BarCode.Generation;
```

## Πώς να δημιουργήσετε γραμμωτούς κώδικες DataMatrix ECC 200

### Βήμα 1: Αρχικοποίηση του Barcode Generator

Σε αυτό το απόσπασμα δημιουργούμε ένα αντικείμενο `BarcodeGenerator`, του λέμε ότι θέλουμε έναν **DataMatrix** κώδικα και παρέχουμε τα δεδομένα προς κωδικοποίηση. Αντικαταστήστε το `"Your Directory Path"` με το φάκελο όπου θέλετε να αποθηκευτεί η εικόνα.

```csharp
string path = "Your Directory Path";
System.Console.WriteLine("DataMatrixEcc200:");

using (BarcodeGenerator gen = new BarcodeGenerator(EncodeTypes.DataMatrix, "Åspóse.Barcóde©"))
{
    // Your code goes here
}
```

### Βήμα 2: Ορισμός XDimension και ECC Type

Εδώ ορίζουμε το **XDimension** (το μέγεθος κάθε μονάδας) και επιλέγουμε **ECC 200** για ισχυρή διόρθωση σφαλμάτων. Ρυθμίστε την τιμή pixel εάν χρειάζεστε μεγαλύτερες ή μικρότερες μονάδες.

```csharp
gen.Parameters.Barcode.XDimension.Pixels = 4;
gen.Parameters.Barcode.DataMatrix.DataMatrixEcc = DataMatrixEccType.Ecc200;
```

### Βήμα 3: Δημιουργία και αποθήκευση της εικόνας του γραμμωτού κώδικα

Η μέθοδος `Save` γράφει τον γραμμωτό κώδικα σε αρχείο PNG. Μπορείτε να αλλάξετε το `BarCodeImageFormat.Png` σε `Jpeg` ή `Tiff` εάν χρειάζεται. Αυτό είναι το βασικό μέρος του **generate barcode image C#** χρησιμοποιώντας το Aspose.

```csharp
gen.Save($"{path}DataMatrixEcc200.png", BarCodeImageFormat.Png);
```
