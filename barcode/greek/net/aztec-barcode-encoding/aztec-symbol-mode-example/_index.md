## Πώς να ρυθμίσετε το Barcode Generator για κωδικούς Aztec;

Η κλάση `BarcodeGenerator` είναι το βασικό στοιχείο που δημιουργεί εικόνες barcode βάσει της επιλεγμένης συμβολισμού και των επιλογών διαμόρφωσης. Παρέχει ένα απλό API για να καθορίσετε τον τύπο του barcode, τα δεδομένα προς κωδικοποίηση και διάφορες παραμέτρους απόδοσης πριν αποθηκεύσετε το αποτέλεσμα σε αρχείο εικόνας.

```csharp
string path = "Your Directory Path";
System.Console.WriteLine("AztecSymbolModeExample:");

BarcodeGenerator gen = new BarcodeGenerator(EncodeTypes.Aztec, "Åspóse.Barcóde©");
```

Σε αυτό το βήμα, έχουμε ρυθμίσει το γεννήτρια και έχουμε δώσει το κείμενο κώδικα για κωδικοποίηση.

## Πώς να ορίσετε τη Λειτουργία Συμβόλου Aztec σε Auto;

Η απαρίθμηση `AztecSymbolMode` ορίζει τις τέσσερις πιθανές λειτουργίες συμβόλου για κωδικούς Aztec, και η επιλογή **Auto** επιτρέπει στη βιβλιοθήκη να επιλέγει αυτόματα το πιο συμπαγές μέγεθος διατηρώντας όλα τα δεδομένα και τις απαιτήσεις διόρθωσης σφαλμάτων. Αυτή η λειτουργία είναι ιδανική για τις περισσότερες περιπτώσεις όπου θέλετε τον μικρότερο δυνατό κωδικό χωρίς χειροκίνητη ρύθμιση.

```csharp
gen.Parameters.Barcode.Aztec.AztecSymbolMode = AztecSymbolMode.Auto;
gen.Save($"{path}AztecSymbolModeAuto.png", BarCodeImageFormat.Png);
```

Αυτό το βήμα εξασφαλίζει ότι η Λειτουργία Συμβόλου Aztec καθορίζεται αυτόματα και η προκύπτουσα εικόνα του barcode αποθηκεύεται.

## Πώς να εξαναγκάσετε τη Λειτουργία Συμβόλου FullRange;

Όταν χρειάζεστε τη μέγιστη χωρητικότητα δεδομένων, μπορείτε να επιλέξετε την τιμή **FullRange** της απαρίθμησης `AztecSymbolMode`. Αυτή η λειτουργία απενεργοποιεί την αυτόματη μείωση μεγέθους, επιτρέποντας στο barcode να αποθηκεύει όλο το εύρος χαρακτήρων και να επιτυγχάνει τη μέγιστη δυνατή πυκνότητα πληροφορίας, κάτι που είναι χρήσιμο για μεγάλα σύνολα δεδομένων.

```csharp
gen.Parameters.Barcode.Aztec.AztecSymbolMode = AztecSymbolMode.FullRange;
gen.Save($"{path}AztecSymbolModeFullRange.png", BarCodeImageFormat.Png);
```

Αυτό θα δημιουργήσει ένα barcode με τη Λειτουργία Συμβόλου Aztec FullRange.

## Πώς να δημιουργήσετε ένα Compact Aztec barcode;

Η τιμή **Compact** της απαρίθμησης `AztecSymbolMode` παράγει ένα μικρότερο barcode μειώνοντας τον αριθμό των επιπέδων που χρησιμοποιούνται στο πλέγμα. Αυτό οδηγεί σε μια πιο εξοικονομητική σε χώρο εικόνα, καθιστώντας το κατάλληλο για εφαρμογές με περιορισμένη επιφάνεια προβολής όπως κινητές οθόνες ή μικρές ετικέτες.

```csharp
gen.Parameters.Barcode.Aztec.AztecSymbolMode = AztecSymbolMode.Compact;
gen.Save($"{path}AztecSymbolModeCompact.png", BarCodeImageFormat.Png);
```

Αυτό το βήμα διαμορφώνει το barcode ώστε να δημιουργείται με τη Λειτουργία Συμβόλου Aztec Compact.

## Πώς να δημιουργήσετε ένα Rune Aztec barcode;

Η λειτουργία **Rune** είναι μια εξειδικευμένη παραλλαγή της συμβολισμού Aztec που κωδικοποιεί αριθμητικά δεδομένα χρησιμοποιώντας ένα προσαρμοσμένο σύνολο χαρακτήρων, προσφέροντας ένα ξεχωριστό οπτικό στυλ ενώ διατηρεί την ίδια ισχύ διόρθωσης σφαλμάτων με άλλες λειτουργίες. Είναι χρήσιμη όταν χρειάζεστε ένα barcode που τονίζει αριθμητικές πληροφορίες.

```csharp
gen.CodeText = "123"; // Change the code text if needed
gen.Parameters.Barcode.Aztec.AztecSymbolMode = AztecSymbolMode.Rune;
gen.Save($"{path}AztecSymbolModeRune.png", BarCodeImageFormat.Png);
```

Αυτό το βήμα αλλάζει το κείμενο κώδικα σε "123" και δημιουργεί ένα barcode με τη Λειτουργία Συμβόλου Aztec Rune.
