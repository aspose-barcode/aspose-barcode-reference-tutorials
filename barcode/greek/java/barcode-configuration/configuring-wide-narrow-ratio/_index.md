# Πώς να δημιουργήσετε CODE_128 Barcode με λόγο Wide‑Narrow σε Java

## Εισαγωγή

Αν χρειάζεστε **δημιουργία code128 barcode** με ακριβείς οπτικές αναλογίες, η ρύθμιση του λόγου wide‑narrow είναι το κλειδί. Σε αυτό το tutorial θα περάσουμε βήμα‑βήμα τη διαδικασία **δημιουργίας barcode** χρησιμοποιώντας το Aspose.BarCode for Java, δείχνοντάς σας πώς να διαμορφώσετε το λόγο, **να δημιουργήσετε barcode PNG** εικόνες, και **να αποθηκεύσετε την εικόνα barcode** στο δίσκο. Είτε δημιουργείτε ετικέτες αποθέματος, ετικέτες αποστολής ή οποιαδήποτε εφαρμογή που απαιτεί προσαρμοσμένο CODE_128 barcode, θα βρείτε όλα όσα χρειάζεστε εδώ.

## Γρήγορες Απαντήσεις
- **Τι είναι ο λόγος wide‑narrow;** Ελέγχει το σχετικό πλάτος των φαρδιών μπαρών σε σχέση με τις στενές μπαρ σε ένα barcode.  
- **Ποια συμβολική υποστηρίζει ρύθμιση του λόγου;** Οι περισσότερες 1‑D συμβολικές, συμπεριλαμβανομένου του CODE_128, επιτρέπουν τον ορισμό προσαρμοσμένου λόγου.  
- **Χρειάζομαι άδεια;** Διατίθεται δωρεάν δοκιμαστική έκδοση, αλλά απαιτείται εμπορική άδεια για παραγωγική χρήση.  
- **Μπορώ να δημιουργήσω εικόνα barcode σε μορφή PNG;** Ναι—χρησιμοποιήστε `generator.save(...)` για να δημιουργήσετε εικόνες barcode PNG.  
- **Είναι ο κώδικας συμβατός με Java 8+;** Απόλυτα· το Aspose.BarCode λειτουργεί με όλες τις σύγχρονες εκδόσεις της Java.

## Προαπαιτούμενα

Πριν βυθιστούμε στον κώδικα, βεβαιωθείτε ότι έχετε τα εξής:

- Java Development Kit (JDK) εγκατεστημένο στον υπολογιστή σας.  
- Aspose.BarCode for Java library. Κατεβάστε την από το [download link](https://releases.aspose.com/barcode/java/).

## Εισαγωγή Πακέτων

Για να ξεκινήσετε, εισάγετε την απαραίτητη κλάση Aspose.BarCode στο έργο σας.

```java
// Import Aspose.BarCode library
import com.aspose.barcode.generation.BarcodeGenerator;
```

## Τι είναι ο λόγος wide‑narrow και γιατί να τον ρυθμίσετε;

Ο λόγος wide‑narrow καθορίζει πόσο παχύς φαίνεται ο “wide” μπαρ σε σύγκριση με τον “narrow”. Η ρύθμιση αυτού του λόγου μπορεί να βελτιώσει την αναγνωσιμότητα από σαρωτές, να πληροί συγκεκριμένα πρότυπα εκτύπωσης ή απλώς να ταιριάζει με το οπτικό στυλ μιας μάρκας.

## Πώς να δημιουργήσετε code128 barcode με λόγο wide‑narrow σε Java

Παρακάτω βρίσκεται ένας **step‑by‑step barcode** οδηγός που σας καθοδηγεί σε κάθε βήμα της διαδικασίας.

### Βήμα 1: Ορισμός Καταλόγου Εγγράφου

```java
// The path to the resource directory.
String dataDir = "Your Document Directory";
```

Βεβαιωθείτε ότι ο φάκελος υπάρχει και έχετε δικαιώματα εγγραφής· εδώ θα τοποθετηθεί το αρχείο **save barcode image**.

### Βήμα 2: Δημιουργία Αντικειμένου Barcode

```java
// Instantiate barcode object
// Create an instance of BarcodeGenerator, specify codetext and symbology in the constructor
BarcodeGenerator generator = new BarcodeGenerator(EncodeTypes.CODE_128, "12345678");
```

Εδώ **δημιουργούμε code128 barcode** περνώντας `EncodeTypes.CODE_128` στον κατασκευαστή.

### Βήμα 3: Ορισμός Λόγου Wide‑Narrow

```java
// Set the wide to narrow ratio for the barcode
generator.getParameters().getBarcode().setWideNarrowRatio(3.0f);
```

Η μέθοδος `setWideNarrowRatio` σας επιτρέπει να ρυθμίσετε ακριβώς το οπτικό διάστημα. Μια τιμή `3.0f` σημαίνει ότι η φαρδύς μπαρ είναι τρεις φορές το πλάτος μιας στενής μπαρ.

### Βήμα 4: Αποθήκευση Εικόνας στο Δίσκο

```java
// Save the image to disk in PNG format
generator.save(dataDir + "wideNarrowRatio.png");
```

Καλώντας το `save` θα **δημιουργήσει εικόνα barcode** και θα την αποθηκεύσει ως αρχείο PNG, ολοκληρώνοντας το βήμα **save barcode image**.

## Γιατί να ρυθμίσετε το λόγο wide‑narrow;

- **Συμβατότητα Σαρωτών:** Ορισμένοι σαρωτές προτιμούν λόγο μεταξύ 2.0 και 3.0 για βέλτιστη ανάγνωση.  
- **Ποιότητα Εκτύπωσης:** Η ρύθμιση του λόγου μπορεί να αντισταθμίσει περιορισμούς DPI του εκτυπωτή, αποτρέποντας παραμορφωμένες μπαρ.  
- **Συνέπεια Μάρκας:** Κάποιες εταιρείες θέλουν barcodes που ταιριάζουν οπτικά με τα εταιρικά χρώματα ή τις οδηγίες σχεδίασης.

## Συχνά Προβλήματα και Λύσεις

| Πρόβλημα | Αιτία | Διόρθωση |
|----------|-------|----------|
| Το barcode φαίνεται παραμορφωμένο | Ο λόγος είναι πολύ υψηλός/χαμηλός για τον εκτυπωτή | Ρυθμίστε την τιμή που περνάτε στο `setWideNarrowRatio` (π.χ., 2.0‑2.5). |
| Το αρχείο δεν δημιουργήθηκε | Μη έγκυρη διαδρομή `dataDir` ή ανεπαρκή δικαιώματα | Επαληθεύστε τη διαδρομή του καταλόγου και βεβαιωθείτε ότι η εφαρμογή έχει δικαιώματα εγγραφής. |
| Ο σαρωτής δεν μπορεί να διαβάσει το barcode | Ο λόγος εκτός του προτεινόμενου εύρους για τη συμβολική | Χρησιμοποιήστε τυπικούς λόγους (2.0‑3.0) ή δοκιμάστε με τον στόχο σαρωτή. |
