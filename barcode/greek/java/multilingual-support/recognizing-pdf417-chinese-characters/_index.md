# Ανάκτηση Κειμένου από Barcode: PDF417 Κινέζικοι Χαρακτήρες σε Java

Στο τέλος, θα γνωρίζετε ακριβώς πώς να διαβάζετε τα δεδομένα του barcode και να τα αποκωδικοποιείτε σε αναγνώσιμο κείμενο.

## Τι είναι η “εξαγωγή κειμένου από barcode”?

Η εξαγωγή κειμένου από ένα barcode σημαίνει τη μετατροπή των κωδικοποιημένων δεδομένων πίσω στην αρχική, ανθρώπινα αναγνώσιμη μορφή. Για barcode PDF417 που αποθηκεύουν κινέζικους χαρακτήρες, αυτό περιλαμβάνει επίσης την επιλογή του σωστού κωδικοποίησης χαρακτήρων ώστε τα byte να αντιστοιχούν στα κατάλληλα γλύφους.

## Γιατί να χρησιμοποιήσετε Aspose.BarCode for Java;

Το Aspose.BarCode παρέχει ισχυρή υποστήριξη για ένα ευρύ φάσμα συμβολισμών barcode, συμπεριλαμβανομένου του PDF417, και διαχειρίζεται σύνθετα σύνολα χαρακτήρων αμέσως. Απομονώνει την επεξεργασία εικόνας χαμηλού επιπέδου, επιτρέποντάς σας να εστιάσετε στη λογική της εφαρμογής αντί στην αποκωδικοποίηση.

## Εισαγωγή Πακέτων

Στο έργο Java, εισάγετε τις απαραίτητες κλάσεις για εργασία με Aspose.BarCode:

```java
import java.nio.ByteBuffer;
import java.nio.charset.Charset;

import com.aspose.barcode.barcoderecognition.BarCodeReader;
import com.aspose.barcode.barcoderecognition.BarCodeResult;
import com.aspose.barcode.barcoderecognition.DecodeType;
```

## Βήμα 1: Ορισμός του Καταλόγου Εγγράφου

Καθορίστε το φάκελο όπου βρίσκεται η εικόνα του barcode:

```java
String dataDir = "Your Document Directory";
```

Αντικαταστήστε το `"Your Document Directory"` με την πραγματική διαδρομή στο μηχάνημά σας.

## Βήμα 2: Φόρτωση Εικόνας Barcode

Δημιουργήστε μια παρουσία `BarCodeReader` που δείχνει στο αρχείο PNG και ενημερώνει τον αναγνώστη να ψάξει για συμβολισμό PDF417:

```java
BarCodeReader reader = new BarCodeReader(dataDir + "barcode.png", DecodeType.PDF_417);
```

## Βήμα 3: Ανάγνωση Barcode

Περιηγηθείτε στα αποτελέσματα ανίχνευσης, εξάγετε τον ακατέργαστο πίνακα byte και αποκωδικοποιήστε τον χρησιμοποιώντας το σύνολο χαρακτήρων GBK (MS936):

```java
for (BarCodeResult result : reader.readBarCodes()) {
    byte[] bytes = result.getCodeBytes();
    ByteBuffer bytebuf = ByteBuffer.wrap(bytes);
    System.out.println(Charset.forName("MS936").decode(bytebuf).toString());
}
```

Αυτός ο βρόχος **εξάγει κείμενο από το barcode** και εκτυπώνει τους αποκωδικοποιημένους κινέζικους χαρακτήρες στην κονσόλα.

## Πώς να διαβάσετε barcode με PDF417;

Ο παραπάνω κώδικας δείχνει **πώς να διαβάσετε δεδομένα barcode** βήμα προς βήμα:

1. **Αρχικοποίηση** του αναγνώστη με το σωστό `DecodeType`.  
2. **Επανάληψη** σε κάθε `BarCodeResult` που επιστρέφεται.  
3. **Μετατροπή** των ακατέργαστων bytes χρησιμοποιώντας το κατάλληλο charset (`MS936` για Κινέζικα).  

Αν το barcode σας περιέχει άλλες γλώσσες, απλώς αλλάξτε το charset στο αντίστοιχο των δεδομένων σας.
