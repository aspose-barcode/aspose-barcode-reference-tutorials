# Μετρική Ποιότητας Barcode – Aspose.Barcode Java

## Εισαγωγή

Εάν χρειάζεστε **αξιολόγηση της ποιότητας ανάγνωσης barcode** σε μια εφαρμογή Java, η **μετρική ποιότητας barcode** που προσφέρει η **Aspose.Barcode Java** παρέχει ένα σαφές σκορ εμπιστοσύνης για κάθε αποκωδικοποιημένο σύμβολο. Σε αυτό το tutorial θα περάσουμε βήμα‑βήμα τις ακριβείς ενέργειες που απαιτούνται για την ανάκτηση αυτού του ποσοστού, θα εξηγήσουμε γιατί η μετρική είναι σημαντική και θα σας δείξουμε πώς να ενσωματώσετε την κλήση στον υπάρχοντα κώδικά σας ώστε να μπορείτε να **επαληθεύετε αξιόπιστα τα αποτελέσματα σάρωσης barcode**.

## Γρήγορες** Είναι το σκορ εμπιστοσύνης (0‑100 %) που η βιβλιοθήκη αντιστοιχεί σε κάθε αποκωδικοποιημένο barcode.  
- **Ποια έκδοση της βιβλιοθήκης απαιτείται;** Οποιαδήποτε πρόσφατη έκδοση Aspose.Barcode Java (το παράδειγμα χρησιμοποιεί τη σειρά 24.x).  
- **Χρειάζομαι άδεια;** Μια προσωρινή άδεια λειτουργεί για δοκιμές· πλήρης άδεια απαιτείται για παραγωγή.  
- **Μπορώ να διαβάσω όλους η σημαία `DecodeType.ALL_SUPPORTED_TYPES` ενεργοποιεί κάθε μορφή που υποστηρίζει η Aspose.Barcode.  
- **Είναι αξιόπιστη η τιμή ποιότητας για QR codes;** Απόλυτα – ο ίδιος αλγόριθμος εμπιστοσύνης εφαρμόζεται σε συμβολές 1‑D και 2‑D.

## Εισαγωγή Ονομάτων Χώρων

Οι παρακάτω εισαγωγές σας δίνουν πρόσβαση στις κλάσεις reader και result που χρειάζονται για την εξαγωγή της ποιότητας.

```java
import com.aspose.barcode.barcoderecognition.BarCodeReader;
import com.aspose.barcode.barcoderecognition.BarCodeResult;
```

### Βήμα 1: Ορισμός Διαδρομής Καταλόγου Πόρων

Ορίστε το φάκελο που περιέχει τη δειγματική εικόνα. Η `Utils.getDataDir` είναι βοηθητική μέθοδος που επιλύει την απόλυτη διαδρομή για το τρέχον έργο.

```java
// The path to the resource directory.
String dataDir = Utils.getDataDir(GetBarCodeRecognitionQualityInPercent.class)
        + "BarcodeReader/advanced_features/";
```

### Βήμα 2: Αρχικοποίηση του Αντικειμένου BarCodeReader

Δημιουργήστε ένα στιγμιότυπο `BarCodeReader`, δείχνοντάς του το αρχείο εικόνας και ζητώντας του να δοκιμάσει **όλους τους υποστηριζόμενους τύπους barcode**.

```java
// Initialize the BarCodeReader object
BarCodeReader reader = new BarCodeReader(dataDir + "code39Extended.jpg",
        com.aspose.barcode.barcoderecognition.DecodeType.ALL_SUPPORTED_TYPES);
```

### Βήμα 3: Ανάγνωση των Barcodes και Ανάκτηση του Ποσοστού Ποιότητας

Διατρέξτε κάθε εντοπισμένο barcode, εκτυπώστε το κείμενο, τον τύπο και το **ποσοστό ποιότητας ανάγνωσης** που επιστρέφει η `getReadingQuality()`.

```java
// Call the read method
for (BarCodeResult result : reader.readBarCodes()) {
    System.out.println(result.getCodeText() + " Type: " + result.getCodeType());
    double percent = result.getReadingQuality();
    System.out.println("Percent: " + percent);
}
```

**Τι συμβαίνει εδώ;**  
- Η `readBarCodes()` επιστρέφει μια συλλογή αντικειμένων `BarCodeResult`, ένα για κάθε barcode που βρέθηκε.  
- Η `getReadingQuality()` παρέχει ένα `double` μεταξύ `0` και `100`, που αντιπροσωπεύει το επίπεδο εμπιστοσύνης – τη **μετρική ποιότητας barcode**.  
- Μπορείτε να χρησιμοποιήσετε αυτήν την τιμή για να αποφασίσετε αν η σάρωση είναι αποδεκτή ή αν χρειάζεται να προτρέψετε τον χρήστη για μια νέα προσπάθεια, επαληθεύοντας έτσι **τη σάρωση barcode**.

| **Δεν εντοπίστηκαν barcodes** | Λάθος σημαία `DecodeType`. | Βεβαιωθείτε ότι χρησιμοποιείτε `DecodeType.ALL_SUPPORTED_TYPES` ή καθορίστε τον ακριβή τύπο που περιμένετε. |
