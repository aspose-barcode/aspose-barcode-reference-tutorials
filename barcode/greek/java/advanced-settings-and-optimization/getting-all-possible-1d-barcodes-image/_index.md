## Εισαγωγή Ονοματοχώρων

Προσθέστε τις απαιτούμενες δηλώσεις `import` ώστε ο μεταγλωττιστής να εντοπίζει τις κλάσεις του Aspose.

```java
import java.awt.Point;

import com.aspose.barcode.barcoderecognition.BarCodeReader;
import com.aspose.barcode.barcoderecognition.BarCodeResult;
import com.aspose.barcode.barcoderecognition.DecodeType;
```

## Παράδειγμα κώδικα αναγνώστη γραμμωτών κωδίκων Java

### Βήμα 1: Αρχικοποίηση του αντικειμένου BarCodeReader

Δημιουργήστε μια παρουσία `BarCodeReader` που δείχνει στο αρχείο εικόνας σας. Η παράμετρος `DecodeType` ενημερώνει τη μηχανή για το ποιους συμβολισμούς πρέπει να αναζητήσει· η χρήση του `CODE_128` ως παράδειγμα λειτουργεί για πολλούς κοινούς 1D κώδικες.

```java
BarCodeReader reader = new BarCodeReader("path/to/your/image.png", DecodeType.CODE_128);
```

> **Συμβουλή:** Αν θέλετε να σαρώσετε *όλους* τους υποστηριζόμενους τύπους 1D, περάστε `DecodeType.ALL_1D` αντί για έναν μόνο συμβολισμό.

### Βήμα 2: Ανάγνωση όλων των πιθανών γραμμωτών κωδίκων

Επανάληψη μέσω της συλλογής που επιστρέφει η `readBarCodes()`. Για κάθε `BarCodeResult` εκτυπώνουμε το αποκωδικοποιημένο κείμενο, το όνομα του συμβολισμού, τη γωνία ανίχνευσης και τις τέσσερις συντεταγμένες γωνιών της περιοχής του κώδικα.

```java
int iCount = 0;
for (BarCodeResult result : reader.readBarCodes()) {
    // Display code text, symbology, detected angle, recognition percentage of the barcode
    System.out.println("Code Text: " + result.getCodeText() + " Symbology: " + result.getCodeTypeName()
            + " Recognition percentage: " + result.getRegion().getAngle());

    // Display x and y coordinates of barcode detected
    Point[] point = result.getRegion().getPoints();

    System.out.println("Top left coordinates: X = " + point[0].getX() + ", Y = " + point[0].getY());
    System.out.println("Bottom left coordinates: X = " + point[1].getX() + ", Y = " + point[1].getY());
    System.out.println("Bottom right coordinates: X = " + point[2].getX() + ", Y = " + point[2].getY());
    System.out.println("Top right coordinates: X = " + point[3].getX() + ", Y = " + point[3].getY());

    iCount = iCount + 1;
}
```

Ο βρόχος επεξεργάζεται αυτόματα κάθε κώδικα που βρέθηκε, οπότε δεν χρειάζεταιiCount` περιέχει το συνολικό αριθμό των κωδίκων που εντοπίστηκαν – ιδανικό για πολλαπλούς γραμμωτούς κώδικες από εικόνα** σε μία μόνο διεργασία.
