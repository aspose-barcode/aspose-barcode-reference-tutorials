## Vanliga frågor

**Q: Kan jag använda Aspose.BarCode i ett kommersiellt projekt?**  
A: Ja. En giltig Aspose‑licens krävs för produktionsanvändning, och en gratis provversion finns för utvärdering.

**Q: Fungerar biblioteket med lösenordsskyddade PDF‑filer?**  
A: Absolut. Du kan ange PDF‑lösenordet när du laddar dokumentet, och streckkoddetektering fortsätter som vanligt.

**Q: Vilka Java‑versioner stöds?**  
A: API:et stöder Java 8 och senare, inklusive Java 11, Java 17 och nyare LTS‑utgåvor.

**Q: Hur kan jag förbättra detekteringshastigheten för stora dokumentbatcher?**  
A: Använd `BarCodeReader` i multitrådad läge och aktivera `ReadQuality`‑alternativet för att balansera hastighet och noggrannhet.

**Q: Vad händer om en streckkod inte upptäcks?**  
A: Verifiera att streckkodssymbologin stöds och att bildens upplösning uppfyller miniminivån för DPI‑rekommendation (300 dpi för de flesta 1D‑streckkoder).
