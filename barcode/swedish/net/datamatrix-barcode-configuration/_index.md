## Vanliga frågor

**Q: Hur bestämmer jag vilken ECC‑mode jag ska använda?**  
A: Välj ECC 000‑140 för små datamängder med begränsad felkorrigering, eller ECC 200 för större data och högre tillförlitlighet. Macro‑läge lägger till ytterligare datalager.

**Q: Kan jag bädda in anpassad text i en DataMatrix‑streckkod?**  
A: Ja, använd `CodeText`‑egenskapen för att ange utökad kodtext, och konfigurera sedan kodningsläget efter behov.

**Q: Finns det ett sätt att automatiskt välja det bästa kodningsläget?**  
A: Ställ in `EncodeMode` till `Auto` så kommer Aspose.BarCode att utvärdera indata och välja det optimala läget (ASCII, C40, Bytes, etc.).

**Q: Vilka prestandaöverväganden finns för stora streckkodsbatcher?**  
A: Återanvänd `BarCodeGenerator`‑instansen och aktivera flertrådad bearbetning där det är möjligt. Generera också bilder i ett lämpligt format (PNG för förlustfri, JPEG för mindre storlek).

**Q: Stöder Aspose.BarCode .NET Core och .NET 5/6?**  
A: Absolut – biblioteket är fullt kompatibelt med .NET Framework, .NET Core och de senaste .NET‑utgåvorna.
