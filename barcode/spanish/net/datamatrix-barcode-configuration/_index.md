Un código de barras DataMatrix codifica datos en un patrón bidimensional de celdas negras y blancas. Usando Aspose.BarCode, puedes **generate datamatrix barcode** imágenes, PDFs o SVGs directamente desde tus aplicaciones .NET, dándote control total sobre el tamaño, la corrección de errores y el modo de codificación.

## Preguntas Frecuentes

**Q: ¿Cómo decido qué modo ECC usar?**  
A: Elige ECC 000‑140 para conjuntos de datos pequeños con corrección de errores limitada, o ECC 200 para datos más grandes y mayor fiabilidad. El modo macro agrega capas de datos adicionales.

**Q: ¿Puedo incrustar texto personalizado en un código de barras DataMatrix?**  
A: Sí, usa la propiedad `CodeText` para establecer el texto de código extendido, luego configura el modo de codificación según sea necesario.

**Q: ¿Existe una forma de seleccionar automáticamente el mejor modo de codificación?**  
A: Configura `EncodeMode` a `Auto` y Aspose.BarCode evaluará la entrada y elegirá el modo óptimo (ASCII, C40, Bytes, etc.).

**Q: ¿Cuáles son las consideraciones de rendimiento para lotes grandes de códigos de barras?**  
A: Reutiliza la instancia `BarCodeGenerator` y habilita el multihilo donde sea posible. Además, genera imágenes en un formato adecuado (PNG para sin pérdida, JPEG para menor tamaño).

**Q: ¿Aspose.BarCode soporta .NET Core y .NET 5/6?**  
A: Absolutamente – la biblioteca es totalmente compatible con .NET Framework, .NET Core y las últimas versiones de .NET.
