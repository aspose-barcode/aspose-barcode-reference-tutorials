## 常见问题

**Q: 如何决定使用哪种 ECC 模式？**  
A: 小数据量且错误纠正需求有限时选择 ECC 000‑140，数据量大且需要更高可靠性时选择 ECC 200。宏模式会添加额外的数据层。

**Q: 能否在 DataMatrix 条形码中嵌入自定义文本？**  
A: 可以，使用 `CodeText` 属性设置扩展代码文本，然后根据需要配置编码模式。

**Q: 有办法自动选择最佳编码模式吗？**  
A: 将 `EncodeMode` 设置为 `Auto`，Aspose.BarCode 会评估输入并自动挑选最优模式（ASCII、C40、Bytes 等）。

**Q: 大批量条形码的性能考虑有哪些？**  
A: 重用 `BarCodeGenerator` 实例，并在可能的情况下启用多线程。同时，选择合适的图像格式（无损 PNG 或更小的 JPEG）进行生成。

**Q: Aspose.BarCode 是否支持 .NET Core 和 .NET 5/6？**  
A: 完全支持 – 该库兼容 .NET Framework、.NET Core 以及最新的 .NET 发行版。
