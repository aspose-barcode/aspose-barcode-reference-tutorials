## Häufig gestellte Fragen

**Q: Wie entscheide ich, welchen ECC‑Modus ich verwenden soll?**  
A: Wählen Sie ECC 000‑140 für kleine Datenmengen mit begrenzter Fehlerkorrektur oder ECC 200 für größere Daten und höhere Zuverlässigkeit. Der Makro‑Modus fügt zusätzliche Datenschichten hinzu.

**Q: Kann ich benutzerdefinierten Text in einen DataMatrix‑Barcode einbetten?**  
A: Ja, verwenden Sie die `CodeText`‑Eigenschaft, um erweiterten Code‑Text festzulegen, und konfigurieren Sie anschließend den gewünschten Codierungsmodus.

**Q: Gibt es eine Möglichkeit, den besten Codierungsmodus automatisch auszuwählen?**  
A: Setzen Sie `EncodeMode` auf `Auto` und Aspose.BarCode bewertet die Eingabe und wählt den optimalen Modus (ASCII, C40, Bytes usw.) aus.

**Q: Welche Leistungsaspekte sind bei großen Barcode‑Chargen zu beachten?**  
A: Verwenden Sie dieselbe `BarCodeGenerator`‑Instanz wieder und aktivieren Sie nach Möglichkeit Multi‑Threading. Außerdem sollten Sie Bilder in einem geeigneten Format erzeugen (PNG für verlustfrei, JPEG für kleinere Dateigröße).

**Q: Unterstützt Aspose.BarCode .NET Core und .NET 5/6?**  
A: Ja – die Bibliothek ist vollständig kompatibel mit .NET Framework, .NET Core und den neuesten .NET‑Versionen.
