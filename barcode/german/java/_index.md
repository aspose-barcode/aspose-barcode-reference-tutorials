## Häufig gestellte Fragen

**Q:** *Kann ich Barcodes ohne Lizenz für die Entwicklung generieren?*  
**A:** Ja, Aspose stellt eine temporäre kostenlose Lizenz für Evaluation und Entwicklung bereit. Für den Produktionseinsatz ist eine Voll‑Lizenz erforderlich.

**Q:** *Welche Bildformate kann ich zum Export von Barcodes verwenden?*  
**A:** PNG, JPEG, BMP, GIF, TIFF und mehrere Vektorformate wie SVG und PDF werden unterstützt.

**Q:** *Wie lese ich Barcodes aus gescannten PDF‑Dokumenten?*  
**A:** Verwenden Sie die **document barcode recognition**‑API, um das PDF zu laden, und rufen Sie anschließend den `BarCodeReader` auf, um Barcodes von jeder Seite zu extrahieren.

**Q:** *Ist es möglich, die Farbe der Barcode‑Balken und des Hintergrunds zu ändern?*  
**A:** Absolut. Die Klasse `BarCodeImage` ermöglicht das Setzen von `BarColor` und `BackColor` auf jeden beliebigen `java.awt.Color`‑Wert.

**Q:** *Kann ich den Beschriftungstext unter einem QR‑Code anpassen?*  
**A:** Ja, die **barcode text styling**‑Optionen erlauben das Hinzufügen einer Beschriftung, die Auswahl von Schriftart, Größe und Ausrichtung.
