## Häufig gestellte Fragen

**F: Kann ich Aspose.BarCode in einem kommerziellen Projekt verwenden?**  
A: Ja. Für die Produktion ist eine gültige Aspose‑Lizenz erforderlich, eine kostenlose Testversion steht für Evaluierungen zur Verfügung.

**F: Arbeitet die Bibliothek mit passwortgeschützten PDFs?**  
A: Absolut. Sie können das PDF‑Passwort beim Laden des Dokuments angeben, und die Barcode‑Erkennung wird normal fortgesetzt.

**F: Welche Java‑Versionen werden unterstützt?**  
A: Die API unterstützt Java 8 und höher, einschließlich Java 11, Java 17 und neueren LTS‑Versionen.

**F: Wie kann ich die Erkennungsgeschwindigkeit für große Dokumenten‑Batches verbessern?**  
A: Verwenden Sie den `BarCodeReader` im Multi‑Thread‑Modus und aktivieren Sie die Option `ReadQuality`, um Geschwindigkeit und Genauigkeit auszubalancieren.

**F: Was, wenn ein Barcode nicht erkannt wird?**  
A: Stellen Sie sicher, dass die Barcode‑Symbologie unterstützt wird und die Bildauflösung die empfohlene Mindest‑DPI (300 dpi für die meisten 1D‑Barcodes) erfüllt.
